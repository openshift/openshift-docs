= GCP architecture models
include::_attributes/attributes-openshift-dedicated.adoc[]
:context: osd-architecture-models-gcp

toc::[]

{product-title} on {GCP} has three deployment offerings supporting the following cluster network configuration types:

* Public cluster
* Private cluster with Private Service Connect (PSC)
* Private cluster without PSC

You can customize the access patterns for your API service endpoint and Red Hat SRE Management by choosing one of these network configuration types.

[IMPORTANT]
====
PSC is a capability of Google Cloud networking that allows private consumption of services across VPC networks and is supported on existing VPCs only. Red Hat recommends the use of Private Service Connect-based architecture when deploying private {product-title} clusters on Google Cloud Platform.
====
