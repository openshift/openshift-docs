[role="_abstract"]
Create a {product-title} cluster by using {cluster-manager-first} on the {hybrid-console-url}. After you create your cluster, you can grant user access, deploy your application, revoke user access, and delete your cluster.

[NOTE]
====
If you are looking for a comprehensive getting started guide for {product-title}, see the comprehensive guide to getting started with {product-title}.
====

You can create a cluster that uses AWS Security Token Service (STS).
