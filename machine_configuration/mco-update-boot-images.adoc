= Boot image management

toc::[]

[role="_abstract"]
For clusters on specific {product-title} platforms, the Machine Config Operator (MCO) can manage and update the boot image on each node. The MCO uses this boot image to scale up new nodes. Updating the {op-system-first} version of the boot image ensures it the matches the {op-system-first} version appropriate for your {product-title} version.
