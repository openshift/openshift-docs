= Using credentials to access external APIs with Vault
:context: mcp-gateway-vault

toc::[]

[role="_abstract"]
You can connect AI agents to tools with the Model Context Protocol (MCP) by integrating HashiCorp Vault (Vault) with the {mcpg}. By using Vault, you can store sensitive credentials such as API keys or long-term Personal Access Tokens (PATs) to access external APIs.
