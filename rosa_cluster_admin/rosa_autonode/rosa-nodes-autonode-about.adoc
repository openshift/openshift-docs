= About Red{nbsp}Hat build of Karpenter
:context: rosa-nodes-autonode-about

toc::[]

[role="_abstract"]
{autonode} is a Kubernetes autoscaler based on the upstream project, Karpenter. Karpenter automatically launches the exact compute resources to handle your cluster's applications. It is designed to let you take full advantage of the cloud with fast and simple compute provisioning for Kubernetes clusters.
