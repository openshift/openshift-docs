= About `AutoNode`
:context: rosa-nodes-autonode-about

toc::[]

// OSDOCS-18308, ROSA-17

[role="_abstract"]
`AutoNode` automates worker node provisioning and removal on {product-title} clusters when pods cannot schedule on existing nodes.
You can use `AutoNode` together with machine pools when you need dynamic capacity that responds to scheduling constraints.
