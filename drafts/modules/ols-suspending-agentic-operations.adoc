[id="ols-suspending-agentic-operations_{context}"]
= Suspending all agentic operations

[role="_abstract"]
Suspend all agentic operations across the cluster to immediately halt agent activity.

.Procedure

. To suspend all agentic operations:
+
[source,terminal]
----
$ oc agentic suspend --yes
----

. To check the current suspension state:
+
[source,terminal]
----
$ oc agentic status
----

. To resume operations after investigation:
+
[source,terminal]
----
$ oc agentic resume
----
