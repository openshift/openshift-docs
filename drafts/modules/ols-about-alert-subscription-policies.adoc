= Alert subscription policies

[role="_abstract"]
A configurable policy framework determines which alerts the agent is authorized to act upon.

As a cluster administrator, you configure alert subscription policies to control the following:

* Which Prometheus alerts the agent responds to
* Which namespaces the agent is authorized to investigate (opt-in)
* What level of remediation the agent can propose

The system focuses only on critical system events in user-specified namespaces. The agent does not act on alerts outside the scope defined by your policies.

The `RemediationPolicy` custom resource definition (CRD) enforces policy at the platform level. The policy engine, combined with dynamically scoped RBAC, sandboxed execution, and mandatory human approval, provides multiple layers of defense.
