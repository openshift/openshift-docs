= Recover from incomplete agentic remediations

[role="_abstract"]
Recover from incomplete agentic remediations by inspecting partially applied changes, suspending agent operations to prevent further modifications, and reporting issues through the available feedback channels. Rollback is not guaranteed if execution is stopped while in progress.
