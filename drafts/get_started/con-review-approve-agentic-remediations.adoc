= Review and approve agentic remediations

[role="_abstract"]
Use the Agentic Runs view in the {ocp-product-title} web console or the `oc agentic` CLI to monitor agent-initiated investigations, review proposed diagnoses and remediation plans, and approve or reject remediation actions before execution.
