= Removing the admin user
include::modules/common-attributes.adoc[]
:context: remove-admin-user

toc::[]

[role="_abstract"]
{rh-rhacs-first} creates an administrator account, `admin`, during the installation process that you can use to log in with a user name and password.
The password is dynamically generated unless you specifically override it and is unique to your {product-title-short} instance.

In production environments, create an authentication provider and remove the `admin` user.
