include::modules/common-attributes.adoc[]
:context: manage-role-based-access-control

toc::[]

[role="_abstract"]
{rh-rhacs-first} includes role-based access control (RBAC) that you can use to configure roles and grant various levels of access to {product-title} for different users.

Beginning with version 3.63, {product-title-short} includes a scoped access control feature. You can configure fine-grained and specific sets of permissions that define how a given {product-title-short} user or a group of users can interact with {product-title-short}, the resources they can access, and the actions they can perform.
