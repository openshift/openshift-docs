= Scanning {op-system} node hosts
include::modules/common-attributes.adoc[]
:context: scan-rhcos-node-host

toc::[]

[role="_abstract"]
For {ocp}, {op-system-first} is the only supported operating system for control plane. For node hosts, {ocp} supports both {op-system} and {op-system-base-full}. With {rh-rhacs-first}, you can scan {op-system} nodes for vulnerabilities and detect potential security threats.
