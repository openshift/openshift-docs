= Configuring Tekton Results for multicluster environments
include::_attributes/common-attributes.adoc[]
:context: configuring-tekton-results-multicluster

toc::[]

[role="_abstract"]
You can configure Tekton Results in a multicluster {pipelines-title} environment to store pipeline run records and logs centrally on the hub cluster. This configuration enables you to query pipeline run records and logs from a single location, even when pipeline runs execute across multiple spoke clusters.

Tekton Results provides two main capabilities:

* *Records storage*: Stores pipeline run and task run metadata in a PostgreSQL database
* *Logs storage*: Stores pipeline run logs in a LokiStack deployment

Both capabilities are optional and can be configured independently based on your requirements.

:FeatureName: Multicluster support for {pipelines-shortname}
