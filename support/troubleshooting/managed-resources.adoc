== Overview

{product-title} resources are managed or protected by the Service Reliability Engineering Platform (SRE-P) Team. Customers should not attempt to modify these resources because doing so can lead to cluster instability.

If you are experiencing issues, see more information here: xref:../../rosa_architecture/rosa_policy_service_definition/rosa-policy-responsibility-matrix.adoc#sd-managed-resources[Red Hat Managed resources]
