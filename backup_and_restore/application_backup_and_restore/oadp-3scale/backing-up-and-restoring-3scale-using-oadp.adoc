With Red Hat 3scale API Management (APIM), you can manage your APIs for internal or external users. Share, secure, distribute, control, and monetize your APIs on an infrastructure platform built with performance, customer control, and future growth in mind.
3scale components can be deployed on-premise, in the cloud, as a managed service, or in any combination based on your requirement.
This document is using the non-service affecting approach to backup and restore 3scale on cluster storage using the {oadp-first} Operator. Here, either the source or destination cluster are the same or if not, both the clusters are using the same custom domain.
