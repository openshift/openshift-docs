= Using the OADP CLI plugin
include::_attributes/common-attributes.adoc[]
:context: oadp-cli-plugin

toc::[]

[role="_abstract"]
The {oadp-first} command-line interface (CLI) is a plugin for the OpenShift CLI (`oc`) that provides a kubectl-native interface for managing backup and restore operations on an {product-title} cluster. The plugin is available as `oc oadp` and supports both cluster administrator and non-administrator workflows.
