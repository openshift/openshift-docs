= {oadp-short} virtual machine file restore
:context: oadp-vmfr

toc::[]

[role="_abstract"]
Use {oadp-short} virtual machine file restore (VMFR) to recover individual files from `kubevirt` VM backups without restoring the entire VM. You can browse, compare, and download files from multiple backups simultaneously.
