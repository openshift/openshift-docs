= Configure and use {oadp-short} VMFR
:context: oadp-using-vmfr

toc::[]

[role="_abstract"]
Use {oadp-short} virtual machine (VM) file restore (VMFR) to discover VM backups, restore files, and access restored files through a web browser or SSH-based tools. You can recover individual files from VM backups without restoring the entire VM.
