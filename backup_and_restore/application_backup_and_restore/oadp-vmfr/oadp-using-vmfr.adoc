= Configuring and using {oadp-short} VMFR
:context: oadp-using-vmfr

toc::[]

[role="_abstract"]
Use {oadp-short} virtual machine file restore (VMFR) to discover virtual machine backups, restore files, and access restored files through a web browser or SSH-based tools. This helps you to recover individual files from VM backups without restoring the entire virtual machine.
