As a cluster administrator, following are some backup and restore scenarios where you enable {oadp-short} Self-Service and configure the `DataProtectionApplication` accordingly:

* Filesystem backup and restore by using Kopia.
* Data Mover backup and restore.
* Container Storage Interface (CSI) backup and restore.
