= Configuring the OpenShift API for Data Protection with more than one BackupStorageLocation
include::_attributes/common-attributes.adoc[]
:context: configuring-oadp-multiple-bsl
:configuring-oadp-multiple-bsl:


toc::[]


You can configure one or more Backup Storage Location (BSL) in the Data Protection Application (DPA) and can select the location at backup creation time, in which the backup should be stored. With this configuration, you can store your backups:

* To different regions.
* To a different storage provider.

OADP supports multiple credentials for configuring more than one BSL, allowing you to specify the credentials to use with any BSL.
