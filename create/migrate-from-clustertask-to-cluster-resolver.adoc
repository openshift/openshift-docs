= Migrate from ClusterTask to the cluster resolver
:context: migrate-from-clustertask-to-cluster-resolver

toc::[]

[role="_abstract"]
`ClusterTask` resources are no longer supported in {pipelines-title}. Pipelines that reference `kind: ClusterTask` must be updated to use the cluster resolver for task resolution.

Migrate from `ClusterTask` resources to the cluster resolver and update pipeline definitions to access tasks from the {pipelines-shortname} installation namespace or other namespaces.
