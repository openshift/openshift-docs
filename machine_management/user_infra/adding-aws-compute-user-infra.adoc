= Adding compute machines to AWS by using CloudFormation templates
include::_attributes/common-attributes.adoc[]
:context: adding-aws-compute-user-infra

toc::[]

[role="_abstract"]
To scale your {product-title} cluster on {aws-first} after user-provisioned installation, you can add compute machines by creating CloudFormation stacks from your installation templates. You can then approve certificate signing requests so the new nodes join the cluster.

[id="prerequisites_adding-aws-compute-user-infra"]
== Prerequisites

* You installed your cluster on AWS by using the provided xref:../../installing/installing_aws/upi/installing-aws-user-infra.adoc#installing-aws-user-infra[AWS CloudFormation templates].
* You have the JSON file and CloudFormation template that you used to create the compute machines during cluster installation. If you do not have these files, you must recreate them by following the instructions in the xref:../../installing/installing_aws/upi/installing-aws-user-infra.adoc#installing-aws-user-infra[installation procedure].
