= Control plane configuration options for {azure-full}
:context: cpmso-config-options-azure

toc::[]

[role="_abstract"]
You can change the configuration of your {azure-first} control plane machines by updating values in the control plane machine set.
You might make updates to reflect changes to your infrastructure or environment.

When you save an update to the control plane machine set, the Control Plane Machine Set Operator updates the control plane machines according to your configured update strategy.
For more information, see "Updating the control plane configuration".
