= Uninstalling {builds-shortname}
:context: uninstalling-openshift-builds

toc::[]

[role="_abstract"]
As a cluster administrator, you can uninstall the {builds-operator}. Uninstalling the Operator does not automatically remove the custom resources (CRs) and {builds-shortname} components.

To uninstall the {builds-operator}, perform the following tasks:

. Delete the custom resources (CRs) that were created by default when you installed the Operator.
+
[NOTE]
====
If you uninstall the Operator without removing the CRs, you cannot remove them later.
====

. Uninstall the Operator.
