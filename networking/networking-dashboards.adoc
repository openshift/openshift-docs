Some networking metrics are viewable in dashboards within the {product-title} web console, under *Observe* -> *Dashboards*.

[id="network-observability-operator-operator-dashboards"]
== Network Observability Operator
If you have the Network Observability Operator installed, you can view network traffic metrics dashboards by selecting the *Netobserv* dashboard from the *Dashboards* dropdown. For more information about metrics available in this *Dashboard* view, see xref:../../network_observability/network-observability-overview.adoc#network-observability-dashboards[Network Observability metrics dashboards]
