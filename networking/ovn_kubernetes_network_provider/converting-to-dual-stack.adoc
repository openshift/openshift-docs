= Converting to IPv4/IPv6 dual-stack networking
include::_attributes/common-attributes.adoc[]
:context: converting-to-dual-stack

toc::[]

[role="_abstract"]
To enable IPv4 and IPv6 on your cluster network in {product-title}, you can convert a single-stack cluster to dual-stack networking. After conversion, new and existing pods can use both address families when you re-create workloads as needed.

[IMPORTANT]
====
When using dual-stack networking where IPv6 is required, you cannot use IPv4-mapped IPv6 addresses, such as `::FFFF:198.51.100.1`.
====
