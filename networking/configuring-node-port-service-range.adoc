= Configuring the node port service range
include::_attributes/common-attributes.adoc[]
:context: configuring-node-port-service-range

toc::[]

As a cluster administrator, you can expand the available node port range. If your cluster uses of a large number of node ports, you might need to increase the number of available ports.

The `30000-32767` port range is the default port range on {product-title}. You can extend the range on either side but you must preserve `30000-32767` within your new port range.

Expanding the node port range for a cluster is not limited to a one-time-only operation.

[IMPORTANT]
====
As a cluster administrator, ensure that the new port range does not conflict with a port that is already assigned to a process on a host or a pod that is configured for host networking. 
====
