The Network Observability Operator default installation and enablement makes the magic of observing your network as you set things up, possible.

:FeatureName: Network Observability Operator Day 0
include::snippets/technology-preview.adoc[]

When you create a new {product-title} cluster, the Network Observability Operator and its dependencies are automatically installed and configured. This provides immediate access to network traffic monitoring capabilities through the {product-title} web console.

The Network Observability Operator documentation is available as a separate documentation set.
