= Tune SR-IOV network interface parameters
include::_attributes/common-attributes.adoc[]
:context: configuring-sysctl-interface-sriov-device

toc::[]

[role="_abstract"]
Fine-tune network performance by configuring interface-level sysctls, promiscuous mode, all-multicast mode, MTU, and MAC address for SR-IOV network devices.

Use the tuning Container Network Interface (CNI) meta plugin to change interface-level network sysctls and interface attributes for workloads that require specific network tuning.

[id="configuring-interface-sysctl-sriov-device-prerequisites"]
== Prerequisites

* You installed the SR-IOV Network Operator. For more information, see xref:../../networking/networking_operators/sr-iov-operator/installing-sriov-operator.adoc#installing-sriov-operator[Installing the SR-IOV Network Operator]. Verify by running:
+
[source,terminal]
----
$ oc get csv -n openshift-sriov-network-operator --no-headers | grep sriov
----
+
The output shows the SR-IOV Network Operator with phase `Succeeded`.
