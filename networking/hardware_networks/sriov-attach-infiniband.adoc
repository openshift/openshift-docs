= Attach workloads to SR-IOV InfiniBand networks
include::_attributes/common-attributes.adoc[]
:context: configuring-sriov-ib-attach

toc::[]

[role="_abstract"]
Connect pods to Single Root I/O Virtualization (SR-IOV) InfiniBand networks for high-performance computing and storage networking that requires Remote Direct Memory Access (RDMA).

[NOTE]
====
The process for configuring InfiniBand attachments is nearly identical to Ethernet attachments. The only difference is using the `SriovIBNetwork` object instead of `SriovNetwork`.

For conceptual information about IPAM configuration, dual-stack IP addresses, and Whereabouts that applies to both protocols, see xref:../../networking/hardware_networks/sriov-attach-ethernet.adoc#configuring-sriov-net-attach[Attach workloads to SR-IOV Ethernet networks].
====

[id="configuring-sriov-ib-attach-prerequisites"]
== Prerequisites

* You installed the SR-IOV Network Operator. For more information, see xref:../../networking/networking_operators/sr-iov-operator/installing-sriov-operator.adoc#installing-sriov-operator[Installing the SR-IOV Network Operator].
* You configured an SR-IOV network device. For more information, see xref:../../networking/hardware_networks/sriov-configure-device.adoc#configuring-sriov-device[Configure SR-IOV network devices].
