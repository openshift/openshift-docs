= Enable RDMA for low-latency communication
include::_attributes/common-attributes.adoc[]
:context: configuring-sriov-rdma-cni

toc::[]

[role="_abstract"]
Enable Remote Direct Memory Access (RDMA) on SR-IOV networks to achieve low-latency communication between containers by bypassing the operating system network stack.

RDMA allows direct memory access between two systems without involving the operating system of either system. When you combine RDMA with SR-IOV, you provide a mechanism to expose hardware counters of Mellanox Ethernet devices for use inside Data Plane Development Kit (DPDK) applications.

[id="configuring-sriov-rdma-cni-prerequisites"]
== Prerequisites

* You installed the SR-IOV Network Operator. For more information, see xref:../../networking/networking_operators/sr-iov-operator/installing-sriov-operator.adoc#installing-sriov-operator[Installing the SR-IOV Network Operator].
