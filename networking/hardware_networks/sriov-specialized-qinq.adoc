= Enable VLAN stacking with QinQ
include::_attributes/common-attributes.adoc[]
:context: configuring-qinq-support

toc::[]

[role="_abstract"]
Run multiple VLANs over a single SR-IOV interface by enabling QinQ (802.1Q-in-802.1Q) VLAN stacking, also known as double VLAN tagging.

QinQ, formally known as 802.1Q-in-802.1Q, is a networking technique defined by IEEE 802.1ad. IEEE 802.1ad extends the IEEE 802.1Q-1998 standard and enriches VLAN capabilities by introducing an additional 802.1Q tag to packets already tagged with 802.1Q.

[id="configuring-qinq-support-prerequisites"]
== Prerequisites
