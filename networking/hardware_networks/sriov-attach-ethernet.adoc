= Attach workloads to SR-IOV Ethernet networks
include::_attributes/common-attributes.adoc[]
:context: configuring-sriov-net-attach

toc::[]

[role="_abstract"]
Connect pods to Single Root I/O Virtualization (SR-IOV) Ethernet networks to enable high-performance networking with direct hardware access.

[id="configuring-sriov-net-attach-prerequisites"]
== Prerequisites

* You installed the SR-IOV Network Operator. For more information, see xref:../../networking/networking_operators/sr-iov-operator/installing-sriov-operator.adoc#installing-sriov-operator[Installing the SR-IOV Network Operator]. Verify by running:
+
[source,terminal]
----
$ oc get csv -n openshift-sriov-network-operator --no-headers | grep sriov
----
+
The output shows the SR-IOV Network Operator with phase `Succeeded`.
* You configured an SR-IOV network device. For more information, see xref:../../networking/hardware_networks/sriov-configure-device.adoc#configuring-sriov-device[Configure SR-IOV network devices]. Verify by running:
+
[source,terminal]
----
$ oc get sriovnetworknodepolicies -n openshift-sriov-network-operator
----
+
The output shows one or more SR-IOV network node policies.
