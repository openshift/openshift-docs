= Configure IP address management for SR-IOV networks
include::_attributes/common-attributes.adoc[]
:context: configuring-sriov-ipam

toc::[]

[role="_abstract"]
Configure IP address assignment for SR-IOV secondary networks using static IPs, DHCP, or Whereabouts. This guide provides step-by-step procedures for each method.

[id="configuring-sriov-ipam-prerequisites"]
== Prerequisites

* You installed the SR-IOV Network Operator. For more information, see xref:../../networking/networking_operators/sr-iov-operator/installing-sriov-operator.adoc#installing-sriov-operator[Installing the SR-IOV Network Operator].
* You configured at least one SR-IOV network device. For more information, see xref:../../networking/hardware_networks/sriov-configure-device.adoc#configuring-sriov-device[Configure SR-IOV network devices].
* You understand the differences between IPAM methods. For more information, see xref:../../networking/hardware_networks/sriov-concept-ipam.adoc#sriov-concept-ipam[Understand IP address management for SR-IOV networks].
