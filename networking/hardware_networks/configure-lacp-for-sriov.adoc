= Switch failure detection for bonded SR-IOV networks
include::_attributes/common-attributes.adoc[]
:context: sriov-lacp-sriov

toc::[]

For workloads using bonded SR-IOV virtual functions (VFs), despite an upstream switch failure, an underlying physical function (PF) might still report an `up` state. This creates a silent failure, as attached VFs remain up and pods continue to send traffic to a dead endpoint, causing packet loss.

The PF Status Relay Operator enables {product-title} to use Link Aggregation Control Protocol (LACP) as an active health check on PFs. When a switch failure occurs, the operator detects it and propagates the PF state to the attached VFs, allowing pods to automatically switch to a backup network path and maintain high availability.
