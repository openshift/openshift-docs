= Configuring the network at installation time
include::modules/common-attributes.adoc[]
:context: configuring-network-operator

toc::[]

The Network Operator installs and upgrades the networking components on an
{product-title} cluster. If you want to customize the network configuration, you
must modify the settings before installing the cluster.

The Network Operator is configured from two objects: the _Cluster configuration_
and the _Operator configuration_.

* The Cluster configuration object is defined by the
'Network.config.openshift.io' Custom Resource Definition (CRD).
* The Operator configuration object is defined by the
'NetworkConfig.networkoperator.openshift.io' CRD. {product-title} generates the
'Operator configuration' from the Cluster configuration.

[IMPORTANT]
====
You cannot modify your network settings after you deploy the cluster.
====
