= Deploying isolated {mcpg} instances
:context: mcp-gateway-isolated-deployment

toc::[]

[role="_abstract"]
You can deploy multiple {mcpg} instances within a single cluster to isolate traffic between teams. Each instance uses its own `MCPGatewayExtension` custom resource (CR) to manage its associated MCP servers independently.
