= Configuring {builds-shortname} 
include::_attributes/common-attributes.adoc[]
:context: configuring-openshift-builds

toc::[]


A `Build` custom resource (CR) helps you to define the source, build strategy, parameter values, builder or docker file, output, retention parameters, and volumes to configure a build. A `Build` resource is available for use within a namespace. 

For configuring a build, create a `Build` resource YAML file and apply it to the {ocp-product-title} cluster.
