= Viewing cluster dashboards
include::modules/common-attributes.adoc[]

toc::[]

The *Logging/Elasticsearch Nodes* or *Openshift Logging* dashboards in the {product-title} web console show in-depth details about the Elasticsearch instance and individual Elasticsearch nodes, which provide information you can use to prevent and diagnose problems.

The *OpenShift Logging* dashboard contains charts that show details about your Elasticsearch instance at a cluster-level, including cluster resources, garbage collections, shards in the cluster, and Fluentd statistics.

The *Logging/Elasticsearch Nodes* databse contains charts that show details about your Elasticsearch cluster, many at node-level, including details on indexing, shards, resources, and so forth. 

[NOTE]
====
For more detailed data, click the *Grafana UI* link to launch the Grafana dashboard. Grafana is shipped with xref:../monitoring/cluster_monitoring/about-cluster-monitoring.adoc#about-cluster-monitoring[{product-title} cluster monitoring].
====
