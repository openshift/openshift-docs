== Verification

You can verify the installation by listing cluster service versions (CSVs) and pods and observing the results.

. To verify the Operator installation, run the following command:
+
[source,terminal]
----
$ oc get csv --all-namespaces
----
+
There should be an OpenShift Elasticsearch Operator in each namespace. The version number might be different than shown.
+
.Example output
[source,terminal]
----

. To verify that the {logging} components were created, run the following command:
+
[source,terminal]
----
$ oc get pods -n openshift-logging
----
+
