= Installing Logging
include::_attributes/common-attributes.adoc[]
include::_attributes/attributes-openshift-dedicated.adoc[]

toc::[]

You can deploy the {logging-title} by installing the Loki Operator and the Red{nbsp}Hat OpenShift Logging Operator. The Loki Operator creates and manages log storage. The Red{nbsp}Hat OpenShift Logging Operator creates and manages the components of the logging stack.
