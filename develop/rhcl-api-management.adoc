= Create, share, and consume APIs across teams
:context: rhcl-api-management

toc::[]

[role="_abstract"]
When you need to make internal APIs discoverable and control who can access them, you can publish API products in a self-service catalog where developers request access and receive credentials. API management is available when you are using the {prodname} {ocp} web console plugin on {ocp} 4.20 or newer.

:FeatureName: API management with the OpenShift web console
