= Enabling mTLS for {ProductShortName} integration
:context: serverless-enabling-mtls-ossm
include::modules/common-attributes.adoc[]

toc::[]

Enabling mTLS for all Knative services and Knative `data-plane` pods ensures that all traffic sent to and between services them is encrypted. mTLS functionality can be enabled by cluster administrators by completing the following procedures.

[id="prerequisites_serverless-enabling-mtls-ossm"]
== Prerequisites

* Install {ProductName} version xref:../../service_mesh/v2x/installing-ossm.adoc#installing-ossm[2.x].
+
[IMPORTANT]
====
mTLS functionality is only supported by {ServerlessProductName} when used with {ProductName} v2.0.5 and newer.
====
* Enable the xref:../../serverless/service_mesh/serverless-enabling-ossm.adoc#serverless-ossm-setup-without-kourier_serverless-enabling-ossm[{ProductName} with {ServerlessProductName} integration].
