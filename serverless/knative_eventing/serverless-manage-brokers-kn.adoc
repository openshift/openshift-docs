= Creating and managing Brokers using the kn CLI
include::modules/common-attributes.adoc[]
include::modules/serverless-document-attributes.adoc[]
:context: serverless-using-brokers

toc::[]

You can use the xref:../../serverless/installing_serverless/installing-kn.adoc#installing-kn[`kn` CLI] to create, list, describe, and delete brokers.

[NOTE]
====
The following commands refer to a broker with name `<NAME>` in the current namespace.
To use `kn` commands with a broker in a namespace other than the current namespace, you must specify the namespace in the command.

.Example

[source,terminal]
----
$ kn broker describe <NAME> --namespace <NAMESPACE>
----
====

[id="serverless-manage-brokers-kn-create"]
.Creating a broker

* Create a broker by entering the following command:
+

[source,terminal]
----
$ kn broker create <NAME>
----

[id="serverless-manage-brokers-kn-describe"]
.Describe a broker

* Describe a broker by entering the following command:
+

[source,terminal]
----
$ kn broker describe <NAME>
----

[id="serverless-manage-brokers-kn-list"]
.List existing brokers

* List all existing brokers by entering the following command:
+

[source,terminal]
----
$ kn broker list <NAME>
----

[id="serverless-manage-brokers-kn-delete"]
.Deleting a broker

* Delete a broker by entering the following command:
+

[source,terminal]
----
$ kn broker delete <NAME>
----
