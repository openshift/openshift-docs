This guide provides information about how you can configure your OpenShift Serverless deployment to integrate with Red Hat AMQ Streams.

[id="serverless-amq-streams-prerequisites"]
== Prerequisites
* You need to have link:https://access.redhat.com/documentation/en-us/red_hat_amq/7.7/html/using_amq_streams_on_openshift/getting-started-str[Red Hat AMQ Streams] installed on your {product-title} cluster.
* You need to have link:https://access.redhat.com/documentation/en-us/red_hat_amq/7.7/html/using_amq_streams_on_openshift/getting-started-str#deploy-create-cluster_str[created a Kafka cluster] and any other components required for a basic Kafka deployment using AMQ Streams on {product-title}.

* You will need cluster administrator permissions on an {product-title} cluster to install the *Red Hat Integration - AMQ Streams* Operator.
