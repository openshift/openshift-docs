This guide provides information about how cluster administrators can install and configure Red Hat AMQ Streams to integrate with an OpenShift Serverless deployment.

.Prerequisites

* You will need cluster administrator permissions on an {product-title} cluster to install the *Red Hat Integration - AMQ Streams* Operator.
* You must have the {ServerlessOperatorName}, Knative Serving, and Knative Eventing installed on your cluster.
* You must have access to the {product-title} web console.
