= Connecting channels to sinks
:context: connecting-channels-sinks
toc::[]

After events have been sent to a channel from an event source or producer, these events can be sent to an event sink by using a subscription. Subscriptions are created by configuring a `Subscription` object, which specifies the channel and the sink (also known as a _subscriber_) to deliver events to.
