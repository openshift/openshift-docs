== Overview
The OpenShift API exposes operations for managing an OpenShift cluster, including security and user management, application deployments, image and source builds, HTTP(s) routing, and project management.

=== Version information
Version: v1

=== URI scheme
Host: 127.0.0.1:8443
BasePath: /
Schemes: HTTPS
