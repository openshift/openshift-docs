Description::
  LocalResourceAccessReview is a means to request a list of which users and groups are authorized to perform the action specified by spec in a particular namespace

Type::
  `object`

Required::
  - `namespace`
  - `verb`
  - `resourceAPIGroup`
  - `resourceAPIVersion`
  - `resource`
  - `resourceName`
  - `path`
  - `isNonResourceURL`
