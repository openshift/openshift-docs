Description::
  SubjectAccessReview is an object for requesting information about whether a user or group can perform an action

Type::
  `object`

Required::
  - `namespace`
  - `verb`
  - `resourceAPIGroup`
  - `resourceAPIVersion`
  - `resource`
  - `resourceName`
  - `path`
  - `isNonResourceURL`
  - `user`
  - `groups`
  - `scopes`
