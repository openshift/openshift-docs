= Understanding persistent storage
include::_attributes/common-attributes.adoc[]
:context: understanding-persistent-storage

toc::[]

[role="_abstract"]
Persistent storage decouples data from pod lifecycles, allowing stateful applications to retain data across restarts and failures. Persistent volumes (PVs) and persistent volume claims (PVCs) enable admins to provision storage and developers to request it without infrastructure knowledge.
