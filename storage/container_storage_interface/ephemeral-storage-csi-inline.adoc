= CSI inline ephemeral volumes
include::_attributes/common-attributes.adoc[]
:context: ephemeral-storage-csi-inline

toc::[]

When deploying a pod, the Container Storage Interface (CSI) inline ephemeral volumes enable you to define a Pod spec that creates inline ephemeral volumes and deletes them upon the pod's destruction.

This feature is only available with supported Container Storage Interface (CSI) drivers.
