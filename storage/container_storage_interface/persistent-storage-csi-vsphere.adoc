= VMware vSphere CSI Driver Operator
include::_attributes/common-attributes.adoc[]
:context: persistent-storage-csi-vsphere

toc::[]

[role="_abstract"]
You can provision and manage vSphere storage in {product-title} by using the vSphere Container Storage Interface (CSI) Driver Operator and driver, which provide dynamic volume provisioning and eliminate the need to pre-provision storage.
