[role="_abstract"]
You can use {product-title-short} with ServiceNow by using the following methods:

* Use the Red{nbsp}Hat Security Container Vulnerability Response Integration to pull data from your {product-title-short} instances to the ServiceNow Vulnerability Response application. See link:https://store.servicenow.com/store/app/edea7344476072502ec7c1c4f16d4343#summary[RHACS Vulnerability Response] on the ServiceNow marketplace for more information.
* Configure {product-title} to send events to ServiceNow by configuring a generic webhook integration in {product-title-short}.

To integrate {product-title-short} with ServiceNow by using a generic webhoook integration:

. In ServiceNow, configure a REST API endpoint to use in {product-title-short}. For more information that includes steps for ServiceNow configuration, see link:https://www.redhat.com/en/blog/how-integrate-red-hat-advanced-cluster-security-kubernetes-servicenow[How to integrate {product-title} with ServiceNow].
. In {product-title-short}:
.. Configure the generic webhook.
.. Identify policies for which you want to send notifications, and update the notification settings for those policies.
