= Integrating with ServiceNow

:context: integrate-with-servicenow

toc::[]

[role="_abstract"]
You can use {rh-rhacs-first} with ServiceNow by using one of the following methods:

* Use the Red{nbsp}Hat Security Container Vulnerability Response Integration to pull data from your {product-title-short} instances to the ServiceNow Vulnerability Response application. For more information, see link:https://store.servicenow.com/store/app/edea7344476072502ec7c1c4f16d4343[RHACS Vulnerability Response] on the ServiceNow marketplace.
* Configure {product-title} to send events to ServiceNow by configuring a generic webhook integration in {product-title-short}.

To integrate {product-title-short} with ServiceNow by using a generic webhook integration, complete the following steps:

. In ServiceNow, configure a REST API endpoint to use in {product-title-short}. For more information that includes steps for ServiceNow configuration, see link:https://www.redhat.com/en/blog/how-integrate-red-hat-advanced-cluster-security-kubernetes-servicenow[How to integrate {product-title} with ServiceNow].
. In {product-title-short}, complete these steps:
.. Configure the generic webhook.
.. Identify policies for which you want to send notifications, and update the notification settings for those policies.
