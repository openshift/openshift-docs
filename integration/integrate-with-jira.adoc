If you are using Jira, you can forward alerts from {product-title} to Jira.

The following steps represent a high-level workflow for integrating {product-title} with Jira:

. Setup a user in Jira.
. Use the Jira URL, username, and password to integrate Jira with {product-title}.
. Identify policies for which you want to send notifications, and update the notification settings for those policies.
