= Integrating with email
include::modules/common-attributes.adoc[]
:context: integrate-with-email

toc::[]

[role="_abstract"]
With {rh-rhacs-first}, you can configure your existing email provider and send notifications about policy violations.
Or, if you are using {rh-rhacscs-first}, you can use your existing email provider or the built-in email notifier to send email notifications.

You can use the `Default recipient` field to forward alerts from {product-title-short} and the {product-title-managed-short} to an email address.
Otherwise, you can use annotations to define an audience and notify them about policy violations associated with a specific deployment or namespace.

The following rules govern how {product-title-short} determines the recipient of an email notification:

* If the deployment has an annotation key, the annotation's value overrides the default value.
* If the namespace has an annotation key, the namespace's value overrides the default value.
* If a deployment has an annotation key and a defined audience, {product-title-short} sends an email to the audience specified in the key.
* If a deployment does not have an annotation key, {product-title-short} checks the namespace for an annotation key and sends an email to the specified audience.
* If no annotation keys exist, {product-title-short} sends an email to the default recipient.

== Integrating with email on {product-title-short}

You can use email as a notification method by forwarding alerts from {product-title-short}.

== Integrating with email on {product-title-managed-short}

You can use your existing email provider or the built-in email notifier in {product-title-managed-short} to send email alerts about policy violations.

* To use your own email provider, you must configure the email provider as described in the section xref:../integration/integrate-with-email.adoc#configure-acs-email_integrate-with-email[Configuring the email plugin].
* To use the built-in email notifier, you must configure the {product-title-managed-short} email plugin.
