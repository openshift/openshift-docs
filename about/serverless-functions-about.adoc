= OpenShift Serverless functions overview
include::_attributes/common-attributes.adoc[]
:context: serverless-functions-about

toc::[]

[role="_abstract"]
Developers use {FunctionsProductName} to create and deploy stateless, event-driven functions as Knative Services on {ocp-product-title}. The Knative `kn` CLI includes the `kn func` plugin. You can use the `kn func` CLI to create, build, and deploy container images as Knative Services on the cluster.

{FunctionsProductName} provides templates for creating basic functions in Quarkus, Node.js, and TypeScript runtimes.
