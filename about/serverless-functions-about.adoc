= About {FunctionsProductName}
:context: serverless-functions-about

toc::[]

{FunctionsProductName} enables developers to create and deploy stateless, event-driven functions as a Knative service on {ocp-product-title}. The `kn func` CLI is provided as a plugin for the Knative `kn` CLI. You can use the `kn func` CLI to create, build, and deploy the container image as a Knative service on the cluster.
