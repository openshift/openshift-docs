{ServerlessProductName} provides Kubernetes-native building blocks for creating and deploying serverless, event-driven applications on {ocp-product-title}. These applications scale up and down (to zero) on-demand and respond to events from several sources. {ServerlessProductName} uses the open source Knative project to deliver portability and consistency across hybrid and multicloud environments.
