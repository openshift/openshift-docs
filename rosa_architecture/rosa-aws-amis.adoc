= AWS AMIs for {product-title}
include::_attributes/attributes-openshift-dedicated.adoc[]
:context: rosa-aws-ami

toc::[]

The following list contains the Amazon Machine Images (AMIs) for {hcp-title-first} clusters. For more information, see link:https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/AMIs.html#ami-using[Amazon Machine Images (AMI)] in the AWS documentation.

The AMIs are stored in a Red Hat AWS account. Your customer account needs to create a trust relationship with all of the AMIs you want to use. You could trust all AMIs in that Red Hat account or manually add each AMI to your allowlist. To allow all of the AMIs in the Red Hat AWS account, you can add the following to your AWS permissions:

[source,terminal]
----
"Condition": {   
    "StringEquals": {  
        "ec2:Owner": [
            <Red_Hat_accounts>
        ]
    }
}
----
