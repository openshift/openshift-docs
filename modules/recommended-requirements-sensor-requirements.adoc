= Memory and CPU requirements

[role="_abstract"]
The following table lists the minimum memory and CPU values required to run Sensor on a secured cluster.

|===
| Deployments | CPU | Memory

| < 25,000
| 2 cores
| 10 GiB

| < 50,000
| 2 cores
| 20 GiB
|===
