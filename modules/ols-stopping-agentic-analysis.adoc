= Stopping an in-progress analysis

[role="_abstract"]
You can stop an agent investigation while it is actively analyzing an alert.

.Procedure

. In the *Agentic Runs* view, locate the run in the *Analyzing* state.

. Click the red stop button to halt the agent's investigation.

[NOTE]
====
Stopping an analysis does not affect your cluster. No changes are made to cluster resources during the analysis phase. No remediation is proposed for a stopped analysis.
====
