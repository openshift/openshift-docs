= Strategy definition

[role="_abstract"]
Configure the strategy for a build in the `Build` CR.

{builds-shortname} supports the following build strategies:
* `buildah`
* `source-to-image`
* `buildpacks`

To configure a build strategy, define the `spec.strategy.name` and `spec.strategy.kind` fields in the `Build` CR, as shown in the following example:

[source,yaml]
----
apiVersion: shipwright.io/v1beta1
kind: Build
metadata:
  name: buildah-build
spec:
  strategy:
    name: buildah
    kind: ClusterBuildStrategy
----
