= Disabling Dex

Dex is installed by default for all the Argo CD instances created by the Operator. You can disable Dex.

.Procedure

* Set the environmental variable `DISABLE_DEX` to true in the *YAML* resource of the Operator:
+
[source,yaml]
----
  spec:
  config:
    env:
    - name: DISABLE_DEX
      value: "true"
----
