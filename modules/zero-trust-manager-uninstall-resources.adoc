= Uninstalling {zero-trust-full} resources by using the CLI

Once you have uninstalled the {zero-trust-full}, you have the option to delete its associated resources from your cluster.

.Prerequisites

* You have access to the cluster with `cluster-admin` privileges.

.Procedure

. Uninstall the Operator objects by running each of the following commands:
+
[source,terminal]
----
$ oc delete SpireServerConfig cluster
$ oc delete SpireAgentConfig cluster
$ oc delete SpiffeCSIDriverConfig cluster
$ oc delete SpireOIDCDiscoveryProviderConfig cluster
----

. Uninstall the Persistent Volume Claim (PVC) and services by running each of the following commands:
+
[source,terminal]
----
$ oc delete pvc -l=app.kubernetes.io/managed-by=zero-trust-workload-identity-manager
$ oc delete service -l=app.kubernetes.io/managed-by=zero-trust-workload-identity-manager
----

. Uninstall the namespace by running the following command:
+
[source,terminal]
----
$ oc delete ns zero-trust-workload-identity-manager
----

. Uninstall the cluster-wide role-based access control (RBAC) by running each of the following commands:
+
[source,terminal]
----
$ oc delete clusterrolebinding -l=app.kubernetes.io/managed-by=zero-trust-workload-identity-manager
$ oc delete clusterrole -l=app.kubernetes.io/managed-by=zero-trust-workload-identity-manager
----

. Uninstall the Custom Resource Definitions (CRDs) by running each of the following commands:
+
[source,terminal]
----
$ oc delete crd spireserverconfigs.operator.openshift.io
$ oc delete crd spireagentconfigs.operator.openshift.io
$ oc delete crd spiffecsidriverconfigs.operator.openshift.io
$ oc delete crd spireoidcdiscoveryproviderconfigs.operator.openshift.io
$ oc delete crd clusterfederatedtrustdomains.spire.spiffe.io
$ oc delete crd clusterspiffeids.spire.spiffe.io
$ oc delete crd clusterstaticentries.spire.spiffe.io
----

.Verification

To verify that the resources have been deleted, replace each `oc delete` command with `oc get`, and then run the command. If no resources are returned, the deletion was successful.
