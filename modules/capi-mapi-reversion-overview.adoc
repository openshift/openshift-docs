= Converting Cluster API resources to Machine API resources

On clusters that support converting between Machine API and Cluster API resources, the two-way synchronization controller supports converting a Cluster API resource to a Machine API resource.

You can convert resources that you originally converted from the Machine API to the Cluster API, or resources that you created as Cluster API resources initially.
Converting an original Machine API resource to a Cluster API resource and then converting it back provides an opportunity to verify that the migration process works as expected.

[NOTE]
====
You can only convert some resources on supported infrastructure types.
====
