= Configure Build and BuildRun settings

[role="_abstract"]
The `Build` and `BuildRun` custom resources provide configurable fields that control how container images are built and executed.
Understanding these fields helps you create build configurations that match your application requirements and deployment workflows.

The `Build` resource defines the specification for your builds, including source location, strategy, output destination, and default runtime settings.
The `BuildRun` resource triggers a build execution and can override specific `Build` settings for individual runs.

The build controller processes these resources and applies system-wide settings that affect all builds in the cluster.
By using the parameter definitions, you can create reusable build configurations that accept different values at runtime.
