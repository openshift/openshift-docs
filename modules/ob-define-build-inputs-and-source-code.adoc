= Define build inputs and source code

[role="_abstract"]
Configure the source code location, build strategy, and runtime settings for your builds. The `Build` custom resource (CR) provides fields to specify where your source code resides, which build strategy to use, and how to configure the build environment including container files, service accounts, environment variables, and volumes.
