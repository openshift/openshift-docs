= Upgrade changes from version 2.5 to version 2.6

//Jaeger disabled by default goes in "Upgrading"
== {JaegerName} default setting change

This release disables {JaegerName} by default.

{SMProductName} 2.6 is the last release that includes support for {JaegerName} and {es-op}. Both {JaegerShortNamee} and {es-op} will be removed in the next release. If you are currently using {JaegerShortName} and {es-op}, you need to switch to {TempoName} and {OTELName}.

== Envoy sidecar container default setting change

To enhance pod startup times, Istio now includes a `startupProbe` in sidecar containers by default. The pod's readiness probes will not start until the Envoy sidecar has started.

You can disable the startup probe:

[source,yaml]
----
readinessInitialDelaySeconds: 1
readinessPeriodSeconds: 2
readinessFailureThreshold: 30
startupProbe:
enabled: false
----
