= Configuring ingress cluster traffic on AWS using a Network Load Balancer

[role="_abstract"]
To enable high-performance communication between external services and your {product-title} cluster, configure an {aws-full} Network Load Balancer (NLB). You can set up an NLB on a new or existing {aws-short} cluster to manage ingress traffic with low latency.
