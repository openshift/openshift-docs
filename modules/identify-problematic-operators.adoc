= Identifying problematic Operators

The `oc adm upgrade status` command returns detailed information about the status of your cluster update, including which Operators are causing issues during a failed cluster update.

.Prerequisites
* You have access to the cluster as a user with the `cluster-admin` role.
* You have installed the {oc-first}.

.Procedure

* Determine which Operator could be causing issues with the following command:
+
[source,terminal]
----
$ oc adm upgrade status
----
+
