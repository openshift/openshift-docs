[id="shp-build-delete_{context}"]
= shp build delete

Delete a build.

----
$ shp build delete <name> [flags]
----

.Example: Delete a build and all of the buildruns
[source,terminal]
----
$ shp build delete mybuild -r
----

== Options

`-r, --delete-runs`:: Also delete all of the buildruns.
`-h, --help`:: Help for `delete`.
