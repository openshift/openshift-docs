= Updating the controller revision to the latest version

[role="_abstract"]
As a VM owner, if you want to use a newer version of the controller revision for your VM, you must manually remove the reference to the existing controller revision in your `VirtualMachine` custom resource (CR). You can do this by clearing the `revisionName` field. When you clear this field, the controller is forced to look up the latest version of the referenced instance type name and create a new `ControllerRevision` CR.

.Prerequisites

* You created the VM by using an instance type.
* You have installed the {oc-first}.

.Procedure

. Stop the VM.

. Clear the `revisionName` field in the `VirtualMachine` CR by running the following command, replacing `{vm-name}` with the name of your VM:
+
[source,terminal]
----
$ oc patch vm/{vm-name} --type merge -p '{"spec":{"instancetype":{"revisionName":""}}}'
----

. Restart the VM.
