= Tested cluster maximums ({rosa-classic-short} and {hcp-title})

[role="_abstract"]
The following table lists tested resource maximums for {product-title} 4.21 for {rosa-classic-short} and {hcp-title} clusters, measured without changing default cluster configuration beyond what the test describes.

:fn-perfscale-rosa-1: footnote:perfscale-rosa-fn1[Simple pause pods staying idle on the cluster, including default control plane pods on {rosa-classic-short}; used `node-density` workload. Actual pod counts depend on workload requirements.]
:fn-perfscale-rosa-2: footnote:perfscale-rosa-fn2[Workload shape can lower pods per node; for example, many PostgreSQL pods on one node with high disk IOPS, or limits on persistent volume claims (PVCs) per node.]

:fn-perfscale-rosa-4: footnote:perfscale-rosa-fn4[ClusterIP services only; other service types can have different limits. Used `cni-workload`.]
