= Installing {microshift-short} Observability

You can install the {microshift-short} Observability service at any time, including during the initial {microshift-short} installation.

.Procedure
. Install the service by entering the following command from a terminal window:
+
[source,terminal]
----
$ sudo dnf install microshift-observability -y
----

. Start the service by rebooting the host or restarting the service.
+
.Reboot the host
+
[source,terminal]
----
$ sudo systemctl reboot
----
+
.Restart the service
+
[source,terminal]
----
$ sudo systemctl restart microshift-observability
----

The installation procedure is successful when no output returns.
