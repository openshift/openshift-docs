= Installing {microshift-short} Observability

You can install the {microshift-short} Observability service at any time, including during the initial {microshift-short} installation.

.Procedure
. Install the service by entering the following command from a terminal window:
+
[source,terminal]
----
$ sudo dnf install microshift-observability -y
----

. Enable and start the service.
+
.Enable the service
[source,terminal]
----
$ sudo systemctl enable microshift-observability
----
+
.Start the service
+
[source,terminal]
----
$ sudo systemctl start microshift-observability
----

The service is successfully installed if there is no output after you start the service.
