= Configuring your `ServiceMeshControlPlane` resource for production

If you have installed a basic `ServiceMeshControlPlane` resource to test {ProductShortName}, you must configure it to production specifications before your deploy your production service mesh.

You cannot change the `metadata.name` field of an existing `ServiceMeshControlPlane` resource. 
 
For production deployments you _must_ customize the default link:https://github.com/maistra/istio-operator/blob/maistra-2.0/deploy/examples/maistra_v2_servicemeshcontrolplane_cr_auth.yaml[template]
