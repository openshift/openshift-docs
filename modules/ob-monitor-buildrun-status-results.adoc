= Monitor BuildRun status and results

[role="_abstract"]
Monitor `BuildRun` status and results to verify build completion, troubleshoot failures, and understand the relationship between `BuildRun` resources and underlying Tekton components. Use status fields, mappings, and result data to track build execution.
