= Splunk integration improvements

[role="_abstract"]

// JIRA: ROX-26107
The Splunk integration has been enhanced with improved data formatting and additional event types.

These improvements provide better visibility into {product-title-short} security events within Splunk, including enhanced file activity monitoring notifications and more detailed violation data.
