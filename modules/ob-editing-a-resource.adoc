= Editing a resource

You can edit or delete a `Build` resource as required in your project.

.Prerequisites

* You have installed the `oc` CLI.

.Procedure

. Run the following command open the YAML definition in the default editor:
+
[source,terminal]
----
$ oc edit resource <resource-name> # <1>
----
<1> Replace <build_resource-name> with the name of the resource that you want to edit.

. Edit the YAML definition and save the file.
