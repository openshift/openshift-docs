= Connecting to a pod

You can directly connect to a currently running pod with the `oc rsh` command, which provides you with a shell on that pod.

[IMPORTANT]
====
Only use the `oc rsh` command if you cannot connect to the node with the `oc debug` command.
====

[WARNING]
====
The use of the `oc rsh` command can cause latency issues in pods running a low-latency application.
====

.Procedure

* Connect to your pod by running the following command:
+
[source,terminal]
----
$ oc rsh -n <namespace> busybox-1
----
