= Enabling {oadp-short} Self-Service

You must be a cluster administrator to enable the {oadp-short} Self-Service. You can use the `spec.nonAdmin.enable` section of the `DataProtectionApplication` custom resource (CR) to enable and disable the Self-Service feature.
