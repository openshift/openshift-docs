= Introduction to {ibm-power-server-name} Block CSI Driver Operator 

[role="_abstract"]
The {ibm-power-server-name} Block CSI Driver is installed through the {ibm-power-server-name} Block CSI Driver Operator and the operator is based on `library-go`. 

The {product-title} `library-go` framework is a collection of functions that allows users to build OpenShift operators easily. Most of the functionality of a CSI Driver Operator is already available there. The {ibm-power-server-name} Block CSI Driver Operator is installed by the Cluster Storage Operator. The Cluster Storage Operator installs the {ibm-power-server-name} Block CSI Driver Operator if the platform type is Power Virtual Servers.
