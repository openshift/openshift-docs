[id="roxctl-sensor-options_{context}"]
= Options

[role="_abstract"]
Options for the `roxctl sensor` command.

[cols="2,2",options="header"]
|===
|Option |Description

|`--retry-timeout duration`
|Set the timeout before retrying API requests. A value of zero disables retries and waits for the entire request duration. The default value is `20s`.

|`-t`, `--timeout duration`
|Set the timeout for API requests representing the maximum duration of a request. The default value is `1m0s`.
|===
