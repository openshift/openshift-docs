= Run the Terraform script via the C3 region

You now create all infrastructure resources on the {oci-c3-short} system, including LBs, VCNs, and Instances.

.Procedure

. Set the C3 Certificate location and export the certificate. 

. Apply the terraform script to create three control plane nodes and three compute nodes. 

. Update the tags for the control plane and compute nodes.

. Stop and start the instances one by one on the {oci-c3-short} portal.

For the full procedure, see link:https://www.oracle.com/a/otn/docs/compute_cloud_at_customer_agent_based_installation.pdf?source=:em:nl:mt::::PCATP[Terraform Script Execution - Part 2 (Oracle documentation)].
