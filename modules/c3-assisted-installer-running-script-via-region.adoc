= Run the Terraform script via the C3 region

You now create all infrastructure resources on the C3 system, including LBs, VCNs, and Instances. During this procedure, you apply the terraform script to create three control plane nodes and three compute nodes. You then update the tags for the nodes nodes, and then stop and start the instances one by one on the C3 portal.

For the full procedure, see link:https://app.slack.com/client/E030G10V24F/D05CCFA8UVB[Terraform Script Execution - Part 2 (Oracle documentation)].
