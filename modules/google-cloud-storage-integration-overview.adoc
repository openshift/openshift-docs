= Google Cloud Storage integration overview

[role="_abstract"]
After integrating with Google Cloud Storage, you can schedule daily or weekly backups and perform manual on-demand backups.

The backup includes the {product-title} entire database, which includes all configurations, resources, events, and certificates.
Make sure that you store backups securely.

[NOTE]
====
If you are using {product-title} version 3.0.53 or older, the backup does not include certificates.
====
