= Adding `kernel-devel` to worker nodes

To install the {IBMFusionSAN} Operator, you must first install `kernel-devel` on bare-metal worker nodes with shared storage.

.Prerequisites

* You installed the `oc` CLI.
* Have a running {product-title} cluster.
* Log in to the cluster as a user with administrative privileges.

.Procedure

. Create a machine config for extensions: Create a YAML file (for example, `kernel-devel-extension.yaml`) that includes `kernel-devel` as the `extensions` object.
+
.Example `MachineConfig`
+
[source,terminal]
----
$ cat << EOF > kernel-devel-extension
apiVersion: machineconfiguration.openshift.io/v1
kind: MachineConfig
metadata:
  labels:
    machineconfiguration.openshift.io/role: worker
  name: kernel-devel-extension
spec:
  config:
    ignition:
      version: 5.14.0 (ASK ABOUT THIS)
  extensions:
    - kernel-devel
EOF
----

. Add the machine config to the cluster:
+
[source,terminal]
----
$ oc create -f kernel-devel-extension.yaml
----
+
This sets all worker nodes to have rpm packages for `kernel-devel` installed.

.Verification

. Check that the extensions were applied:
+
[source,terminal]
----
$ oc get machineconfig kernel-devel-extension
----
+
.Example output
+
[source,terminal]
----
NAME                 GENERATEDBYCONTROLLER IGNITIONVERSION AGE
kernel-devel-extension                      5.14.0           57s
----

. Check that the new `MachineConfig` is applied and that the nodes are not in a degraded state. This may take a few minutes. The worker pool shows the updates in progress, as the new `MachineConfig` is successfully applied to each machine:
+
[source,terminal]
----
$ oc get machineconfigpool
----
+
.Example output
+
[source,terminal]
----
NAME   CONFIG             UPDATED UPDATING DEGRADED MACHINECOUNT READYMACHINECOUNT UPDATEDMACHINECOUNT DEGRADEDMACHINECOUNT AGE
master rendered-master-35 True    False    False    3            3                 3                   0                    34m
worker rendered-worker-d8 False   True     False    3            1                 1                   0                    34m
----
