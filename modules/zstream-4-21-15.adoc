== Enhancements

{product-title} 4.21 support for installing a cluster without an external registry (Technology Preview)::
As a Technology Preview feature, you can deploy a 4.21 cluster without an external registry, using self-contained installation media that also provides a simplified user interface similar to the {ai-full} during on-premise installations.

For more information, see the following documentation:
