= Key features

[role="_abstract"]
{pipelines-title} provides a serverless, cloud-native CI/CD system that runs on {OCP}. It supports decentralized teams working on microservice-based architectures and integrates with existing {OCP} tools.

* {pipelines-title} is a serverless CI/CD system that runs pipelines with all the required dependencies in isolated containers.
* {pipelines-title} support decentralized teams that work on microservice-based architecture.
* {pipelines-title} use standard CI/CD pipeline definitions that are easy to extend and integrate with the existing {OCP} tools, enabling you to scale on-demand.
* You can use {pipelines-title} to build images with {OCP} tools such as Source-to-Image (S2I), Buildah, Buildpacks, and Kaniko that are portable across any {OCP} platform.
* You can use the {OCP} Developer console to create Tekton resources, view logs of pipeline runs, and manage pipelines in your {OCP} namespaces.
