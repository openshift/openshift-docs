= Upgrading all the OLM Operators

In the second phase of a multi-version upgrade, you need to approve all of the Operators and additionally add install plans for any other Operators that you want to upgrade.

.Procedure
* Follow the same procedure as outlined in "Monitoring the cluster upgrade".
Ensure that you also update any non-OLM Operators as required.
