= Monitoring RHACS by using a Prometheus server

[role="_abstract"]
You can create a dedicated Prometheus server and configure it to use custom metrics from {product-title-short}. You can then configure another application to use this server. For example, you can create a Perses dashboard to use in the {ocp} console that displays {product-title-short} data.
