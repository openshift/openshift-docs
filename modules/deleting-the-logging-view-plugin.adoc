= Deleting the logging view plugin

[role="_abstract"]
When you update from Logging 5 to Logging 6, delete the logging view plugin before installing the `UIPlugin`.

.Prerequisites
* You have administrator permissions.
* You installed the {oc-first}.

.Procedure

* Delete the logging view plugin by running the following command:
+
[source,terminal]
----
$ oc get consoleplugins logging-view-plugin && oc delete consoleplugins logging-view-plugin
----
