= Deleting cloned volumes in {sno}

You can delete cloned volumes.

.Procedure

* To delete the cloned volume, delete the cloned PVC by running the following command:
+
[source,terminal]
----
# oc delete pvc <clone_pvc_name> -n <namespace>
----
