= Converting Machine API resources to Cluster API resources

You can convert individual Machine API objects to Cluster API equivalents of those objects. 

Machine API and Cluster API represent different API group versions, but with similar resources. This workflow allows switching management of the infrastructure resources (e.g. EC2 instances) from the former API, to the new API.

Cluster administrators can use the conversion process to verify that migrating existing resources to the Cluster API works as expected before transitioning to use the Cluster API in production clusters.
By converting and checking individual resources, you can experiment with the Cluster API before configuring a running cluster to use it. 

After converting a Machine API resource to a Cluster API equivalent, you can examine the new resource to verify that the features and configuration match the original Machine API resource.
