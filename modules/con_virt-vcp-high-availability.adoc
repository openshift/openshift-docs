= High availability for virtualized control planes

[role="_abstract"]
A virtualized control plane relies on `etcd` quorum to remain operational.
With three control plane VMs, the cluster tolerates the loss of one VM but becomes unavailable if two VMs fail simultaneously.
To protect quorum, distribute VMs across physical nodes and prevent disruptive operations from affecting more than one control plane VM at a time.

[IMPORTANT]
====
Live migration is not supported for virtualized control plane VMs.
Set the eviction strategy for control plane VMs to `None` to prevent the hosting cluster from attempting to migrate them during node drain operations.
====
