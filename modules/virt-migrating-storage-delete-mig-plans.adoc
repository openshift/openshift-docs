= Deleting storage migration plans by using the web console 

[role="_abstract"]
You can delete old or unwanted storage migration plans by using the {product-title} web console.

.Procedure

. Navigate to *Virtualization* -> *Migration* -> *Storage migrations* in the web console.

. (Optional) Use the *Filter* dropdown to filter the migration plan list by *Running*, *Pending*, *Failed*, and *Completed* plans.

. (Optional) Perform the following tasks to search the migration plan list:

.. Select either *Name* or *Label* from the search type dropdown list.
.. Enter the search text in the search textbox. The migration plan list filters as the search criteria entered.

. Click the *Options* menu {kebab} beside the migration plan.

. Click *Delete MultiNamespaceVirtualMachineStorageMigrationPlan*.

. Click *Delete*.
