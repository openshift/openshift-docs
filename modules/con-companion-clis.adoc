= Install and use companion CLIs

[role="_abstract"]
When you need to manage specific OpenShift platform services from the command line, learn how to install and use the purpose-built CLI for that service so that you can perform specialized tasks like pipeline management, serverless deployment, GitOps, Operator catalog building, or hosted cluster provisioning.

Pipelines CLI (tkn):: Installing the {pipelines-title} CLI on Linux, Installing the {pipelines-title} CLI on Linux using an RPM, Installing the {pipelines-title} CLI on Windows, Installing the {pipelines-title} CLI on macOS, Enabling tab completion, Utility commands, Pipelines management commands, Pipeline run commands, Task management commands, Task run commands, Condition management commands, Pipeline Resource management commands, ClusterTask management commands, Trigger management commands, Hub interaction commands
opm CLI:: About the opm CLI, Installing the opm CLI, generate, index, init, migrate, render, serve, validate
Hosted control planes CLI (hcp):: Installing the {hcp} command-line interface from the terminal, Installing the {hcp} command-line interface by using the web console, Installing the {hcp} command-line interface by using the content gateway
