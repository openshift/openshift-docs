= API and authentication health checks

[role="_abstract"]
You can verify API server responsiveness, authentication, and certificate expiration before starting a cluster update.

API server responsiveness::

[source,terminal]
----
$ oc version
----

Pass criteria: Command completes within 5 seconds.

Authentication::

[source,terminal]
----
$ oc whoami
----

Pass criteria: Command returns your username successfully.

Certificate expiration::

[source,terminal]
----
$ oc get certificates -A
----

Pass criteria: No certificates expire within 30 days.
