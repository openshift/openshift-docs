== Uninstalling {microshift-short} from an RPM package
include::_attributes/attributes-microshift.adoc[]
:context: microshift-proc-uninstall-from-rpm
toc::[]

* To uninstall a {microshift-short} from an RPM installation, run the following command:

[source,terminal,subs="+quotes"]
----
$ sudo rpm -e microshift microshift-release-info
----
