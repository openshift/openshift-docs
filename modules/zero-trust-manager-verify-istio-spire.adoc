= Verifying Istio installation with {zero-trust-full}

[role="_abstract"]
Deploy a test httpbin workload and verify that it receives SPIRE-issued certificates. This verification confirms that Istio sidecars are successfully communicating with SPIRE through the Workload API.

.Prerequisites

* Istio deployed with SPIRE integration.

* `istioctl` CLI installed.

.Procedure

. Define the verification namespace by running the following command:
+
[source,terminal]
----
$ export VERIFY_NS=verify-ossm-ztwim
----

. Create the verification namespace and enable sidecar injection by running the following commands:
+
[source,terminal]
----
$ oc new-project "${VERIFY_NS}"
----
+
[source,terminal]
----
$ oc new-project "${VERIFY_NS}"
----

. Deploy the httpbin test workload by running the following command:
+
[source,terminal]
----
$ cat <<EOF | oc apply -f -
apiVersion: apps/v1
kind: Deployment
metadata:
  name: httpbin
  namespace: ${VERIFY_NS}
spec:
  replicas: 1
  selector:
    matchLabels:
      app: httpbin
      version: v1
  template:
    metadata:
      annotations:
        inject.istio.io/templates: "sidecar,spire"
        spiffe.io/audience: "sky-computing-demo"
      labels:
        app: httpbin
        version: v1
    spec:
      containers:
      - image: docker.io/mccutchen/go-httpbin:v2.15.0
        imagePullPolicy: IfNotPresent
        name: httpbin
        ports:
        - containerPort: 8080
EOF
----

. Wait for the httpbin deployment to become ready by running the following commands:
+
[source,terminal]
----
$ until oc get deployment httpbin -n "${VERIFY_NS}" &> /dev/null; do sleep 3; done
----
+
[source,terminal]
----
$ oc wait --for=condition=Available deployment/httpbin -n "${VERIFY_NS}" --timeout=300s
----

. Extract the certificate from the httpbin pod by running the following commands:
+
[source,terminal]
----
$ HTTPBIN_POD=$(oc get pod -l app=httpbin -n "${VERIFY_NS}" -o jsonpath="{.items[0].metadata.name}")
----
+
[source,terminal]
----
$ istioctl proxy-config secret "$HTTPBIN_POD" -n "${VERIFY_NS}" -o json | jq -r  '.dynamicActiveSecrets[0].secret.tlsCertificate.certificateChain.inlineBytes' | base64  --decode > chain.pem
----

. Verify that the certificate was issued by SPIRE by running the following command:
+
[source,terminal]
----
$ openssl x509 -in chain.pem -text | grep SPIRE
----
+
.Example output
[source,terminal]
----
        Issuer: C=US, O=Sky Computing Corporation, CN=SPIRE Server CA/serialNumber=276066964103311963277244026411475463737
        Subject: C=US, O=SPIRE
----
