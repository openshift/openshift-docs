= Impact of the failed management cluster component

If the management cluster component fails, your workload remain unaffected. In {product-title} management cluster, the control plane is decoupled from the data plane to provide resiliency.

See the following table to know the impact of a failed management cluster component on the control plane and the data plane.

.Impact of the failed component on hosted control planes
[cols="1,1,1",options="header"]
|===
|Name of the failed component |Hosted control plane API status |Hosted cluster data plane status

|Worker node
|Available
|Available

|Availability zone
|Available
|Available

|Management cluster control plane
|Available
|Available

|Management cluster control plane and worker nodes
|Not available
|Available
|===

The table does not cover all scenarios for the management cluster component failures.
