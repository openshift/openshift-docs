= Validating the deployment

[role="_abstract"]
After installing your controller, you can verify the installation by using the {oc-first} tool.

.Procedure
. Deploy an S3 bucket resource:
+
[source,terminal]
----
$ cat << EOF | oc apply -f -
apiVersion: s3.services.k8s.aws/v1alpha1
kind: Bucket
metadata:
   name: ${CLUSTER-NAME}-bucket
   namespace: ack-system
spec:
   name: ${CLUSTER-NAME}-bucket
EOF
----
+
. Verify the S3 bucket was created in AWS:
+
[source,terminal]
----
$ aws s3 ls | grep ${CLUSTER_NAME}-bucket
----
+
**Example output**:
+
[source,text]
----
2023-10-04 14:51:45 mrmc-test-maz-bucket
----
