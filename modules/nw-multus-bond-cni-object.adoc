= Configuration for a bond-cni secondary network

The following object describes the configuration parameters for the bond CNI plugin:

.Bond CNI plugin JSON configuration object
[cols=".^2,.^2,.^6",options="header"]
|====
|Field|Type|Description


|`name`
|`string`
|The value for the `name` parameter you provided previously for the CNO configuration.

|`cniVersion`
|`string`
|The CNI specification version.

|`type`
|`string`
|The name of the CNI plugin to configure: `bridge`.

|`miimon`
|`string`
|Specifies the arp link monitoring frequency in milliseconds.

|`mtu`
|`string`
|Optional: The maximium transmission unit (mtu) of the bond. The default is 1500. 

|`failOverMac`
|`integer`
|Optional: Specifies the `failOverMac` setting for the bond. Should be set to 1 for `active-backup` bond modes. Default is 0.

|`mode`
|`string`
|Specifies the policy that governs how the network traffic is distributed and managed across the multiple network interfaces aggregated into a single logical bonded interface. The bonding modes supported are `balance-rr`, `active-backup` and `balance-xor`. 

|`linksInContainer`
|`boolean`
|Optional: Specifies if slave links are in container to start. Default is `false` that is look for interfaces on host before bonding.

|`links`
|`object`
|Master interface names.

|`ipam`
|`object`
|The configuration object for the IPAM CNI plugin. The plugin manages IP address assignment for the attachment definition.

|====

[id="nw-multus-bond-cni-config-example_{context}"]
== Bond CNI plugin configuration example

The following example configures a secondary network named `bond-net1`:

[source,json]
----
{
 "type": "bond", 
 "cniVersion": "0.3.1",
 "name": "bond-net1",
 "mode": "active-backup", 
 "failOverMac": 1, 
 "linksInContainer": true, 
 "miimon": "100",
 "mtu": 1500,
 "links": [ 
       {"name": "net1"},
       {"name": "net2"}
   ],
  "ipam": {
        "type": "host-local",
        "subnet": "10.56.217.0/24",
        "routes": [{
        "dst": "0.0.0.0/0"
        }],
        "gateway": "10.56.217.1"
    }
}
----
