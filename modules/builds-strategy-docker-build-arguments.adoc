= Adding Docker build arguments

You can set link:http://docs.docker.com/v1.7/reference/api/hub_registry_spec/#docker-registry-1-0[Docker build arguments] using the `BuildArgs` array. The build arguments will be passed to Docker when a build is started.

.Procedure

To set Docker build arguments, add entries to the `BuildArgs` array, which is located in the `dockerStrategy` definition of the `BuildConfig`. For example:

[source,yaml]
----
dockerStrategy:
...
  buildArgs:
    - name: "foo"
      value: "bar"
----
