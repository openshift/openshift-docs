[id="shp-build-list_{context}"]
= shp build list

List the builds.

----
$ shp build list [flags]
----

.Example: List the builds without a header column
[source,terminal]
----
$ shp build create mybuild --no-header
----

== Options

`-h, --help`:: Help for `list`.
`--no-header`:: Do not show columns header in list output.
