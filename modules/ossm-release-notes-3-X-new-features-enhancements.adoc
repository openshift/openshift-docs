= {SMProductName} version 3.X new features and enhancements

[role="_abstract"]

This release makes {SMProductName} 3.X generally available, adds new features, addresses Common Vulnerabilities and Exposures (CVEs), and is supported on {ocp-product-title} 4.18 and later.

For a list of supported component versions and support features, see "Service Mesh feature support tables".

When upgrading from {SMProduct} 2.x, first you must migrate to version 3.0. Then, you can upgrade to version 3.1. For more information see, "Migrating from Service Mesh 2 to Service Mesh 3".
