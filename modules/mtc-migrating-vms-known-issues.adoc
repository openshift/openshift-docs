= Known issues

The following known issues apply when migrating virtual machine storage: 

* Cannot switch Persistent Volume volume mode.
* Can only storage migrate in the same namespace.

[id="mtc-migrating-vms-limitations_{context}"]
== Online migration limitations

The following limitations apply to the online migration: 

* The virtual machine must be running.
* The volume housing the disk must not have any of the following properties:
** Shared disks cannot be migrated live.
** Hotplugged volumes cannot be live migrated.
** Virtio-fs: filesystem shared inside the VM, since virtio-fs volume cannot be migrated live.
** LUN to Disk and Disk to LUN migrations are not supported in libvirt.
** LUNs must have persistent reservation.
** Target Persistent Volume size must match the source Persistent Volume size.
* Virtual machine must be migrated to a different node.
