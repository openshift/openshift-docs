= Applying firewall settings

[role="_abstract"]
To activate your custom network access configurations, apply the firewall settings in {product-title}. You can use a single command to reload the firewall service and ensure your new rules take effect immediately.

To apply firewall settings, use the following one-step procedure:

.Procedure

* After you have finished configuring network access through the firewall, run the following command to restart the firewall and apply the settings:
+
[source,terminal]
----
$ sudo firewall-cmd --reload
----
