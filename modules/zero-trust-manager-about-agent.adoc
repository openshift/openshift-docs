= SPIRE agent


The SPIRE agent acts as a secure intermediary between the SPIRE server and the workloads running on a node. It performs node attestation against the server as well as workload attestation againt applications running on the node. The SPIRE agent is also responsible for retrieving, delivering, and renewing SVIDs. A SPIRE agent runs on every node on which an identified workload runs. For more information, see link:https://spiffe.io/docs/latest/spire-about/spire-concepts/#all-about-the-agent[SPIRE Agent].
