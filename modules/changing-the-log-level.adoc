= Changing the log level

.Procedure

* Run the following command to change the log level:
+
[source,terminal,subs="+quotes"]
----
$ roxctl central debug log --level=_<log_level>_
----
+
where:
+
--
`<log_level>`:: Specifies the log level. The accepted values for the log level are `Panic`, `Fatal`, `Error`, `Warn`, `Info`, and `Debug`.
--
