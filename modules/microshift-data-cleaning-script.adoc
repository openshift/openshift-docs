= Data cleaning script

.Procedure

* Stop the {microshift-short} services and clean all the {microshift-short} and OVN data by running the following command:
+
[source,terminal]
----
$ sudo microshift-cleanup-data --all
----
+
.Example output
[source,terminal]
----
//To be added
----

* Stop the {microshift-short} services, clean the {microshift-short} and OVN data while keeping the container images by running the following command:
+
[source,terminal]
----
$ sudo microshift-cleanup-data --all [--keep-images]
----
+
.Example output
[source,terminal]
----
//To be added
----

* Stop the {microshift-short} services, clean the OVN data by running the following command:
+
[source,terminal]
----
$ sudo microshift-cleanup-data --ovn
----
+
.Example output
[source,terminal]
----
//To be added
----

* Stop the {microshift-short} services and clean up the custom certificates by running the following command:
+
[source,terminal]
----
$ sudo microshift-cleanup-data --cert
----
+
.Example output
[source,terminal]
----
Stopping MicroShift services
Removing MicroShift certificates
MicroShift service was stopped
Cleanup succeeded
----

* Stop the {microshift-short} services, clean the {microshift-short}, OVN data and custom certificates while keeping the container images by running the following command:
+
[source,terminal]
----
$ sudo microshift-cleanup-data <--all [--keep-images] | --ovn | --cert>
