= About the {cli-manager}

The {cli-manager} uses Krew to make installing and updating cli plugins easier. It can be used in both connected and disconnected environments, but has particular utility in disconnected environments. Cluster administrators can add CLI plugins to the {cli-manager} and users can then add CLI plugins when needed regardless of whether or not the environment is disconnected.
