[id="roxctl-central-backup-options_{context}"]
= Options

[role="_abstract"]
Options for the `roxctl central backup` command.

.Options
[cols="2,2",options="header"]
|===
|Option |Description

|`--certs-only`
|Specify to only back up the certificates. When using an external database, the command uses this option to generate a backup bundle with certificates. The default value is `false`.

|`--output string`
a|Specify where you want to save the backup. The behavior depends on the specified path:

* If the path is a file path, the command writes the backup to the file and overwrites it if it already exists. The directory must exist.
* If the path is a directory, the command saves the backup in this directory under the file name that the server specifies.
* If you omit this argument, the command saves the backup in the current working directory under the file name that the server specifies.

|`-t`, `--timeout duration`
|Specify the timeout for API requests. It represents the maximum duration of a request. The default value is `1h0m0s`.
|===
