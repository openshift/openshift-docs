= Cluster update scenarios

[role="_abstract"]
You can perform z-stream, y-stream, and EUS-to-EUS updates on clusters managed by {rh-rhacm-first} and {cgu-operator-full} ({cgu-operator}).
Each update scenario has different risk profiles, release cadences, and use cases that align with your operational requirements and maintenance windows.
