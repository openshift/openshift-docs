= Edit a Perses Dashboard by using the CLI

[role="_abstract"]
You can edit an existing Perses dashboard by editing the custom resource (CR) directly.

.Prerequisites

* You have edit permissions in the namespace where the dashboard resides.

.Procedure

. Retrieve the current dashboard specification by running the following command:
+
[source,terminal]
----
$ oc get persesdashboard <dashboard_name> \
    -n <namespace> \
    -o yaml > dashboard.yaml
----

. Edit the `dashboard.yaml` file and apply your changes.

. Apply the updated resource by running the following command:
+
[source,terminal]
----
$ oc apply -f dashboard.yaml
----

.Verification

* In the {product-title} web console, open the dashboard and confirm that your changes are reflected.
