= Options

[role="_abstract"]
Options for the `roxctl collector support-packages upload` command.

.Options
[cols="2,2",options="header"]
|===
|Option |Description

|`--overwrite`
|Specify whether you want to overwrite existing but different files. The default value is `false`.

|`--retry-timeout duration`
|Set the timeout for retrying API requests. A value of zero waits for the entire request duration without retrying. The default value is `20s`.

|`-t`, `--timeout duration`
|Set the timeout for API requests. This option represents the maximum duration of a request. The default value is `1m0s`.
|===
