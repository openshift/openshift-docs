== Network Observability Operator 1.10 deprecated features

The following features have been deprecated in the Network Observability Operator 1.10.

[id="flowcollector-api-v1beta1_{context}"]
== FlowCollector API version v1beta1

FlowCollector API version `v1beta1` has been deprecated. Use `v1beta2`.
