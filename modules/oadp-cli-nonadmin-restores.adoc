= Non-admin restore management commands

[role="_abstract"]
You can use the {oadp-short} command-line interface (CLI) to create, view, describe, and delete non-admin restores in your namespace.

== Prerequisites

* The {oadp-short} CLI plugin is installed and configured in non-admin mode.
* You are logged in to the {product-title} cluster and your current namespace context is set to the namespace you want to restore into.
* You have editor roles for `nonadminrestores.oadp.openshift.io` in your namespace.
* A completed non-admin backup exists to restore from.

== Non-admin restore creation command

To create a restore from an existing non-admin backup, use the following command:

[source,terminal,subs="+quotes"]
----
$ oc oadp nonadmin restore create [__<restore_name>__] [flags]
----

The restore name is optional. If you do not provide a name, a name is automatically generated.

.Flags for the `oc oadp nonadmin restore create` command
[cols="1,3",options="header"]
|===
|Flag |Description

|`--backup-name`
|The name of the non-admin backup to restore from. This flag is required.

|`--include-resources`
|The resources to include in the restore. You can specify simple kind names, for example, `deployments,services`, or you can use the `resource.group` format for disambiguation, for example, `deployments.apps`. The default value is `*` (all resources).

|`--exclude-resources`
|The resources to exclude from the restore. This flag uses the same format as the `--include-resources` flag.

|`--selector`, `-l`
|Specifies to restore only the resources that match this label selector.

|`--or-selector`
|Specifies to restore resources that match at least one of the label selectors, separated by ` or `.

|`--item-operation-timeout`
|The timeout for asynchronous plugin operations.
|===

.Example of the non-admin restore creation command
[source,terminal]
----
$ oc oadp nonadmin restore create my-restore \
    --backup-name my-backup \
    --include-resources deployments,services \
    --selector app=myapp
----

== Non-admin restore listing command

To list all restores in your current namespace, use the following command:

[source,terminal,subs="+quotes"]
----
$ oc oadp nonadmin restore get [__<restore_name>__] [flags]
----

.Flags for the `oc oadp nonadmin restore get` command
[cols="1,3",options="header"]
|===
|Flag |Description

|`-o`
|The output format. Supported values are `json` and `yaml`.
|===

== Non-admin restore description command

To view the details of a restore, use the following command:

[source,terminal,subs="+quotes"]
----
$ oc oadp nonadmin restore describe __<restore_name>__ [flags]
----

.Flags for the `oc oadp nonadmin restore describe` command
[cols="1,3",options="header"]
|===
|Flag |Description

|`--details`
|Specifies whether to display additional restore details.

|`--request-timeout`
|The timeout for fetching restore details from the server.
|===

== Non-admin restore logs command

To view the logs for a restore, use the following command:

[source,terminal,subs="+quotes"]
----
$ oc oadp nonadmin restore logs __<restore_name>__ [flags]
----

.Flags for the `oc oadp nonadmin restore logs` command
[cols="1,3",options="header"]
|===
|Flag |Description

|`--request-timeout`
|The timeout for fetching logs from the server.
|===

[NOTE]
====
Restore logs are available only when you use a `NonAdminBackupStorageLocation` object. Logs are not available for restores associated with backups that use the default cluster backup storage location.
====

== Non-admin restore deletion command

To delete one or more restores, use the following command:

[source,terminal,subs="+quotes"]
----
$ oc oadp nonadmin restore delete [__<restore_name>__...] [flags]
----

.Flags for the `oc oadp nonadmin restore delete` command
[cols="1,3",options="header"]
|===
|Flag |Description

|`--confirm`
|Specifies whether to skip the confirmation prompt and delete immediately.

|`--all`
|Specifies whether to delete all restores in the current namespace.
|===

Restore deletion is performed asynchronously by the {oadp-short} non-admin controller.

.Example of the non-admin restore deletion command
[source,terminal]
----
$ oc oadp nonadmin restore delete my-restore --confirm
----
