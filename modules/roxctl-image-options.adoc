= Options

[role="_abstract"]
Options for the `roxctl image` command.

[cols="2,2",options="header"]
|===
|Option |Description

|`-t`, `--timeout duration`
|Set the timeout for API requests representing the maximum duration of a request. The default value is `10m0s`.
|===
