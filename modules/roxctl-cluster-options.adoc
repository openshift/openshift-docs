[id="roxctl-cluster-options_{context}"]
= Options

[role="_abstract"]
Options for the `roxctl cluster` command.

[cols="2,2",options="header"]
|===
|Option |Description

|`--retry-timeout duration`
|Set the retry timeout for API requests. A value of zero allows the full request duration without retry. The default value is `20s`.

|`-t`, `--timeout duration`
|Set the timeout for API requests representing the maximum duration of a request. The default value is `1m0s`.
|===
