= Build architecture

[role="_abstract"]
{builds-shortname} provides Kubernetes-native APIs to build container images from source code and Dockerfiles. It supports S2I and Buildah strategies, allows custom strategies, offers a CLI for managing builds, and integrates with developer perspective of the {ocp-product-title} console.

{builds-shortname} consists of the following custom resources (CRs):

* `Build`
* `BuildStrategy` and `ClusterBuildStrategy`
* `BuildRun`
