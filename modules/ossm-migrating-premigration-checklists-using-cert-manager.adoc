= Check if you are using the cert-manager tool with your deployment

It is important to know if you are using the cert-manager tool with {SMProduct} 2. There are a series of configuration steps you will need to perform before you can start migrating your deployments and workloads.

.Procedure

. You can check to see if you are using the cert-manager tool with your deployment :

.. Inspecting your `ServiceMeshControlPlane` resource:
+
[source,yaml]
----
spec:
  security:
    certificateAuthority:
      ...
      type: cert-manager
----

.. Running the following command:
+
[source,yaml]
----
oc get smcp <smcp-name> -n <smcp-namespace> -o jsonpath='{.spec.security.certificateAuthority.type}'
----
+
.Output

[source,yaml]
----
cert-manager
----
