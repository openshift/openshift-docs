= Getting your cluster ID using kube-sytem namespace

.Procedure

* Get the namespace UID of the running cluster by entering the following command:
+
[source,terminal]
----
$ # oc get namespaces kube-system -o jsonpath={.metadata.uid}
----

* Get the namespace UID of a non-running cluster by entering the following command:
+
[source,terminal]
----
# cat /var/lib/microshift/cluster-id
----

[NOTE]
====
The cluster-id file contains the namespace UID for a non-running cluster. The cluster-id file is found in the directory '/var/lib/microshift'.
====

* You can view the {microshift-short} database files in the directory '/var/lib/microshift' by entering the following command.
+
[source,terminal]
----
# ls -l /var/lib/microshift
----
