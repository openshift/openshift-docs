= Preparing the Oracle Bastian server

Currently Compute Cloud@Customer (C3) does not support OCI Resource Manager Stack (RMS). To run the 
Terraform script, install a Bastion Server within the targeted OpenShift Cluster deployment. 
Then load the Terraform and OCI CLI on to the Bastion Server for execution.

For details, see link:https://app.slack.com/client/E030G10V24F/D05CCFA8UVB[Bastian Server - Prerequisites (Oracle documentation)]
