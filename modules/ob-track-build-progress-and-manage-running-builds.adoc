= Track build progress and manage running builds

[role="_abstract"]
Monitor build execution, inspect results, and control running builds through status fields, build snapshots, and cancellation mechanisms. The build controller continuously updates `BuildRun` status to reflect the current state of image construction and push operations.

Build tracking and management includes:

* Buildrun status: Real-time information about whether a build is pending, running, succeeded, or failed, including the specific reason and error messages
* Build run cancelation: Ability to stop in-progress builds by setting the build run state, which cascades to underlying Tekton tasks
