= Policy enforcement overview

[role="_abstract"]
{product-title-short} helps you secure your container infrastructure by establishing security guardrails, informing about policy violations, and selectively enforcing policies to prevent insecure situations.

Specifically, {product-title-short} can help you fail the CI pipeline in the build stage, reject deployments in the deployment stage, and shut down running deployments that violate runtime policies. {product-title-short} also offers policy management and violation management tools described in the following sections.
