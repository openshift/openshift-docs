[id="rosa-download_{context}"]
= Download CLI tools with the {rosa-cli}

[role="_abstract"]
Download and verify the `rosa` and `oc` CLI tools by using the {rosa-cli-first}.

[id="rosa-download-rosa-client_{context}"]
== download rosa

Download the latest compatible version of the {rosa-cli}.

After you download {rosa-cli}, extract the contents of the archive and add it to your path.

.Syntax
[source,terminal]
----
$ rosa download rosa [arguments]
----
