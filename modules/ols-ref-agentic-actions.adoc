= AI agent actions

[role="_abstract"]
The agent can perform some actions autonomously, while other actions require explicit human approval.

== Actions the agent performs autonomously

The agent performs the following actions without requiring human approval:

* Subscribes to Prometheus alerts based on configured policies
* Runs investigations in response to matching alerts
* Accesses cluster data through the Core MCP server
* Generates diagnoses, risk assessments, and remediation proposals

== Actions that require human approval

The following actions require explicit cluster administrator approval through the Decision Queue:

* All remediation executions
* Any changes to cluster resources proposed by the agent

== Actions the agent cannot perform

The agent has the following restrictions:

* It cannot execute any remediation without administrator approval.
* It cannot access data outside the scope of the Core MCP server.
* It cannot modify alert subscription policies or its own RBAC permissions.
