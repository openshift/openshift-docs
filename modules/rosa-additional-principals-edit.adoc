= Adding additional principals to your existing ROSA cluster

You can add additional principals to your cluster by using the CLI.

.Procedure

* Run the following command to edit your cluster and add an additional principal who can access this cluster's endpoint:
+
[source,terminal]
----
$ rosa edit cluster -c <cluster_name> --additional-allowed-principals <arn_string>
----
+
You can use `arn:aws:iam::account_id:role/role_name` to a approve a specific role.
