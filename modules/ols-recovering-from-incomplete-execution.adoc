= Recovering from an incomplete execution

[role="_abstract"]
If you stop an execution in an incomplete state, manually inspect the cluster and revert any partially applied changes.

.Procedure

. Inspect the cluster state manually to determine which remediation steps were applied:
+
[source,terminal]
----
$ oc get pods --all-namespaces
$ oc get events --sort-by='.lastTimestamp'
----

. Review the agentic run logs to identify which steps completed and which were interrupted:
+
[source,terminal]
----
$ oc agentic run get <run_id>
$ oc agentic run logs <run_id> --step=Execution
----

. Revert any partially applied changes manually, based on the remediation steps described in the run details.

. If you cannot determine the cluster state or safely revert the changes, escalate to Red{nbsp}Hat Support.
