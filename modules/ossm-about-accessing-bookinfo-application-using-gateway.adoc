= About accessing the Bookinfo application using a gateway

[role="_abstract"]

The {SMProductName} Operator does not deploy gateways. Gateways are not part of the control plane. As a security best-practice, you can deploy Ingress and Egress gateways in a separate namespace than the namespace that has the control plane.

You can deploy gateways by using either the Gateway API or the gateway injection method.
