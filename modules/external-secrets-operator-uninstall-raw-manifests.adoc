= Uninstalling a raw manifest installed community {external-secrets-operator-short}

.Procedure

If you installed the community {external-secrets-operator} by raw manifests, delete the Operator by running the following command:
+
[source,terminal]
----
$ oc delete -f /path/to/your/old/manifests.yaml -n <operator_namespace>
----
