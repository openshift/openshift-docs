= {VirtProductName} 4.20.14 updates

[role="_abstract"]
{VirtProductName} {product-version}.14 is now available with updates to packages and images that fix several bugs and add enhancements.

[id="virt-4-20-14-new_{context}"]
== New features and enhancements

// CNV-73518
Migrate VM disks to different storage classes::
Before this update, installation of {mtc-first} was required before migrating VM disks to different storage classes. With this update, this requirement no longer exists because VM disk storage class migration is native to {VirtProductName} 4.20.14.
+
In addition, you can select VMs from different namespaces for each bulk migration.
+
link:https://issues.redhat.com/browse/CNV-73518[CNV-73518]
