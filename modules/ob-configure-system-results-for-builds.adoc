= Configure System Results for Builds

[role="_abstract"]
Capture build outputs and execution metadata through standardized result files that the build controller reads and exposes in BuildRun status. System results provide essential information about the built image, source code details, and error diagnostics for downstream automation and troubleshooting.

System results configuration includes:

* System results definition: Standardized output fields and file paths where build strategies write execution metadata, image details, and error information for the controller to consume

Build strategies write results to predefined paths during execution, and the build controller reads these files to populate the BuildRun status fields. This mechanism decouples build tools from the controller while ensuring consistent output formats across different strategies.
