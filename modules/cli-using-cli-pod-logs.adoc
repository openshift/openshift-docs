= Viewing pod logs

[role="_abstract"]
Use the `oc logs` command to view logs for a particular pod.

.Procedure

* View logs for a pod by running the following command:
+
[source,terminal]
----
$ oc logs cakephp-ex-1-deploy
----
+
.Example output
[source,terminal]
----
--> Scaling cakephp-ex-1 to 1
--> Success
----
