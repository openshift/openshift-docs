= Lvm cluster custom resource status reporting

[role="_abstract"]
The `LVMVolumeGroupNodeStatus` CR records excluded devices alongside a human-readable reason. 

For devices that static discovery excludes, the reason uses the following format:

[source,text]
----
<device> was not part of <vg_name> at creation (static device discovery enabled)
----

The `VGStatus.DeviceDiscoveryPolicy` parameter reports the effective discovery policy as one of the following values: 

* `Preconfigured`
* `RuntimeDynamic`
* `RuntimeStatic`.
