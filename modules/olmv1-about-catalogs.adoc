= About catalogs in {olmv1}

You can discover installable content by querying a catalog for Kubernetes extensions, such as Operators and controllers, by using the catalogd component. Catalogd is a Kubernetes extension and is part of the {olmv1-first} suite of microservices.
ifndef::openshift-rosa-hcp[]
Currently, catalogd unpacks catalog content that is packaged and distributed as container images for on-cluster clients.
endif::[]
ifdef::openshift-rosa-hcp[]
Currently, catalogd unpacks catalog content securely off-cluster within the Red{nbsp}Hat-managed control plane.
endif::[]
ifdef::openshift-rosa,openshift-dedicated[]
However, these on-cluster infrastructure pods are managed exclusively by Red{nbsp}Hat SRE and should not be modified.
endif::[]
