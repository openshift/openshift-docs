= Migrate Grafana dashboards to a Perses by using the web console

[role="_abstract"]
You can import a Grafana dashboard directly from the {product-title} web console by pasting the Grafana JSON configuration. The console automatically detects the Grafana format and converts it to a `PersesDashboard` resource.

.Prerequisites

* You have access to Grafana dashboard JSON export files.
* You have edit permissions in the target namespace.
* You have created the necessary datasources in Perses.

You can import a Grafana dashboard directly from the {product-title} web console by pasting the Grafana JSON configuration. The console automatically detects the Grafana format and converts it to a `PersesDashboard` resource.

.Procedure

. In Grafana, export the dashboard JSON:
+
.. Open the dashboard you want to migrate.
.. Click *Dashboard settings* (gear icon).
.. Click *JSON Model*.
.. Copy the full JSON content.

. In the {product-title} web console, navigate to *Observe* -> *Dashboards (Perses)*.

. Select the target namespace from the project selector.

. Click *Import Dashboard*.

. Paste the Grafana JSON into the import field.
+
The console detects the Grafana format and converts it to a `PersesDashboard` resource.

. Review the converted dashboard configuration:
+
* Verify that panel types and queries are correct.
* Update datasource references to match your Perses datasources.
* Check variable configurations.

. Click *Import*.

.Verification

* The imported dashboard appears in the dashboards list.
* Open the dashboard and confirm that panels display data correctly.
* Test all dashboard variables.
