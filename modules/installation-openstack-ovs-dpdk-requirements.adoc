= Requirements for clusters on {rh-openstack} that use OVS-DPDK

[role="_abstract"]
To use Open vSwitch with the Data Plane Development Kit (OVS-DPDK) with your deployment, you must meet the following requirements:

* Plan your {rh-openstack-first} OVS-DPDK deployment by referring to link:https://docs.redhat.com/en/documentation/red_hat_openstack_services_on_openshift/18.0/html/deploying_a_network_functions_virtualization_environment/plan-ovs-dpdk-deploy_rhoso-nfv[Planning your OVS-DPDK deployment] in the Deploying a network functions virtualization environment guide.

* Prepare the RHOSO data plane for OVS-DPDK workloads according to link:https://docs.redhat.com/en/documentation/red_hat_openstack_services_on_openshift/18.0/html/deploying_a_network_functions_virtualization_environment/assembly_create-data-plane-sriov-dpdk_rhoso-nfv[Creating a data plane for SR-IOV and DPDK] in the Deploying a network functions virtualization environment guide.
