[id="roxctl-central-db-restore-options_{context}"]
= Options

[role="_abstract"]
Options for the `roxctl central db restore` command.

.Options
[cols="2,2",options="header"]
|===
|Option |Description

|`-f`, `--force`
|If set to `true`, the command performs the restoration without confirmation. The default value is `false`.

|`--interrupt`
|If set to `true`, it interrupts the running restore process to allow it to continue. The default value is `false`.
|===
