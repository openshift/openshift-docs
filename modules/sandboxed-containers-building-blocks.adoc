= OpenShift sandboxed containers building blocks

The {sandboxed-containers} encapsulates all of the pieces from Kata containers and manages installation, lifecycle, and configuration tasks.

The {sandboxed-containers} uses the link:https://quay.io/isolatedcontainers/sandboxed-containers-operator[sandboxed-containers-operator] container image. The image contains the controller part of the operator that watches and manages `KataConfig`.
