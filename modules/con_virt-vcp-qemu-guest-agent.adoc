= QEMU guest agent for virtualized control planes

[role="_abstract"]
Install the QEMU guest agent on each control plane VM to enable the hosting cluster to monitor VM health and perform graceful shutdown operations.
The guest agent reports VM status information, such as IP addresses and filesystem details, back to the hosting cluster.

Without the guest agent, the hosting cluster cannot determine whether the VM operating system is responsive, and shutdown operations fall back to hard power-off behavior.
