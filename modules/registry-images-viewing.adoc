= registry images viewing

You can view the images inside the registry by using the `oc get imagestreams` command.

.Procedure

. Use the `oc get imagestreams` command to view the images already inside the image registry:
+
[source,terminal]
----
$ oc get imagestreams
----
+
.Example output
[source,terminal]
----
