= {oadp-short} 1.4.4 release notes

{oadp-first} 1.4.4 is a Container Grade Only (CGO) release, which is released to refresh the health grades of the containers. No code was changed in the product itself compared to that of {oadp-short} 1.4.3.

[NOTE]
====
When you restore a stateful application that uses `azurefile-csi` storage class, the restore operation remains in the `Finalizing` phase. This is because Azure File cloning does not support Network File System protocol. It supports `azurefile-csi` storage class that uses Server Message Block protocol.
====
