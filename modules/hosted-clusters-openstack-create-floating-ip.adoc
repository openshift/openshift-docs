= Optional: Creating a floating IP for ingress

To get ingress healthy in a hosted cluster without manual intervention, you can create a floating IP that will be used by the ingress service.

.Prerequisites

* You have access to the OpenStack cloud.
* You have identified the external network ID.

.Procedure

. Create a floating IP:
+
[source,terminal]
----
openstack floating ip create <external-network-id>
----
+
[NOTE]
====
If you provide the floating IP to the `--openstack-ingress-floating-ip` flag without pre-creating it, cloud-provider-openstack creates it for you only if the Neutron API policy allows a user to create a floating IP with a specific IP address.
====
