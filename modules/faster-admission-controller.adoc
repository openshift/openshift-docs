= Faster admission controller

[role="_abstract"]

Admission controller performance has been improved by keeping the image cache warm longer, and eliminating image fetches for policies that do not evaluate images.
The default memory limit of admission controller pods is now 1 Gi, which has increased from 500 Mi.
