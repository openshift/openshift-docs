= Manage build retention and cleanup

[role="_abstract"]
You can configure retention parameters to control how long builds and build runs persist in your cluster.
This reference lists retention settings and explains how automatic build run deletion works.
Use these parameters to optimize storage and maintain a clean build history.

This section covers the following topics:

* Retention parameters for a build and buildrun
* Automatic build run deletion
