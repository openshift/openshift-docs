= Installation method consolidation to Operator-based deployment

[role="_abstract"]

// JIRA: ROX-32416
Installation methods have been consolidated to focus on Operator-based deployment as the primary installation approach.

This consolidation simplifies the installation experience and aligns with Kubernetes-native deployment patterns. The {product-title-short} Operator provides a consistent, automated installation and upgrade experience across all supported platforms.
