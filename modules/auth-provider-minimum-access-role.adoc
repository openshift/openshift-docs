= Minimum access role

[role="_abstract"]
{product-title-short} assigns a minimum access role to every caller with a {product-title-short} token issued by a particular authentication provider. The minimum access role defaults to `None`.

For example, suppose there is an authentication provider with the minimum access role of `Analyst`. In that case, all users who log in using this provider will have the `Analyst` role assigned to them.
