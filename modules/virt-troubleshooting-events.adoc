= Events

[role="_abstract"]
To monitor and troubleshoot virtual machine (VM), namespace, and resource issues, you can review {product-title} events. Tracking this life-cycle information helps ensure you maintain a healthy cluster environment.

.Procedure

* To view VM events, go to *VirtualMachine details* -> *Events* in the web console.

* To view namespace events, run the following command:
+
[source,terminal]
----
$ oc get events -n <namespace>
----

* To view resource events, run the following command:
+
[source,terminal]
----
$ oc describe <resource> <resource_name>
----
