= Testing your MCP gateway and Vault integration

[role="_abstract"]
You can test your MCP gateway and HashiCorp Vault (Vault) integration by using the general steps that follow. Tailor your testing for your use case.

.Prerequisites

* You installed the {oc-first}.
* You are logged into an {ocp} cluster as an administrator.
* You set up a trust relationship between Authorino and HashiCorp Vault.
* You created an `AuthPolicy` custom resource.

.Procedure

. Store a secret in Vault by running a command specific to your environment.
+
.Example curl command to store a vault token
[source,terminal,subs="+quotes"]
----
$ curl -s -H "X-Vault-Token: $VAULT_TOKEN" -H 'Content-Type: application/json' -X POST \
  --data '{"data":{"test_server2_pat":"s3cr3t"}}' \
  _<vault_service_url:port>/v1/secret/data/mcp-gateway/<sub>_
----
+
Replace `_<vault_service_url:port>/v1/secret/data/mcp-gateway/<sub>_` with your secret endpoint.

. Get an access token by running a command specific to your environment.
+
.Example access token request
[source,terminal,subs="+quotes"]
----
$ ACCESS_TOKEN=$(curl _<issuer_url>_ -s -d 'grant_type=client_credentials' -d 'client_id=_<mcp_client_id>_' -d 'client_secret=_<mcp_client_secret>_' -d 'scope=openid profile groups roles' | jq -r .access_token)
----
+
* Replace `_<vault_service_url:port>/v1/secret/data/mcp-gateway/<sub>_` with your issuer URL.
* Replace `_<mcp_client_id>_` with your MCP client ID.
* Replace `_<mcp_client_secret>_` with your MCP client secret.

. Start a session with the MCP gateway by running a command specific to your environment.

. Send a request to the MCP server route that requires fetching credentials from Vault by running a command specific to your environment.
+
.Example output
[source,json]
----
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "content": [
      {
        "type": "text",
        "text": "Authorization: [Bearer s3cr3t]"
      },
      …
    ],
    …
  }
}
----
+
The output shows that the request was successful and that the `Authorization` header was set using the Secret retrieved from Vault.
