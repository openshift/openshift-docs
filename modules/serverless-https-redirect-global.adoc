= HTTPS redirection global settings

.Example `KnativeServing` CR that enables HTTPS redirection
[source,yaml]
----
apiVersion: operator.knative.dev/v1beta1
kind: KnativeServing
metadata:
  name: knative-serving
spec:
  config:
    network:
      httpProtocol: "redirected"
...
----
