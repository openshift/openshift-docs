= Configuring {product-title}

[role="_abstract"]
You can configure {product-title} to customize its behavior and integrate it with your infrastructure requirements.

Common configuration tasks include:

* Adding custom TLS certificates so users and API clients do not have to bypass certificate security warnings
* Backing up {product-title} data to enable restoration in case of infrastructure disasters or data corruption
* Configuring automatic upgrades for secured clusters to stay up to date with the latest features and security patches
