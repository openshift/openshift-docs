= Viewing logs

[role="_abstract"]
You can view the logs of a build run to identify any runtime errors and to resolve them.

.Prerequisites

* You have installed the `oc` CLI.
* Optional: You have installed the link:https://console.redhat.com/openshift/downloads[`shp` CLI].

.Procedure

* View logs of a build run by using the `oc` CLI or the `shp` CLI.
+
[source,terminal]
----
$ oc logs <buildrun_resource_name>
----

+
[source,terminal]
----
$ shp buildrun logs <buildrun_resource_name>
----
+
where:

<buildrun_resource_name>:: Specifies the name of the `Buildrun` resource for which you want to view the logs.
