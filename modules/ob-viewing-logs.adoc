= View logs with the CLI

[role="_abstract"]
You can view the logs of a build run to identify any runtime errors and to resolve them.

.Prerequisites

* You have installed the `shp` CLI.

.Procedure

* View logs of a build run:
+
[source,terminal]
----
$ shp buildrun logs <buildrun_resource_name>
----

* For build pods with multiple containers, view logs from all containers:
+
[source,terminal]
----
$ shp buildrun logs <buildrun_resource_name> --all-containers
----

* View logs from a specific container in a multi-container build pod:
+
[source,terminal]
----
$ shp buildrun logs <buildrun_resource_name> -c <container_name>
----
