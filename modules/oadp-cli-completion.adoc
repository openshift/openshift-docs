= Enabling shell completion for the OADP CLI

You can use the {oadp-short} CLI to generate and install shell completion scripts for command auto-completion.

.Prerequisites

* The {oadp-short} CLI plugin is installed.

.Installing shell completions automatically

* Install shell completions for your current shell:
+
[source,terminal]
----
$ oc oadp completion install [flags]
----
+
.Flags for `oc oadp completion install`
[cols="1,3",options="header"]
|===
|Flag |Description

|`--shell`
|Shell type to install completions for. Supported values: `bash`, `zsh`, `fish`. If not specified, the current shell is auto-detected.
|===
+
.Example: installing completions for zsh
[source,terminal]
----
$ oc oadp completion install --shell zsh
----

.Generating shell completion scripts manually

* Generate a completion script for your shell without installing it:
+
[source,terminal]
----
$ oc oadp completion bash
$ oc oadp completion zsh
$ oc oadp completion fish
$ oc oadp completion powershell
----
+
You can redirect the output to a file or source it directly.
+
.Example: writing a bash completion script
[source,terminal]
----
$ oc oadp completion bash > /etc/bash_completion.d/oc-oadp
----
