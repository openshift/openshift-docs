Guided Tours respect a console capability setting::
In earlier releases, the web console always displayed Guided Tours (in-product walkthroughs). This behavior interfered with some automated end-to-end (E2E) tests and some administrative workflows. In {product-title} 4.22, cluster administrators can enable or disable Guided Tours by configuring the console `GuidedTourFeature` capability.
+
