= Supported identities to access {azure-short} resources

[role="_abstract"]
An {product-title} cluster requires an {azure-short} identity to create and manage {azure-short} resources. You need a service principal, a system-assigned managed identity, or a user-assigned managed identity to complete the installation.

* A service principal
* A system-assigned managed identity
* A user-assigned managed identity

For more information on Azure identities, see "Managed identity types".
