= Options

[role="_abstract"]
Options for the `roxctl central db generate k8s` command.

.Options
[cols="2,2",options="header"]
|===
|Option |Description

|`--central-db-image string`
|Specify the Central database image that you want to use. If not specified, the command uses a default value corresponding to the `--image-defaults`.

|`--image-defaults string`
|Specify the default settings for container images. It controls the repositories from which the command downloads the images, the image names and the format of the tags. The default value is `development_build`.

|`--output-dir output directory`
|Specify the directory to which you want to save the deployment bundle. The default value is `central-db-bundle`.
|===
