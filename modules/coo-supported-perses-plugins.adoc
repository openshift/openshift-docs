= Supported Perses plugins

[role="_abstract"]
The following tables list the Perses plugins available in {product-title} and indicate whether each plugin is supported in the current release.
 
Plugins are organized by type: panel, datasource, query, variable, and explore.
 
== Panel plugins
 
Panel plugins define the visualization types available when building dashboards.

|Yes

|Yes
 
|Markdown
|Yes
 
|Pie Chart
|Yes
 
|Scatter Chart
|Yes
 
|Stat Chart
|Yes
 
|Status History Chart
|Yes
 
|Table
|Yes
 
|Time Series Chart
|Yes

|Yes

|Yes

|Yes
|===
