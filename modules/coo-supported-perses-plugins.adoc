= Supported Perses plugins

[role="_abstract"]
The following list the supported Perses plugins in {product-title}.
 
== Panel plugins
 
Panel plugins define the visualization types available when building dashboards.

[cols="1,1",options="header"]
|===
|Plugin | Supported
 
|Bar Chart
|Yes
 
|Flame Chart
|Yes
 
|Gauge Chart
|Yes
 
|Heat Map Chart
|Yes
 
|Histogram Chart
|Yes
 
|Logs Table
|Yes
 
|Markdown
|Yes
 
|Pie Chart
|Yes
 
|Scatter Chart
|Yes
 
|Stat Chart
|Yes
 
|Status History Chart
|Yes
 
|Table
|Yes
 
|Time Series Chart
|Yes
 
|Time Series Table
|Yes
 
|Trace Table
|Yes
 
|Tracing Gantt Chart
|Yes
|===

== Datasource plugins
 
Datasource plugins define the backends that Perses can query.

[cols="1,1",options="header"]
|===
|Plugin | Supported
 
|Loki Datasource
|Yes
 
|Prometheus Datasource
|Yes
 
|Tempo Datasource
|Yes
|===
 
== Query plugins
 
Query plugins define the query types that panels can use to retrieve data from a datasource.

[cols="1,1,1",options="header"]
|===
|Plugin | Query type | Supported

|Loki
|Log Query, Time Series Query
|Yes
 
|Prometheus
|Time Series Query
|Yes

|Tempo Trace Query
|Trace Query
|Yes
|===
 
== Variable plugins
 
Variable plugins define the dynamic variable types available in dashboards. Variables allow users to filter and scope dashboard data interactively.

[cols="1,1",options="header"]
|===
|Plugin | Supported
 
|Datasource Variable
|Yes
 
|Loki Label Names Variable
|Yes
 
|Loki Label Values Variable
|Yes
 
|Loki LogQL Variable
|Yes
 
|Prometheus Label Names Variable
|Yes
 
|Prometheus Label Values Variable
|Yes

|Prometheus PromQL Variable
|Yes

|Static List Variable
|Yes
|===

 
== Explore plugins
 
Explore plugins provide interactive query exploration interfaces within Perses.
 
[cols="1,1",options="header"]
|===
|Plugin | Supported
 
|Prometheus Explorer
|No
 
|Pyroscope Explorer
|No
 
|Tempo Traces Explorer
|No
|===
 
[NOTE]
====
Other plugins not listed above may be available but are not supported in this release of {product-title}. Support status may change in future releases.
====
