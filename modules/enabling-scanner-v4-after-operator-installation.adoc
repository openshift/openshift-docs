= Enabling RHACS Scanner V4 after Operator installation

[role="_abstract"]
Scanner V4 is not enabled by default, but you can enable Scanner V4 after installation. Scanner V4 also requires that the StackRox Scanner remain enabled. Do not disable the StackRox Scanner even if Scanner V4 is enabled.

If you installed {product-title-short} by using the Operator, you must first enable Scanner V4 on the cluster where Central is installed.

.Procedure

. Open the ???? file and edit the following lines:
+
[source,yaml]
----
spec:
  scannerV4:
    scannerComponent: Enabled
----

For more information, see " ".

After you enable Scanner V4 on the cluster where Central is installed, you must enable Scanner V4 on the secured cluster.

.Procedure
. Open the ???? file and edit the file as shown in the following example:
+
[source,yaml]
----
spec:
  scanner:
    scannerComponent: AutoSense
  scannerV4:
    scannerComponent: AutoSense
----
+
For more information, see " ".
