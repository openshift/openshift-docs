= Specifying a pipeline, task, or step action from the same cluster

[role="_abstract"]
You can use the cluster resolver to specify a pipeline, task, or `StepAction` definition in a namespace on the {OCP} cluster where {pipelines-title} runs. In particular, you can use the cluster resolver to access tasks that {pipelines-shortname} provides in its installation namespace, which is normally the `openshift-pipelines` namespace.
