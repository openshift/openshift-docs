= Configure the CLI environment

[role="_abstract"]
When you have the OpenShift CLI installed, learn how to configure your shell environment, manage authentication profiles, and understand multi-cluster access so that you can work efficiently across clusters and sessions.

Quick shell setup:: Enabling tab completion for Bash, Enabling tab completion for Zsh, Accessing kubeconfig by using the oc CLI
Understand oc versus kubectl:: The oc binary, The kubectl binary
Multi-cluster profile management:: About switching between CLI profiles, Manually configuring CLI profiles, Manual configuration of CLI profiles, Load and merge rules
