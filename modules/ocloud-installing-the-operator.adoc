= Install the O-Cloud Manager Operator

[role="_abstract"]
You can install the O-Cloud Manager Operator on the hub cluster by creating a `Subscription` resource that deploys the Operator from the Operator Lifecycle Manager (OLM) catalog.

.Prerequisites

* You have access to the hub cluster as a user with `cluster-admin` privileges.
* You have installed the OpenShift CLI (`oc`).
* {rh-rhacm-first} version 2.15 or later is installed on the hub cluster.
* The SiteConfig Operator is enabled in the `MultiClusterHub` resource.
* A default `StorageClass` that supports `ReadWriteOnce` access mode is configured, with at least 20 Gi of available `PersistentVolume` capacity.

.Procedure

. Create a namespace for the O-Cloud Manager Operator:
+
[source,terminal]
----
$ oc create namespace oran-o2ims
----

. Create an `OperatorGroup` resource in the `oran-o2ims` namespace:
+
[source,yaml]
----
apiVersion: operators.coreos.com/v1
kind: OperatorGroup
metadata:
  name: oran-o2ims
  namespace: oran-o2ims
spec:
  targetNamespaces:
    - oran-o2ims
----

. Create a `Subscription` resource to install the Operator:
+
[source,yaml]
----
apiVersion: operators.coreos.com/v1alpha1
kind: Subscription
metadata:
  name: oran-o2ims
  namespace: oran-o2ims
spec:
  channel: "stable"
  name: oran-o2ims
  source: redhat-operators
  sourceNamespace: openshift-marketplace
----

. Apply the resources:
+
[source,terminal]
----
$ oc apply -f <operatorgroup_filename>.yaml
$ oc apply -f <subscription_filename>.yaml
----

.Verification

. Verify that the Operator is installed and running:
+
[source,terminal]
----
$ oc get pods -n oran-o2ims
----

. Verify that the default `Inventory` resource was created:
+
[source,terminal]
----
$ oc get inventory -n oran-o2ims
----
+
.Example output
[source,terminal]
----
NAME      AGE
default   4m20s
----

. Verify that the API routes are available:
+
[source,terminal]
----
$ oc get routes -n oran-o2ims
----
+
The output shows routes for the infrastructure inventory, provisioning, cluster, artifacts, and monitoring API endpoints.
