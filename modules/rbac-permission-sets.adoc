= System permission sets

[role="_abstract"]
{product-title} includes some default system permission sets that you can apply to roles.
You can also create custom permission sets as required.

[cols="1,3"]
|===
| Permission set | Description

| *Admin*
| Grants read and write access to all resources.

| *Analyst*
| Grants read-only access for all resources.

| *Continuous Integration*
| Use this permission set for CI (continuous integration) systems. It includes the permissions required to enforce deployment policies.

| *Network Graph Viewer*
| Grants the minimum permissions to view network graphs.

| *None*
| Grants no read and write permissions for any resource.

| *Sensor Creator*
| Grants permissions for resources that you need to create Sensors in secured clusters.
|===
