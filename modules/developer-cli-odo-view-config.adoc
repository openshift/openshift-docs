= Viewing the current configuration

You can view the current `odo` CLI configuration by using the following command:

[source,terminal]
----
$ odo preference view
----

.Example output
[source,terminal]
----
PARAMETER             CURRENT_VALUE
UpdateNotification
NamePrefix
Timeout
BuildTimeout
PushTimeout
Ephemeral
ConsentTelemetry      true
----
