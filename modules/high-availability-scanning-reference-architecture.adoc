= High availability scanning reference architecture

[role="_abstract"]

// JIRA: ROX-32124
{product-title-short} 4.11 introduces a reference architecture for high availability vulnerability scanning deployments.

This reference architecture provides guidance on configuring Scanner components for high availability, fault tolerance, and scalability in production environments. The documentation includes recommended configurations, resource requirements, and best practices for deploying Scanner in highly available configurations.
