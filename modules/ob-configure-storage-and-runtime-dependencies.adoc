= Configure storage and runtime dependencies

[role="_abstract"]
You can configure storage volumes and runtime dependencies to manage persistent data during builds and build runs.
The parameters control volume definitions, volume mounts, and their application to both build and build run resources.
Use these configuration options to allocate storage and define access paths in your build environment.

This section covers the following topics:

* Volumes definition for a build
* Volumes definition for a buildrun
* Volumes and volume mounts definition
