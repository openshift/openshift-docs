= List services performance and scalability improvements

[role="_abstract"]

// JIRA: ROX-32635

Performance optimizations were implemented to improve performance and resource consumption of certain endpoints returning `List<Type>` objects. The following endpoints are impacted: 

* /v1/alerts
* /v1/alerts/summary/groups
* /v1/alerts/summary/timeseries
* /v1/secrets
* /v1/deployments
* /v1/deploymentswithprocessinfo
* /v1/images
