= {ols-long} version 1.0.9

{ols-official} 1.0.9 is now available on {ocp-product-title} 4.16 and later.

[id="ols-1-0-9-enhancements_{context}"]
== Enhancements

The following enhancements have been made with {ols-official} 1.0.9:

* With this update, {ols-long} supports automatic updates of BYO Knowledge images that use floating tags, such as `latest`. If over time a BYO Knowledge image tag points to different underlying imageS, {ols-long} detects those changes and updates the corresponding BYO Knowledge database accordingly. This feature is built using OpenShift `ImageStream` objects. {ocp-product-title} clusters check for updates to `ImageStream` objects every 15 minutes.
