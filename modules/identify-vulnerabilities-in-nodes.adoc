= Identifying vulnerabilities in nodes by using the dashboard

[role="_abstract"]
You can identify vulnerabilities in core Kubernetes components and container runtimes in your nodes by using the vulnerability management dashboard.

The identified vulnerabilities include vulnerabilities in core Kubernetes components and container runtimes such as Docker, CRI-O, runC, and containerd. For more information about operating systems that {product-title-short} can scan, see "Supported operating systems".

.Procedure
. In the {product-title-short} portal, go to *Vulnerability Management* -> *Dashboard*.
. Click *Nodes* on the header to view a list of all the CVEs that affect your nodes.
. Click a node from the list to view details of all CVEs that affect that node.
.. When you click a node, the *Node* details panel opens for the selected node.
The *Node* view shows in-depth details of the node and includes information about CVEs by CVSS score and fixable CVEs for that node.
.. Click *View All* on the *CVEs by CVSS score* widget header to view a list of all the CVEs in the selected node.
You can also filter the list of CVEs.
