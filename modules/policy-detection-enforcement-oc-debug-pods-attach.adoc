= Policy-based detection and enforcement for oc debug and pods attach

[role="_abstract"]

// JIRA: ROX-24310
You can now create policies to detect and enforce restrictions on `oc debug` and `pods attach` operations.

This feature enhances runtime security by enabling policy-based controls for debugging and attach operations that could potentially be used to access sensitive container environments. Security teams can define policies that alert on or prevent unauthorized use of these operations.
