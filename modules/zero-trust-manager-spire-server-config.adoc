== Deploying and configuring the SPIRE server

You can configure the `SpireServerConfig` Custom Resource (CR) to deploy and configure a SPIRE server.

.Prerequisites

* You have {zero-trust-full} installed in the cluster.

.Procedure

. Create a YAML file that defines the `SpireServerConfig` CR object:
+
[source,yaml]
----
apiVersion: operator.openshift.io/v1alpha1
kind: SpireServerConfig
metadata:
  name: cluster
spec:
  trustDomain: $APP_DOMAIN <1>
  clusterName: $CLUSTER_NAME
  caSubject:
    commonName: "example.org"
    country: "US"
    organization: "RH"
  persistence:
    type: pvc
    size: "5Gi"
    accessMode: ReadWriteOnce
  datastore:
    databaseType: sqlite3
    connectionString: "/run/spire/data/datastore.sqlite3"
    maxOpenConns: 100
    maxIdleConns: 2
    connMaxLifetime: 3600
----
<1> `BaseDomain` of the cluster.

. Run `oc apply -f <filename>.yaml`

.Verification

. Get the name of the SPIRE server by running the following command:
+
[source,terminal]
----
$ oc get statefulset -l app.kubernetes.io/name=server -n zero-trust-workload-identity-manager
----
+
.Example output
[source,terminal]
----
NAME            READY   AGE
spire-server    1/1     65s
----

. Get the name of the pod by running the following command:
+
[source,terminal]
----
$ oc get po -l app.kubernetes.io/name=server -n zero-trust-workload-identity-manager
----
+
.Example output
[source,terminal]
----
NAME               READY   STATUS    RESTARTS        AGE
spire-server-0     2/2     Running   1 (108s ago)    111s
----

. Get the PersistentVolumeClaim by running the following command:
+
[source,terminal]
----
$ oc oc get pvc -l app.kubernetes.io/name=server -n zero-trust-workload-identity-manager
----
+
.Example output
[source,terminal]
----
NAME                        STATUS    VOLUME                                     CAPACITY   ACCESS MODES  STORAGECLASS  VOLUMEATTRIBUTECLASS  AGE
spire-data-spire-server-0   Bound     pvc-27a36535-18a1-4fde-ab6d-e7ee7d3c2744   1G1        RW0           gp3-csi       <unset>               22m
----
