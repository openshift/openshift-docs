= Machine configuration and storage health checks

[role="_abstract"]
You can verify machine config pool status, persistent volume health, and storage class configuration before starting a cluster update.

Machine config pools::

[source,terminal]
----
$ oc get mcp
----

Pass criteria:

* `UPDATED=True`
* `UPDATING=False`
* `DEGRADED=False`
* `MACHINECOUNT` = `READYMACHINECOUNT` = `UPDATEDMACHINECOUNT`

Persistent volumes::

[source,terminal]
----
$ oc get pv
----

Pass criteria: All PVs show `Bound` or `Available` status, none show `Failed`.

Storage classes::

[source,terminal]
----
$ oc get storageclass
----

Pass criteria: At least one storage class exists and is marked as default (indicated by `(default)` annotation).
