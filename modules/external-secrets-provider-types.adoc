= External secrets providers for the {external-secrets-operator}

[role="_abstract"]
The {external-secrets-operator} is tested with the following external secrets provider types:

* AWS Secrets Manager
* HashiCorp Vault
* Google Secret Manager
* Azure Key Vault
* {ibm-cloud-title} Secrets Manager

[NOTE]
====
Red Hat does not test all factors associated with third-party secrets store provider functionality. For more information about third-party support, see the Red{nbsp}Hat third-party support policy.
====
