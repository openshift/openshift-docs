= Set up a shared secret for secure access

[role="_abstract"]
A `SharedSecret` custom resource (CR) enables secure sharing of `Secret` objects containing sensitive information such as passwords, OAuth tokens, and SSH keys across namespaces within a cluster. This eliminates manual duplication and provides a single source for sensitive data with controlled access.
