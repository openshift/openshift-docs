= Security notices overview

[role="_abstract"]
You can configure security notices to display login messages, headers, and footers in the {product-title-short} portal for policy reminders, disclaimers, and legal notifications.

You can also set up an organization-wide message or disclaimers on the top or bottom of the {product-title-short} portal.

This message can serve as a reminder of corporate policies and tell employees of the appropriate policies.
Alternatively, you might want to display these messages for legal reasons, for example, to warn users that you audit their actions.
