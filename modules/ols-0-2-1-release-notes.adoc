= {ols-long} version 0.2.1

{ols-official} 0.2.1 is now available on {ocp-product-title} 4.15 or later.

[id="ols-0-2-1-enhancements_{context}"]
== Enhancements

The following enhancements have been made with {ols-official} 0.2.1:

* With this release, the {ols-long} User Interface is hidden if the *Hide Lightspeed* option is selected in *User Preferences*.

* This release adds the ability to attach a YAML file to a question you ask on any pages accessible from *Virtualization* in the {ocp-product-title} web console. This release also adds the ability to attach events and logs for `VirtualMachines` and `VirtualMachineInstances`.

* When attaching events to a prompt, the text area of the {ols-long} User Interface now generates a preview of what will be attached. This preview presents the actual YAML text for the event, helping users confirm that attaching the event will provide additional context for the question being asked.

* Beginning in this release you can set the `api_version` parameter for the {azure-openai} provider in the `OLSConfig` Custom Resource.

* This release introduces new parameters in the `OLSConfig` custom resource definition file to disable feedback and transcripts.When the parameters are disabled, the {ols-long} Service does not send feedback and transcripts to {red-hat}, conserving cluster resource usage.

* With this release, the {ols-long} User Interface displays an email links that can be used to contact the {ols-official} Development Team.

* This release adds the `Ciphers` and `MinTLSVersion` parameters to the `OLSConfig` Custom Resource Definition file. The parameters can be used to specify the TLS Security Profile for the {ols-long} stack.
