= Accessing BuildConfig logs

You can access `BuildConfig` logs using the web console or the CLI.

.Procedure

To stream the logs of the latest build for a BuildConfig, run:

[source,terminal]
----
$ oc logs -f bc/<buildconfig_name>
----
