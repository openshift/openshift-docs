= {ols-long} version 1.0

[role="_abstract"]
{ols-official} version 1.0 provides an AI-driven virtual assistant that helps users troubleshoot and manage {ocp-short-name} clusters. This release introduces support for {ocp-product-title} 4.15 and later.

== Enhancements

[role="_abstract"]
{ols-official} 1.0 introduces general availability support for {ocp-product-title} 4.15 and later. This release includes new AI-driven cluster interaction capabilities, security updates, and several Technology Preview features.

The following enhancements are made with {ols-official} 1.0:

--
** Cluster interaction
** PostgreSQL persistence
** Token quota
