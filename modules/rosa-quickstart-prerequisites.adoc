= ROSA cluster prerequisites

[role="_abstract"]
Before creating a {product-title} cluster, you must review the product introduction and architecture documentation, read the environment planning guidelines, verify the detailed AWS prerequisites for STS, and ensure you have the required AWS service quotas.

* You reviewed the introduction to {product-title}, and the documentation on {product-title} architecture models and concepts.

* You have read the documentation on the guidelines for planning your environment.
