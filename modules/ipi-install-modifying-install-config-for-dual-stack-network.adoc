= Deploying static IP addressing with dual-stack networking

Use the following procedure to deploy dual-stack static IP addressing for the bootstrap VM by using Ignition:

.Procedure

. Create the Ignition configuration files:
+
[source,terminal]
----
$ ./openshift-baremetal-install --dir <cluster_configs> create ignition-configs
----
+
Replace `<cluster_configs>` with the path to your cluster configuration files.

. Create the `bootstrap_config.sh` file:
+
[source,terminal]
----
#!/bin/bash

BOOTSTRAP_CONFIG="[connection]
type=ethernet
interface-name=ens3
[ethernet]
[ipv4]
method=manual
addresses=<ipv4_address>/<cidr>
gateway=<gateway_ipv4_address>
dns=<dns_ipv4_address>"

"[connection]
type=ethernet
interface-name=ens4
[ethernet]
[ipv6]
method=manual
addresses=<ipv6_address>/<cidr>
gateway=<gateway_ipv6_address>
dns=<dns_ipv6_address>"

cat <<_EOF_ > bootstrap_network_config.ign
{
  "path": "/etc/NetworkManager/system-connections/ens3.nmconnection",
  "mode": 384,
  "contents": {
    "source": "data:text/plain;charset=utf-8;base64,$(echo "${BOOTSTRAP_CONFIG}" | base64 -w 0)"
  }
}

cat <<_EOF_ > bootstrap_network_config.ign
{
  "path": "/etc/NetworkManager/system-connections/ens4.nmconnection",
  "mode": 384,
  "contents": {
    "source": "data:text/plain;charset=utf-8;base64,$(echo "${BOOTSTRAP_CONFIG}" | base64 -w 0)"
  }
}
_EOF_

mv <cluster_configs>/bootstrap.ign <cluster_configs>/bootstrap.ign.orig

jq '.storage.files += $input' <cluster_configs>/bootstrap.ign.orig --slurpfile input bootstrap_network_config.ign > <cluster_configs>/bootstrap.ign
----
+
Replace `<ipv4_address>` and `<cidr>` with the IPv4 address and CIDR of the address range. Replace `<gateway_ipv4_address>` with the IPv4 address of the gateway on the baremetal network. Replace `<dns_ipv4_address>` with the IPv4 address of the DNS server on the baremetal network.
+
Replace `<ipv6_address>` and `<cidr>` with the IPv6 address and CIDR of the address range. Replace `<gateway_ipv6_address>` with the IPv6 address of the gateway on the baremetal network. Replace `<dns_ipv6_address>` with the IPv6 address of the DNS server on the baremetal network.
+
Replace `<cluster_configs>` with the path to your cluster configuration files.
+
. Make the `bootstrap_config.sh` file executable:
+
[source,terminal]
----
$ chmod 755 bootstrap_config.sh
----
+
. Run the `bootstrap_config.sh` script to create the `bootstrap_network_config.ign` file:
+
[source,terminal]
----
$ ./bootstrap_config.sh
----
