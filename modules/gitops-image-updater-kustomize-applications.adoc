= Configuring images for Kustomize applications

[role="_abstract"]
When replacing one image with another, for example, switching registries, use the `manifestTargets.kustomize.name` field to identify the original image in the kustomization.

.Procedure

. Configure the `manifestTargets.kustomize.name` field to specify the original image name that should be replaced. The `imageName` field specifies the new image to use, and the `name` field under `kustomize` specifies the original image name to replace in the kustomization:
+
--
*Example: Replacing an image for Kustomize applications:*
[source,yaml]
----
spec:
  applicationRefs:
    - namePattern: "my-app"
      images:
        - alias: "argocd"
          imageName: "ghcr.io/argoproj/argocd:latest"
          manifestTargets:
            kustomize:
              name: "quay.io/argoproj/argocd"
----
