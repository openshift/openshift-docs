= Enabling confidential VMs with AMD SEV-SNP

[role="_abstract"]
You can use confidential VMs with AMD Secure Encrypted Virtualization-Secure Nested Paging (AMD SEV-SNP) when installing your cluster. AMD SEV-SNP is a CPU feature that provides memory encryption and integrity protection for virtual machines (VMs). Confidential VMs encrypt data while it is being processed. For more information, see link:https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/sev-snp.html[AMD SEV-SNP ({aws-short} documentation)].

[NOTE]
====
Confidential VMs are currently not supported on 64-bit ARM architectures.
====

.Procedure

* Use a text editor to edit the `install-config.yaml` file before deploying your cluster and add one of the following stanzas:
.. To use confidential VMs for only control plane machines:
+
[source,yaml]
----
controlPlane:
  platform:
    aws:
      type: r6a.xlarge
      cpuOptions:
       confidentialCompute: AMDEncryptedVirtualizationNestedPaging
----
+
where: 

`controlplane.platform.aws.type`:: Specifies a instance type that supports confidential VMs. Confidential VMs require the m6a, c6a, or r6a series of instance types. For more information on supported instance types, see link:https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/snp-find-instance-types.html[Instance types that support AMD SEV-SNP].

`controlPlane.platform.aws.cpuOptions.confidentialCompute`:: Enables confidential VMs for control plane machines. Valid values are `Disabled` and `AMDEncryptedVirtualizationNestedPaging`.
+
.. To use confidential VMs for only compute machines:
+
[source,yaml]
----
compute:
  platform:
    aws:
      type: c6a.xlarge
      cpuOptions:
        confidentialCompute: AMDEncryptedVirtualizationNestedPaging
----
+
where: 

`compute.platform.aws.type`:: Specifies a instance type that supports confidential VMs. Confidential VMs require the m6a, c6a, or r6a series of instance types. For more information on supported instance types, see link:https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/snp-find-instance-types.html[Instance types that support AMD SEV-SNP].

`compute.platform.aws.cpuOptions.confidentialCompute`:: Enables confidential VMs for compute machines. Valid values are `Disabled` and `AMDEncryptedVirtualizationNestedPaging`.
+
.. To use confidential VMs for all machines:
+
[source,yaml]
----
platform:
  aws:
    type: m6a.xlarge
    cpuOptions:
      confidentialCompute: AMDEncryptedVirtualizationNestedPaging
----
+
where: 

`platform.aws.type`:: Specifies a instance type that supports confidential VMs. Confidential VMs require the m6a, c6a, or r6a series of instance types. For more information on supported instance types, see link:https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/snp-find-instance-types.html[Instance types that support AMD SEV-SNP].

`platform.aws.cpuOptions.confidentialCompute`:: Enables confidential VMs for all machines. Valid values are `Disabled` and `AMDEncryptedVirtualizationNestedPaging`.
