= Overview of hosting cluster management

[role="_abstract"]
A hosting cluster that runs virtualized control plane VMs requires ongoing management to maintain the availability and performance of all target clusters it supports.

Hosting cluster management includes the following tasks:

* Upgrading the hosting cluster and {VirtProductName} without disrupting the target clusters.
* Performing node maintenance on hosting cluster nodes while preserving `etcd` quorum in target clusters.

Because multiple target clusters can share the same hosting cluster, a misconfigured upgrade or unplanned maintenance event on the hosting cluster can affect multiple target clusters simultaneously.
Plan and execute all hosting cluster operations with awareness of which target cluster VMs run on which physical nodes.
