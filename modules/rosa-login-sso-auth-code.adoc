= Authenticate the {rosa-cli} with a single sign-on authorization code

[role="_abstract"]
You can log in to your account with the {rosa-cli} by using a Red{nbsp}Hat single sign-on authorization code.

.Prerequisites

* You have a web browser available on your system.
* You have {rosa-cli} version 1.2.36 or later installed.

.Procedure

* To log in to the {rosa-cli-first} with a Red{nbsp}Hat single sign-on authorization code, run the following command:
+
.Syntax
[source,terminal]
----
$ rosa login --use-auth-code
----
+
Running this command redirects you to the Red{nbsp}Hat single sign-on login. Log in with your Red{nbsp}Hat login or email.
+
.Optional arguments inherited from parent commands+
[cols="30,70"]
|===
|Option |Definition

|--help
|Shows help for this command.

|--debug
|Enables debug mode.

|===

.Verification

To switch accounts, log out from link:https://sso.redhat.com[https://sso.redhat.com] and run the `rosa logout` command in your terminal before attempting to log in again.
