= Following the progress of the installation


[role="_abstract"]
Monitor the deployment progress using the installation log.

.Procedure

. Check the installation's overall status by issuing the `tail` command to the `.openshift_install.log` log file in the install directory folder:
+
[source,terminal]
----
$ tail -f /path/to/install-dir/.openshift_install.log
----
