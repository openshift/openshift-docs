= Enabling RHACS Scanner V4 when installing with the Operator

[role="_abstract"]
Scanner V4 is not enabled by default, but you can enable it during installation.

.Procedure

. To enable Scanner V4 on the cluster where Central is installed, follow the installation procedures as described in "Installing Central using the Operator method". Choose one of these methods:
* As described in that procedure, when configuring the available options for Central, go to the *Scanner V4 Component Settings* section and in the *Scanner V4 Component* menu, select *Enabled*.
* In the Central custom resource (CR) YAML, configure the following parameter:
+
[source,yaml]
----
scannerV4.scannerComponent: Enabled
----
. After you enable Scanner V4 on the cluster where Central is installed, you configure your init bundle or cluster registration secret (CRS) to allow communication between Central and the secured cluster. Then, you can enable Scanner V4 on the secured cluster by following the installation procedures as described in "Installing {product-title-short} on secured clusters by using the Operator". Choose one of these methods:
* In the *Scanner V4 Component Settings* section, in the *Scanner V4 Component* menu, select *AutoSense*.
* In the SecuredCluster CR YAML, configure the following parameter:
+
[source,yaml]
----
scannerV4.scannerComponent: AutoSense
----
