= Vulnerability data sources

[role="_abstract"]
Sources for vulnerabilities depend on the scanner your system uses. {product-title-short} has two scanners: StackRox Scanner and Scanner V4. The StackRox Scanner is deprecated. Scanner V4 is the default image scanner.

[NOTE]
====
Although the StackRox Scanner is deprecated, you still must enable it on the cluster where you install Central due to software dependencies.
====
