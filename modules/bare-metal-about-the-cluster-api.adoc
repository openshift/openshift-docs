= About the Cluster API

[role="_abstract"]
The Cluster API provides a standardized way to manage and provision compute nodes after your cluster installation is complete. By using this API, you can dynamically manage machine sets and machines to scale your infrastructure, though it does not support control plane machine management.

{product-title} 4.19 and later releases can manage machines by using the Cluster API.
