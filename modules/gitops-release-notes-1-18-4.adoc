== Fixed issues

ApplicationSet controller permissions updated for AppProjects::
Before this update, the `ApplicationSet` controller lacked permission to list AppProject resources at the cluster scope. As a result, reconciliation failed when `ApplicationSet` resources were created in namespaces other than the default Argo CD namespace. With this release, the controller ClusterRole is updated to include the required permissions for `appprojects.argoproj.io`. This change aligns the configuration with the upstream version and enables the `ApplicationSet` controller to generate applications across all namespaces in the cluster successfully.
