== Fixed issues

Upgrades preserve Argo CD server service certificate annotation::
Before this update, upgrading the {gitops-title} Operator from version 1.12.3 to version 1.12.4 removed the `service.beta.openshift.io/serving-cert-secret-name` annotation from the Argo CD server service. The Service-CA ClusterOperator could no longer manage the service certificate, which could cause HTTP 503 errors after the certificate expired. Because the change remained on the service object, the issue could continue even after you upgraded to later Operator versions. With this update, the {gitops-title} Operator reconciles the `service.beta.openshift.io/serving-cert-secret-name` annotation during upgrades. The Service-CA ClusterOperator can continue managing certificate rotation, helping prevent HTTP 503 errors from expired certificates.
