= About the bundle endpoint profiles

[role="_abstract"]
The bundle endpoint profile controls how your cluster exposes its trust bundle to external parties. You can select specific authentication methods, such as HTTPS Web or HTTPS SPIFFE, to meet your security requirements.

[cols="1,1,1",options="header"]
|===
|Profile
|Authentication
|Key Use Case

|https_spiffe
|SPIFFE-based Transport Layer Security (TLS) authentication
|High security, existing SPIRE partners

|https_web
|Standard Web Publick Key Infrastructure (PKI) (public certificate authority (CA) or Automatic Cerficate Management (ACME))
|External organizations, easy setup
|===
