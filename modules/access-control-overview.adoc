= Access control

[role="_abstract"]
User accounts are managed with role-based access control (RBAC), and Red{nbsp}Hat site reliability engineers (SREs) have controlled access to Central instances through OpenShift RBAC.

User accounts are managed with role-based access control (RBAC). See "Managing RBAC in Red{nbsp}Hat Advanced Cluster Security for Kubernetes" for more information. Red{nbsp}Hat site reliability engineers (SREs) have access to Central instances. Access is controlled with OpenShift RBAC. Credentials are instantly revoked upon termination.
