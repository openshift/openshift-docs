= Supported {rhoai} custom resource definitions

The following {rhoai} custom resource definitions (CRDs) are supported:

* `InferenceServices`
* `TrainedModels`
* `ServingRuntimes`
* `InferenceGraphs`
* `ClusterStorageContainers`
* `ClusterLocalModels`
* `LocalModelNodeGroups`
