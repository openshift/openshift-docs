= Targeting a Gateway

[role="_abstract"]
When a `PlanPolicy` object targets a Gateway, {prodname} enforces the plan-based rate limits on all routes attached to that Gateway, as shown in the following example.

.Example enforcement of plan-based rate limits on a Gateway

[source,yaml]
----
apiVersion: extensions.kuadrant.io/v1alpha1
kind: PlanPolicy
metadata:
  name: _<my-gateway-plan>_
  namespace: _<my-namespace>_
spec:
  targetRef:
    group: gateway.networking.k8s.io
    kind: Gateway
    name: _<my-gateway>_
  plans:
    # ...
----
* Replace _<my-gateway-plan>_ with the name of your gateway plan.
* Replace _<my-namespace>_ with your namespace._<my-route-plan>_
* Replace _<my-gateway>_ with the name of the Gateway.
