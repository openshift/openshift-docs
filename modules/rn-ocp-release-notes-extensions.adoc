Deploying cluster extensions that use webhooks (Technology Preview)::
* With this release, you can deploy cluster extensions that use webhooks on clusters with the `TechPreviewNoUpgrade` feature set enabled. For more information, see xref:../extensions/ce/managing-ce.adoc#olmv1-supported-extensions_managing-ce[Supported extensions].
