[id="roxctl-central-init-bundles-options_{context}"]
= Options

[role="_abstract"]
Options for the `roxctl central init-bundles` command.

.Options
[cols="2,2",options="header"]
|===
|Option |Description

|`--retry-timeout duration`
|Specify the timeout after which the command retries API requests. A value of `0s` means that the command waits for the entire request duration without retrying. The default value is `20s`.

|`-t`, `--timeout duration`
|Specify the timeout for API requests representing the maximum duration of a request. The default value is `1m0s`.
|===
