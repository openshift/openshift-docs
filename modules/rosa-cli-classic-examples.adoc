= Required minimum permissions for ROSA CLI (Classic cluster-specific)

The following minimum required permissions for ROSA CLI commands are specific to Classic clusters.


[id="rosa-min-permissions-required-classic_{context}"]
== Create cluster

Required minimum permissions to create Classic clusters

.Input
[source,terminal]
----
$ rosa rosa create cluster
----
.Minimum permissions
[source,json]
----

{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "VisualEditor0",
            "Effect": "Allow",
            "Action": [
                "iam:GetRole",
                "iam:ListRoleTags",
                "iam:ListRoles"
            ],
            "Resource": "*"
        }
    ]
}

----

[id="rosa-create-account-operator-roles-classic_{context}"]
== create account-roles and operator-roles in auto mode

Required minimum permissions to create account and operator roles in auto mode.

.Input
[source,terminal]
----
$ rosa create account-roles --mode auto
----
.Minimum permissions
[source,json]
----

{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "VisualEditor0",
            "Effect": "Allow",
            "Action": [
                "iam:GetRole",
                "iam:UpdateAssumeRolePolicy",
                "iam:ListRoleTags",
                "iam:GetPolicy",
                "iam:TagRole",
                "iam:ListRoles",
                "iam:CreateRole",
                "iam:AttachRolePolicy",
                "iam:TagPolicy",
                "iam:CreatePolicy",
                "iam:ListPolicyTags"
            ],
            "Resource": "*"
        }
    ]
}

----
