= Installing {sno-okd} using the Assisted Service

[role="_abstract"]
You can install {sno-okd} using the Assisted Service or you can generate an installation ISO using `openshift-installer`. To install {sno-okd} with the Assisted Service, use the OKD configuration in the assisted-service repository.
