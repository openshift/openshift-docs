= roxctl central db restore usage

[role="_abstract"]
Usage syntax for the `roxctl central db restore` command.

.Usage
[source,terminal]
----
$ roxctl central db restore <file> [flags]
----

where:

--
`<file>`:: Specifies the database backup file that you want to restore.
--
