= AllocationClaim resource

`AllocationClaim` is a namespaced CRD that records which NVIDIA Multi-Instance GPU (MIG) slice is to be prepared for a pod. Claims start in the `staged` state and transition to `created` when all requests are satisfied. Each claim stores the GPU UUID, slice position, and pod reference.
