= Telco RAN DU architecture overview

The reference configurations in this document are deployed using a centrally managed hub cluster infrastructure as shown in the following image.
The reference configuration includes the configuration of the far edge clusters and hub cluster components.

*Note:* Support for an ARM based platform is currently a Developer Preview feature for {product-title} 4.19.
All information is subject to change on release.
Please see the following Red Hat knowledge base article for more details: https://access.redhat.com/articles/7118870[How to use ARM platform hardware in OpenShift 4.19 Telco RDS (Developer Preview)]
