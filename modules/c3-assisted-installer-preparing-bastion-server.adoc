= Preparing the {oci} Bastian server

By implementing a bastion host, you can securely and efficiently manage access to your {oci-c3-no-rt} resources, ensuring that your private instances remain protected and accessible only through a secure, controlled entry point.

.Prerequisites

* See link:https://www.oracle.com/a/otn/docs/compute_cloud_at_customer_agent_based_installation.pdf?source=:em:nl:mt::::PCATP[Bastion server - prerequisites].

.Procedure

. Install the bastion server. For details, see link:https://app.slack.com/client/E030G10V24F/D05CCFA8UVB[Bastion Installation].

. Install the Terraform script. For details, see link:https://app.slack.com/client/E030G10V24F/D05CCFA8UVB[Terraform installation].

. Install and configure the {oci} command line interface (CLI). For details, see link:https://app.slack.com/client/E030G10V24F/D05CCFA8UVB[Installing and Configuring the {oci} CLI].

.Additional resources

* link:https://docs.oracle.com/en-us/iaas/Content/API/SDKDocs/cliinstall.htm[Quickstart - Installing the CLI (Oracle documentation)].
