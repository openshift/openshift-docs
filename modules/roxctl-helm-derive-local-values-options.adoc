= Options

[role="_abstract"]
Options for the `roxctl helm derive-local-values` command.

[cols="2,2",options="header"]
|===
|Option |Description

|`--input string`
|Specify the path to the file or directory containing the YAML input.

|`--output string`
|Define the path to the output file.

|`--output-dir string`
|Define the path to the output directory.

|`--retry-timeout duration`
|Set the timeout before retrying API requests. The timeout waits for the entire request duration before initiating a retry. The default value is `20s`.

|`-t`, `--timeout duration`
|Set the timeout for API requests representing the maximum duration of a request. The default value is `1m0s`.
|===
