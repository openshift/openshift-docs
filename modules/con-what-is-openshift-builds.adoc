= What is OpenShift Builds

[role="_abstract"]
{builds-shortname} is a Shipwright-based framework for building container images on {ocp-product-title}. It provides developers with automated image creation and administrators with secure, reusable build strategies.

{builds-shortname} addresses the need for standardized, secure image creation in containerized environments.
It simplifies the build process by providing pre-configured strategies while allowing customization when needed.
