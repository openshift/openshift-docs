[id="rhcl-supported-configs_{context}"]
= Supported configurations with {product-title}

[role="_abstract"]
Connectivity Link must run on a supported combination of {ocp} with {rh} {service-mesh} as the Gateway API provider, and use the {cert-manager}. {rh} provides both production and development support for supported configurations and tested integrations according to your subscription agreement.
