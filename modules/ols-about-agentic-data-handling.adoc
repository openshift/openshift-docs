= Data handling

[role="_abstract"]
Cluster state data is processed by your configured LLM provider during autonomous agentic troubleshooting operations.

During agent operations, the following data is transmitted to the configured LLM provider:

* Kubernetes resource metadata from investigated namespaces
* Prometheus alert data and related metrics
* Cluster telemetry and infrastructure conditions
* Diagnostic artifacts collected during investigation

[IMPORTANT]
====
Use of the autonomous agentic troubleshooting feature is subject to the acceptable use policy (AUP) of your configured LLM provider. Ensure that your LLM provider selection and configuration comply with your organization's data governance and privacy requirements.
====
