= Cancel a `BuildRun`

[role="_abstract"]
Stop an active `BuildRun` resource by setting its `state` field to `BuildRunCanceled`. Canceling a `BuildRun` also cancels the underlying `TaskRun` resource.

.Procedure

* Run the following command to cancel a running build:
+
[source,terminal]
----
$ oc patch buildrun <buildrun_name> --type=merge -p '{"spec":{"state":"BuildRunCanceled"}}'
----
+
where:

<buildrun_name>:: Specifies the name of the `BuildRun` resource to cancel.

.Verification

* Run the following command to verify the build run was canceled:
+
[source,terminal]
----
$ oc get buildrun <buildrun_name> -o jsonpath='{.spec.state}'
----
+
Example output:
[source,terminal]
----
BuildRunCanceled
----
