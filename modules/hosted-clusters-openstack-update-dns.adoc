= Updating the DNS record for ingress
// TODO: Do I need this?
If you use a pre-defined floating IP for ingress, you must create a DNS record for the wildcard domain.

.Prerequisites

* You have created a floating IP for ingress.
* You have access to your DNS provider.

.Procedure

. Create a DNS record for the following wildcard domain that points to the ingress floating IP:
+
[source,text]
----
*.apps.<cluster-name>.<base-domain>
----
+
where:
+
--
* `<cluster-name>` is the name of your hosted cluster.
* `<base-domain>` is your base domain.
--
