= Rollback and emergency stop

[role="_abstract"]
Rollback is not guaranteed if execution is stopped while it is in progress. A partially executed remediation can leave your cluster in an inconsistent state.

[IMPORTANT]
====
If you stop a remediation execution while it is in progress, some remediation steps might have already been applied to your cluster. The agent does not automatically undo completed steps.
====
