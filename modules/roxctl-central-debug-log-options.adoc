[id="roxctl-central-debug-log-options_{context}"]
= Options

[role="_abstract"]
Options for the `roxctl central debug log` command.

.Options
[cols="2,2",options="header"]
|===
|Option |Description

|`-l`, `--level string`
|Specify the log level to which you want to set the modules. Valid values include `Debug`, `Info`, `Warn`, `Error`, `Panic`, and `Fatal`.

|`-m`, `--modules strings`
|Specify the modules to which you want to apply the command.

|`--retry-timeout duration`
|Specify the timeout after which the command retries API requests. A value of zero means that the command waits for the entire request duration without retrying. The default value is `20s`.

|`-t`, `--timeout duration`
|Specify the timeout for API requests, which is the maximum duration of a request. The default value is `1m0s`.
|===
