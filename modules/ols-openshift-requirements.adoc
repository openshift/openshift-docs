= OpenShift Requirements 

{ols-long} requires {ocp-product-title} 4.15 or later running on x86 hardware. Any installation type or deployment architecture is supported so long as the cluster is 4.15+ and x86-based.

Telemetry is enabled on {ocp-product-title} clusters by default. If you are using a standard installation process for all {ocp-short-name} clusters in your environment, confirm that the process does not disable telemetry.
