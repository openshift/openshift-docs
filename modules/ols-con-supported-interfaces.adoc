= Supported interfaces

[role="_abstract"]
{ols-long} categorizes programmatic and user interactions by their stability level so that you can determine the support and reliability of specific features. Use these categories to identify which APIs and interfaces are stable for production or intended for testing.

{ols-long} provides the following interfaces:

* *REST API*: The {ols-long} REST API, hosted at the `/v1/` path, is the stable and supported interface for programmatic interactions.

* *Gradio UI and internal endpoints*: The Gradio UI (`/ui`) and all internal endpoints are for development and debugging only. These interfaces are unstable. You must not use these interfaces for programmatic purposes.
