= Verifying the MetalLB installation

[role="_abstract"]
Verify that all MetalLB components are installed and configured correctly by checking the Operator, controller, speaker, address pools, and advertisements.

.Prerequisites

* Install the {oc-first}.
* Log in as a user with `cluster-admin` privileges.

.Procedure

. Verify the Operator CSV is in the `Succeeded` phase:
+
[source,terminal]
----
$ oc get csv -n metallb-system -o custom-columns="NAME:.metadata.name,PHASE:.status.phase"
----

. Verify the MetalLB instance exists:
+
[source,terminal]
----
$ oc get metallb -n metallb-system
----

. Verify the controller deployment is available:
+
[source,terminal]
----
$ oc get deployment controller -n metallb-system -o custom-columns="NAME:.metadata.name,READY:.status.readyReplicas,AVAILABLE:.status.availableReplicas"
----

. Verify the speaker daemon set is running on all nodes:
+
[source,terminal]
----
$ oc get daemonset speaker -n metallb-system -o custom-columns="NAME:.metadata.name,DESIRED:.status.desiredNumberScheduled,READY:.status.numberReady"
----
+
The `READY` count must match the `DESIRED` count.

. Verify that IP address pools are configured:
+
[source,terminal]
----
$ oc get ipaddresspool -n metallb-system -o custom-columns="NAME:.metadata.name,ADDRESSES:.spec.addresses[*]"
----

. Verify that advertisements are configured:
+
[source,terminal]
----
$ oc get l2advertisement,bgpadvertisement -n metallb-system
----

. Optional: Create a test `LoadBalancer` service to confirm that MetalLB assigns an external IP address:
+
[source,terminal]
----
$ oc create deployment test-web --image=registry.access.redhat.com/ubi9/httpd-24
$ oc expose deployment test-web --type=LoadBalancer --port=8080
----
+
.. Verify an external IP is assigned:
+
[source,terminal]
----
$ oc get svc test-web
----
+
The `EXTERNAL-IP` column must show an IP address from the configured pool within a few seconds.
+
.. Clean up the test resources:
+
[source,terminal]
----
$ oc delete svc test-web
$ oc delete deployment test-web
----

.Expected results
[cols="2,2",options="header"]
|===
| Component | Expected state

| Operator CSV
| `Succeeded`

| MetalLB instance
| Created

| Controller deployment
| 1/1 available

| Speaker daemon set
| Running on all nodes (`READY` equals `DESIRED`)

| IPAddressPool
| Created with configured addresses

| L2Advertisement or BGPAdvertisement
| Created and referencing the correct pool

| Test LoadBalancer service
| `EXTERNAL-IP` assigned from the configured pool
|===
