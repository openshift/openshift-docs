= Verifying the acceptance of the PlanPolicy CR

[role="_abstract"]
You can confirm the acceptance of the `PlanPolicy` custom resource (CR) by running a command.

.Procedure

. Confirm that the `PlanPolicy` CR was accepted by running the following command:
+
[source,terminal,subs="+quotes"]
----
$ oc get planpolicy _<sample_app>_ -n ${KUADRANT_DEVELOPER_NS} -o jsonpath='{.status.conditions[?(@.type=="Accepted")].message}{"\n"}{.status.conditions[?(@.type=="Enforced")].message}'
----
+
* Replace `_<sample_app>_` with the name of your application.
* Replace `${KUADRANT_DEVELOPER_NS}` with the namespace for your object.
+
.Expected output

[source,terminal,subs="+quotes"]
----
PlanPolicy has been accepted
PlanPolicy has been successfully enforced
----
