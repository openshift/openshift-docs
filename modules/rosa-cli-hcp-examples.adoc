= Required minimum permissions for ROSA CLI (HCP cluster-specific)

The following required minimum permissions for ROSA CLI commands are specific to HCP clusters.

[id="rosa-create-hcp-cluster_{context}"]
== create cluster

Required minimum permissions to create HCP clusters.

.Input
[source,terminal]
----
$ rosa create cluster --hosted-cp
----
.Minimum permissions
[source,json]
----
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "VisualEditor0",
            "Effect": "Allow",
            "Action": [
                "iam:GetRole",
                "iam:ListRoleTags",
                "iam:ListAttachedRolePolicies",
                "iam:ListRoles",
                "ec2:DescribeSubnets",
                "ec2:DescribeRouteTables"
            ],
            "Resource": "*"
        }
    ]
}
----

[id="rosa-create-account-operator-roles-hcp_{context}"]
== create account-roles and operator-roles in auto mode

Required minimum permissions to create account and operator roles in auto mode.

.Input
[source,terminal]
----
$ rosa create account-roles --mode auto
----
.Minimum permissions
[source,json]
----

{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "VisualEditor0",
            "Effect": "Allow",
            "Action": [
                "iam:GetRole",
                "iam:UpdateAssumeRolePolicy",
                "iam:ListRoleTags",
                "iam:GetPolicy",
                "iam:TagRole",
                "iam:ListRoles",
                "iam:CreateRole",
                "iam:AttachRolePolicy",
                "iam:ListPolicyTags"
            ],
            "Resource": "*"
        }
    ]
}


----
