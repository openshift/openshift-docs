= Federation field reference

[role="_abstract"]
Reference the federation configuration fields to customize your `SpireServer` resource. This table provides details on field types, required values, and defaults to help you fine-tune your federation setup.

[cols="1,1,1,1,1",options="header"]
|===
|Field
|Type
|Required
|Default
|Description

|`federation.bundleEndpoint.profile`
|enum
|No
|https_spiffe
|Authentication profile for our endpoint.

|`federation.bundleEndpoint.httpsWeb.acme.domainName`
|string
|Conditional
|
|Domain name for ACME certs.

|`federation.federatesWith[].trustDomain`
|string
|Yes
|
|Remote trust domain.

|`federation.federatesWith[].bundleEndpointUrl`
|string
|Yes
|
|Remote bundle URL.
|===
