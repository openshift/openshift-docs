= Installation workflow

[role="_abstract"]
Follow this order to ensure a successful deployment:

. *Plan and prepare* (complete BEFORE installing operators):
   - Verify cluster prerequisites (StorageClass availability)
   - Configure object storage backend with TLS certificates
   - Calculate deployment sizing based on log volume
   - Verify cluster has sufficient CPU, memory, and storage capacity

. *Install operators*:
   - Install the {loki-op}
   - Install the {clo}

. *Configure and deploy*:
   - Create `LokiStack` custom resource with correct size
   - Create `ClusterLogForwarder` custom resource for log collection
   - Verify log collection and storage
