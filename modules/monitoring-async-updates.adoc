= Asynchronous errata updates

[role="_abstract"]
Security, fixed issues, and enhancement updates for {ocp} {product-version} are released as asynchronous errata through the Red{nbsp}Hat Network. This includes updates for the monitoring stack and other components.

For more information about monitoring updates, see link:https://docs.redhat.com/en/documentation/openshift_container_platform/4.18/html/release_notes/ocp-4-18-release-notes#ocp-4-18-asynchronous-errata-updates_release-notes[Asynchronous errata updates].
