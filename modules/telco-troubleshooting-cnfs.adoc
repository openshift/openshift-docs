= Understanding cloud-native functions

If you are starting to use OpenShift Container Platform for telecommunications cloud-native network function (CNF) applications, learning about cloud-native functions (CNFs) can help you understand the issues that you might encounter.

To learn more about CNFs and their evolution, see link:https://www.redhat.com/en/topics/cloud-native-apps/vnf-and-cnf-whats-the-difference[VNF and CNF, what’s the difference?].
