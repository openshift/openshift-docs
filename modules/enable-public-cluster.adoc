= Enable an existing private cluster to be public

// TODO: These wordings of "enabling the cluster "to be public/private" could probably be improved. At the very least, these two modules should probably use "Configuring" instead of "Enabling", as it is worded now.

[role="_abstract"]
Configure an existing private cluster to be public by allowing API endpoint access from the internet. This enables access to your cluster without requiring private connection configuration.

.Procedure

. Log in to {cluster-manager-url}.

. Select the private cluster you want to make public.

. On the *Networking* tab, deselect *Make API private* under *Control Plane API endpoint*.

. Click *Change settings*.
+
[NOTE]
====
Transitioning your cluster between private and public can take several minutes to complete.
====

.Verification

* In the {cluster-manager} console, navigate to your cluster and verify that the *Control Plane API endpoint* on the *Networking* tab no longer shows *Private*.
* Verify that the cluster API is publicly accessible.
