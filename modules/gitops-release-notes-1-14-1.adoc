[id="fixed-issues-1-14-1_{context}"]
== Fixed issues

* Before this update, resource limits set on the operator container led to functional issues on clusters containing a large number of `Secrets` and `ConfigMaps`. As a result, the operator container exceeded its allocated memory limits. With this update, the resource limits set on the operator container are removed to enhance functionality. link:https://issues.redhat.com/browse/GITOPS-5665[GITOPS-5665]
