= Types of PVs

[role="_abstract"]
To determine the best storage solution for your cluster workloads, review the available types of `PersistentVolumes` (PVs). {product-name} supports various storage plugins, including the Container Storage Interface (CSI), NFS, and local storage, so that you can match storage characteristics to your application requirements.
