= Navigating the Agentic Runs view

[role="_abstract"]
Use the Agentic Runs view in the {ocp-product-title} web console to monitor the status of agent-initiated investigations and remediation proposals.

.Prerequisites

* You are logged in to the {ocp-product-title} web console as a cluster administrator.
* The {ols-long} Agentic Operator is installed and configured.
* At least one alert subscription policy is configured.

.Procedure

. In the {ocp-product-title} web console, navigate to the *Agentic Runs* view.

. Review the list of agentic runs. Each run displays its current status:
+
--
* *Analyzing*: The agent is actively investigating the alert and forming a diagnosis.
* *Proposed*: The agent has completed its analysis and is presenting a diagnosis and remediation plan for your review.
* *Completed*: The remediation has been executed and the run is finished.
--

. Click a run to drill down into its details, including the alert that triggered the investigation, the diagnostic steps taken, and the proposed remediation.
