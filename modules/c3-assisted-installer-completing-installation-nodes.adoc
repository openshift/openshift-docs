= Assigning node roles

Delete the disconnected hosts and assign roles to the remaining hosts. Rename any hosts with names shorter than 63 characters.

.Procedure

. From the link:link:https://console.redhat.com/openshift/assisted-installer/clusters[{ai-full} web console], go to the *Host discovery* page. 
+
If the Terraform scripts completed successfully, twelve hosts are listed on the page. Six hosts have the status *Disconnected* caused when you restarted the isntance on C3.

. Delete the six hosts with a *Disconnected* status, by clicking the option button for the host and selecting *Remove host*. The status of the remaining hosts changes from *Insufficient* to *Ready*. This can take up to three minutes.

. From the *Role* column, assign the three nodes with a boot size of 1TB the *Control plane* role, and the three nodes with boot size 100GB the *Worker* role. 

. Rename the hosts with a name shorter than 63 characters, by clicking the option button for the host and selecting *Change hostname*. Otherwise the cluster installation will fail. 

. Click *Next*. 

. On the *Storage* page, click *Next*.
