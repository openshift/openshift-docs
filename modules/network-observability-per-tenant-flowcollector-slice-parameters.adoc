= FlowCollectorSlice configuration parameters

[role="_abstract"]
Configure the `FlowCollectorSlice` resource parameters to customize namespace-level flow collection with varying sampling rates and `subnetLabels`.

The `FlowCollectorSlice` resource supports a subset of the global observability configurations. See "FlowCollectorSlice API reference".
