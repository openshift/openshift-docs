= S3 API compatible integration overview

[role="_abstract"]
The backup includes the entire {product-title-short} database, which has all configurations, resources, events, and certificates. Make sure to store backups securely.

[IMPORTANT]
====
* To back up to Amazon S3, use the dedicated Amazon S3 integration to ensure the best compatibility.
* Red{nbsp}Hat does not test this integration with every S3 API compatible provider, so the integration is not guaranteed to work with all providers.
====
