= Embed knowledge into the LLM

{red-hat} provides a tool that you can use to embed knowledge into the large language model (LLM) that the {ols-long} Service uses, customizing responses for your environment. For example, if a network administrator creates standard operating procedures (SOP) for {ocp-product-title} cluster provisioning the information contained in the SOPs can augment the pre-trained knowledge that the LLM uses based on the information that exists in the {ocp-product-title} product documentation.
