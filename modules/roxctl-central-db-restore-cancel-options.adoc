= Options

[role="_abstract"]
Options for the `roxctl central db restore cancel` command.

.Options
[cols="2,2",options="header"]
|===
|Option |Description

|`f`, `--force`
|If set to `true`, proceed with the cancellation without confirmation. The default value is `false`.
|===
