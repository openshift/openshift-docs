= Creating a {product-title} cluster using FIPS encryption

[role="_abstract"]
You can create a {product-title} cluster with Federal Information Processing Standards (FIPS) encryption and a customer-provided KMS key for node root volumes, the etcd database, or both. A different KMS key ARN can be provided for each option.

[NOTE]
====
{product-title} does not automatically configure the `default` storage class to encrypt persistent volumes with the customer-provided KMS key. You can configure this in-cluster after installation.
====
