= MCP server on {product-title} overview

[role="_abstract"]
When there is a problem on your {product-title} cluster, you want to determine exactly what is happening, so that you can fix the issue as soon as possible. The MCP server for Red Hat {product-title} feature provides an AI tool for this purpose to quickly and easily diagnose your {product-title} cluster.
