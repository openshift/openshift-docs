= About EntraID OIDC


EntraID is a cloud-based intentity and access management service that centralizes user management and access control. EntraID servers as the identify provider, verifying user identities and issuing and ID token to the application. This token contains essential user information, allowing the application to confirm who the user is without managing their credentials.
