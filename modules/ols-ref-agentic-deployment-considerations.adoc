= Deployment considerations

[role="_abstract"]
Review the following deployment considerations for the autonomous agentic troubleshooting feature.

.Deployment support matrix
[cols="1,1",options="header"]
|===
| Configuration
| Support

| Deployment model
| Self-managed

| Classic (standalone cluster)
| Supported

| Hosted control planes
| Supported

| Cluster topology
| Multi-node, compact (three-node), and single-node (SNO)

| Network connectivity
| Connected and restricted networks

| Architectures
| x86_64, ARM (aarch64), IBM Power (ppc64le), IBM Z (s390x)

| Operator compatibility
| All

| Minimum version
| {ocp-product-title} 5.0 and later

| User interface
| {ocp-product-title} web console with dynamic plugin
|===

[NOTE]
====
This feature is opt-in. It is available only if you install and enable the {ols-long} Agentic Operator.
====
