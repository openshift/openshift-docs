= Loki network policies for added security

[role="_abstract"]
The {loki-op} can deploy and manage a set of network policies that restrict communications to and from Loki components to enhance security. These network policies control ingress and egress traffic at the pod level, limiting exposure to only necessary services while allowing integration with external monitoring systems when required.
