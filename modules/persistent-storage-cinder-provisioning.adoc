= Manual provisioning with Cinder

[role="_abstract"]
Storage must exist in the underlying infrastructure before it can be mounted as a volume in {product-title}.

Manual provisioning requires that {product-title} is configured for {rh-openstack-first} and that you have the Cinder volume ID.
