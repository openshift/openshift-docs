= Check your deployment model

Your deployment model determines your migration path.

.Procedure

. Find your deployment model by running the following command:
+
[source,terminal]
----
oc get smcp <smcp-name> -n <smcp-namespace> -o jsonpath='{.spec.mode}'
----

If you are not using cert-manager, then continue to the appropriate next step for your deployment model.
