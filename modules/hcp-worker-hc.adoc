= Creating worker nodes for the hosted cluster

If you are working on a bare metal platform, you must create worker nodes to ensure that the details in the `BareMetalHost` are correctly configured.

You can add Bare Metal Hosts to the cluster by using a YAML file that describes the Bare Metal Host.

.Prerequisites

* You have installed a {op-system-first} compute machine on bare-metal infrastructure for use in the cluster.
* You are logged in as a user with `cluster-admin` privileges.
* You have created a `Secret` custom resource (CR) for the Bare Metal Host.

.Procedure

. In the web console, navigate to *Compute* -> *Bare Metal Hosts*.
. Select *Add Host* -> *New from YAML*.
. Enter the details of your host. For more information about what to enter, see "Provisioning new hosts in a user-provisioned cluster by using the BMO".
+
.Example YAML source with host details
[source,yaml]
----
apiVersion: metal3.io/v1alpha1
kind: BareMetalHost
metadata:
  name: <bare_metal_host_name>
spec:
  online: true
  bmc:
    address: <bmc_address>
    credentialsName: <secret_credentials_name>  <1>
    disableCertificateVerification: True <2>
  bootMACAddress: <host_boot_mac_address>
----
+
<1> `credentialsName` must reference a valid `Secret` CR. The `baremetal-operator` cannot manage the Bare Metal Host without a valid `Secret` referenced in the `credentialsName`. For more information about secrets and how to create them, see "Understanding secrets".
<2> Setting `disableCertificateVerification` to `true` disables TLS host validation between the cluster and the baseboard management controller (BMC).

. Select *Create* to save the YAML and create the new Bare Metal Host.
. Scale up the number of replicas to match the number of available Bare Metal Hosts. Navigate to *Compute* -> *MachineSets*, and increase the number of machines in the cluster by selecting *Edit Machine count* from the *Actions* drop-down menu.
+
[NOTE]
====
You can also manage the number of bare-metal nodes using the `oc scale` command and the appropriate bare-metal compute machine set.
