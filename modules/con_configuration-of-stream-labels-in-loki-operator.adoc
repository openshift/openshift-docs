= Configuration of stream labels in Loki Operator

Configuring which labels the {loki-op} will use as stream labels depends on the data model you are using: Viaq or OLTP. 

Both models already come with a pre-defined set of stream labels, for more information, see https://docs.redhat.com/en/documentation/red_hat_openshift_logging/latest/html/configuring_logging/opentelemetry-data-model[OpenTelemetry data model]

ViaQ model::
To configure stream labels for the ViaQ model, add the configuration in the `ClusterLogForwarder` resource.  
ViaQ does not support structured metadata.

OTLP model::
This model supports structured metadata, all labels not specified as stream labels will be attached as structured metadata. If you want to configure stream labels this is done in the tenant config of LokiStack.
