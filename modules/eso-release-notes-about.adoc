= About {external-secrets-operator} release notes

[role="_abstract"]
The {external-secrets-operator} is a cluster-wide service that provides lifecycle management for secrets fetched from external secret management systems. These release notes track the development of {external-secrets-operator-short}. For more information, see xref:../../security/external_secrets_operator/index.adoc#external-secrets-operator-about[{external-secrets-operator-short} overview].
