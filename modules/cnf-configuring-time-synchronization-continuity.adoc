= Configuring time synchronization continuity

Telco Operators and SREs require tools to ensure time synchronization continuity and system stability in environments that demand extremely high time accuracy, typically within 1.5 microseconds. To meet these requirements, {product-title} provides redundancy against potential GNSS signal loss (the primary time source) by automatically transitioning to the Network Time Protocol (NTP) as a secondary source.
