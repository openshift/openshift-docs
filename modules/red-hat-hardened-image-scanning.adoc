= Red Hat hardened image scanning

[role="_abstract"]
Red{nbsp}Hat Advanced Cluster Security Clair can scan Red Hat hardened images (Project Hummingbird) and cross-reference findings with Red Hat VEX metadata, supporting validation and ongoing monitoring of hardened image CVE posture.

[id="known-issue-hummingbird-scan_{context}"]
== Known issue as of July 15, 2026

Due to a recently identified data gap in the Red Hat VEX security data feed, {product-title-short} version 4.11 no longer supports vulnerability reporting for Red Hat hardened images (Project Hummingbird). The Red Hat Product Security Team is working to address this identified data gap so that {product-title-short} can re-introduce support for Red Hat hardened images.
