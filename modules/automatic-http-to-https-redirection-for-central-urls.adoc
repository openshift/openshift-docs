= Automatic HTTP to HTTPS redirection for Central URLs

//ROX-31013
[role="_abstract"]
In {product-title-short} 4.10, you are automatically redirected from `HTTP` to `HTTPS` when accessing the Central URL. Before this update, the system did not automatically redirect `HTTP` requests. As a result, this enhancement improves security, user experience, and consistency with {osp} platform standards.

For example, if you enter `\http://central-stackrox.apps.ocp.example.com`, Central automatically redirects the connection to `\https://central-stackrox.apps.ocp.example.com`.
