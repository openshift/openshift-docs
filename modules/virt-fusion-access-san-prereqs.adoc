= Prerequisites for {IBMFusionSAN}

Installing and configuring {IBMFusionSAN} require the following prerequisites:

* Bare-metal worker nodes with attached SAN storage.
* A working registry enabled.
* All nodes must connect to the same LUNs.
* An IBM pull secret.
