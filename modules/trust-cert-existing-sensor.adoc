= Configuring an existing Sensor to trust custom certificates

[role="_abstract"]
You can configure an existing Sensor to trust custom certificates by creating the secret with the ca-setup-sensor.sh script.

.Prerequisites

* You have downloaded the Sensor bundle.

.Procedure

. Extract the Sensor bundle:
+
[source,terminal]
----
$ unzip -d sensor sensor-<cluster_name>.zip
----

. Run the following command to create the secret:
+
[source,terminal]
----
$ ./sensor/ca-setup-sensor.sh -d sensor/additional-cas/
----
+
where:
+
--
`-d`:: Specifies a directory containing custom certificates.
--
+
[IMPORTANT]
====
If you get the "secret already exists" error message, re-run the script with the `-u` option:

[source,terminal]
----
$ ./sensor/ca-setup-sensor.sh -d sensor/additional-cas/ -u
----
====

. Continue Sensor deployment by using the YAML files.
