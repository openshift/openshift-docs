= Migration to UBI 9 minimal base images

[role="_abstract"]

// JIRA: ROX-31576
{product-title-short} Operator base images have been migrated to UBI 9 minimal variants.

This migration provides a smaller attack surface, reduced image size, and alignment with the latest Red{nbsp}Hat Universal Base Image standards. UBI 9 minimal images include only essential packages required for {product-title-short} operations.
