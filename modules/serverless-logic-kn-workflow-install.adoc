= Installing the Knative Workflow plug-in for Knative CLI

You can use the {ServerlessLogicProductName} plug-in to set up your local workflow project using Knative CLI.

.Prerequisites
* You istalled link:https://knative.dev/docs/client/install-kn/[Knative CLI].

.Procedure 
. Download the latest binary file, suitable for your environment, from the link:https://mirror.openshift.com/pub/cgw/serverless-logic/latest/[release page].

. Install the `kn workflow` command as a plug-in of the Knative CLI using the following steps:

.. Rename the downloaded binary by using fhe following command:
+
[source,terminal]
----
mv kn-workflow-linux-amd64 kn-workflow
----

.. Make the binary file executable by using fhe following command:
+
[source, terminal]
----
chmod +x kn-workflow
----
+
[IMPORTANT]
====
On Apple Mac, some systems might block the application from running due to Apple enforcing policies. To fix this problem, check the Security & Privacy section in the System Preferences → General tab to approve the application to run. For more information, see the Apple support article link:<include-link-to-support-article>[Open a Mac app from an unidentified developer].
====

.. Copy the `kn-workflow` binary file to `/usr/local/bin`.

.. Run the following command to verify that the `kn-workflow` plug-in is installed:
+
[source, terminal]
----
kn plugin list
----
