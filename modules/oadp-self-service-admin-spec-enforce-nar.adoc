= Self-Service administrator spec enforcement for NAR

As a cluster administrator, you can enforce the following fields for a `NonAdminRestore` (NAR) CR:

* `itemOperationTimeout`
* `uploaderConfig`
* `includedResources`
* `excludedResources`
* `restoreStatus`
* `includeClusterResources`
* `labelSelector`
* `orLabelSelectors`
* `restorePVs`
* `preserveNodePorts`
