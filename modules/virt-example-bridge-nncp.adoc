= Example: Linux bridge interface node network configuration policy

[role="_abstract"]
You can create a Linux bridge interface on nodes in the cluster by applying a `NodeNetworkConfigurationPolicy` manifest
to the cluster.

The following YAML file is an example of a manifest for a Linux bridge interface.
It includes samples values that you must replace with your own information.

[source,yaml]
----
apiVersion: nmstate.io/v1
kind: NodeNetworkConfigurationPolicy
metadata:
  name: br1-eth1-policy
spec:
  nodeSelector:
    kubernetes.io/hostname: <node01>
  desiredState:
    interfaces:
      - name: br1
        description: Linux bridge with eth1 as a port
        type: linux-bridge
        state: up
        ipv4:
          dhcp: true
          enabled: true
        bridge:
          options:
            stp:
              enabled: false
          port:
            - name: eth1
----
** `metadata.name` specifies the name of the policy.
** `spec.nodeSelector` this is an optional field that specifies which nodes in the cluster this policy applies to. If you do not include the `nodeSelector` parameter, the policy applies to all nodes in the cluster. This example uses a `hostname` node selector.
** `spec.desiredState.interfaces.name` specifies the name of the interface.
** `spec.desiredState.interfaces.description` is an optional field that specifies a human-readable description for the interface.
** `spec.desiredState.interfaces.type` specifies the type of interface. This example creates an Ethernet networking interface.
** `spec.desiredState.interfaces.state` specifies the requested state for the interface after creation.
** `spec.desiredState.interfaces.ipv4` is an optional field. If you do not use `dhcp`, you can either set a static IP or leave the interface without an IP address. Setting `enabled: true` enables `ipv4` in this example.
** `spec.desiredState.interfaces.bridge.options.stp` disables `stp` in this example.
** `spec.desiredState.interfaces.bridge.port.name` specifies the node NIC to which the bridge attaches.
