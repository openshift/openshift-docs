[id="configure-google-cloud-scc_{context}"]
= Configuring Google Cloud SCC

[role="_abstract"]
Start by adding {product-title} as a trusted Cloud SCC source.

.Procedure
. Follow the "Adding vulnerability and threat sources to Cloud Security Command Center" guide and add {product-title} as a trusted Cloud SCC source.
Make a note of the *Source ID* that Google Cloud creates for your {product-title} integration.
If you do not see a source ID after registering, you can find it on the "Cloud SCC Security Sources page".
. Create a key for the service account you created, or the existing account you used, in the earlier step.
See "Creating and managing service account keys" for details.
