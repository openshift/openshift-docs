= Deleting persistent volume claims by using the web console 

[role="_abstract"]
You can delete old or unwanted persistent volume claims (PVC) by using the {product-title} web console.

If you chose to keep the original source volumes during a storage migration, follow this procedure to delete them when you no longer want to retain them.

.Procedure

. Navigate to *Virtualization* -> *Storage* -> *PersistentVolumeClaims* in the web console.

. (Optional) Perform the following tasks to filter the persistent volume claim list:

.. Use the *Filter* dropdown to filter the persistent volume claim list by *Status*, *Name*, or *Label*.
.. (Optional) If *Status* is selected, select *Pending*, *Bound*, or *Lost* from the *Filter by status* dropdown.
.. (Optional) If *Name* or *Label* is selected, enter the search criteria in the *Filter by name* or *Filter by label* textbox. The list filters as the search criteria is entered.

. Click the *Options* menu {kebab} beside the persistent volume claim.

. Click *Delete PersistentVolumeClaim*.

. Click *Delete*.
