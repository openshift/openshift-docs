= Additional prerequisites for egress zero install

[role="_abstract"]
Before you begin the egress zero install procedure, ensure you have the following:

* An AWS account with sufficient permissions to create VPCs, subnets, and other required infrastructure
* Terraform v1.4.0 or higher installed
* {rosa-cli-first} v1.2.45 or higher installed
* AWS CLI installed and configured with the necessary credentials
* git CLI installed
* The necessary {rosa-cli} and {hybrid-console} firewall rules enabled
