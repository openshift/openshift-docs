= {oadp-short} VMFR custom resources

[role="_abstract"]
Use {oadp-short} VMFR custom resources to discover VM backups and restore individual files from those backups. This provides a file-level recovery mechanism for `kubevirt` virtual machines.

The {oadp-short} VMFR feature uses the following custom resources (CRs) to perform file-level restore operations:

.Custom resources
|===
|*CR* |*Description*

|`VirtualMachineBackupsDiscovery` (VMBD)
|Identifies which Velero backups contain a specified virtual machine. Returns categorized results of valid and invalid backups.

|`VirtualMachineFileRestore` (VMFR)
|Orchestrates the workflow for making discovered backup files accessible through web or SSH access methods.
|===
