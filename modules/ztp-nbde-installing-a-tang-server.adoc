= Installing a Tang server

.Procedure

. You can install a Tang server on a {op-system-base-full} machine using either of the following commands:
+
[source,terminal]
----
$ yum install tang
----
+
[source,terminal]
----
$ dnf install tang
----
+
[NOTE]
====
Installation can also be containerized and is very lightweight.
====
