= Deleting a buildstrategy resource

You can delete a `BuildStrategy` resource if it is not required in your project.

.Prerequisites

* You have installed the `oc` CLI.

.Procedure

* Delete a `BuildStrategy` resource by running the following command:
+
.Using `oc` CLI
[source,terminal]
----
$ oc delete buildstrategies <buildstartegy_resource_name>
----
