= Enforcement on existing deployments

[role="_abstract"]
For existing deployments, policy changes only result in enforcement when a Kubernetes event occurs or when you manually reassess all policies.

If you make changes to a policy, you must reassess policies by selecting *Policy Management* and clicking *Reassess All*. This action applies deploy policies on all existing deployments regardless of whether there are any new incoming Kubernetes events. If a deployment violates a policy, then {product-title-short} performs enforcement.
