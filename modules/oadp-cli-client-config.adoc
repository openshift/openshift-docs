= Configuring the OADP CLI client

You can use the {oadp-short} CLI to view and modify client configuration settings. Configuration is stored in `~/.config/velero/config.json`.

.Prerequisites

* The {oadp-short} CLI plugin is installed.

.Viewing the current configuration

* View the current client configuration:
+
[source,terminal]
----
$ oc oadp client config get
----

.Setting a configuration value

* Set a configuration value:
+
[source,terminal]
----
$ oc oadp client config set <key>=<value>
----
+
.Example: setting the default namespace
[source,terminal]
----
$ oc oadp client config set namespace=openshift-adp
----
