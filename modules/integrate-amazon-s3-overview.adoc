= Amazon S3 backup integration overview

[role="_abstract"]
The backup includes the entire {product-title} database, which includes all configurations, resources, events, and certificates. Make sure to store backups securely.

[IMPORTANT]
====
* If you are using {product-title} version 3.0.53 or older, the backup does not include certificates.
* If your Amazon S3 is part of an air-gapped environment, you must add your AWS root CA as a trusted certificate authority in {product-title}.
====
