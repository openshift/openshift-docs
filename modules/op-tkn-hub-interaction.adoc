= Hub interaction commands

Interact with tekton hub.

== hub
Manage EventListeners.

.Example: Display help
[source,terminal]
----
$ tkn hub -h
----

.Example: Interact with API server
[source,terminal]
----
$ tkn hub --api-server https://api.hub.tekton.dev
----
