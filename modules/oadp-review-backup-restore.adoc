= Preview resources before running backup and restore

{oadp-short} backs up application resources by type, namespace, or label at runtime. This means, you are able to view the resources after the backup is complete. Similarly, you are able to view the restored objects by namespace, PV, or label after a restore operation is complete. You can do a dry run of the backup and restore operations, to preview the resources in advance.

To preview the resources that will be backed up before running the actual backup, run the following `velero` command:

[source, terminal]
----
velero backup create <backup-name> --snapshot-volumes false # <1>
----
<1> Specify `--snapshot-volumes false`.

To preview the resources that will be restored before running the actual restore, run the following `velero` command:

[source, terminal]
----
velero restore create --from-backup <backup-name> # <1>
----
<1> Specify the name of the backup created to review the backup resources.

[IMPORTANT]
====
The `velero restore create` command creates restore resources in the cluster. Make sure to delete the resources created by the restore, after reviewing the resources.
====
