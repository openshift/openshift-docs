= Disabling pipeline templates

By default, the `TektonAddon` custom resource (CR) installs pipeline templates with {pipelines-shortname} on the cluster.

You can disable the installation of pipeline templates by setting the parameter value to `false` in the `TektonConfig` custom resource (CR).

.Procedure

. Edit the `TektonConfig` CR by running the following command:
+
[source,terminal]
----
$ oc edit TektonConfig config
----

. In the `TektonConfig` CR, set the `pipelineTemplates` parameter value in `.addon.params` spec to `false`:
+
.Example of disabling pipeline template resource
[source,yaml]
----
apiVersion: operator.tekton.dev/v1alpha1
kind: TektonConfig
metadata:
  name: config
spec:
# ...
  addon:
    params:
      - name: pipelineTemplates
        value: 'false'
# ...
----
