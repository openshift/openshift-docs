= Business user workflow

[role="_abstract"]
Business users, such as account managers and application owners, can use the data from {prodname} to work with customers.

You can use Grafana-based observability dashboards to monitor the status of applications and APIs in data centers in specific regions, and work with customers on specific performance metrics.

Specifcally, you can view API metrics, such as uptime, requests per second, latency, and errors per minute, to ensure that APIs meet the performance and availability benchmarks that your customers require.
