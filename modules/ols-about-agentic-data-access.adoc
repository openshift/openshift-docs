= What data the agent accesses

[role="_abstract"]
The agent accesses cluster data through the Core Model Context Protocol (MCP) server to perform its investigations.

The Core MCP server provides the agent access to the following types of cluster data:

* Cluster telemetry
* Kubernetes resource metadata
* Prometheus alert data
* Infrastructure conditions

This data is transmitted to your configured LLM provider as part of the agent's investigation and diagnosis workflow.

[IMPORTANT]
====
Ensure that your LLM provider is configured consistently with your organization's data handling requirements. Cluster state data, including resource metadata and telemetry, is sent to the LLM provider during agent operations.
====
