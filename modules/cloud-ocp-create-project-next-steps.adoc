= Next steps

[role="_abstract"]
After creating the project on your secured cluster, generate the secrets required for secure communication between the secured cluster and Central.

In the ACS Console, generate a cluster registration secret or an init bundle. These files contain the secrets that set up the initial secured communication between {product-title-managed-short} secured clusters and Central.
