== Deleting the ArgoCD application

Delete the ArgoCD application in the target cluster that you want to migrate to prevent unexpected syncs or app triggers during the migration.

.Prerequisites

* You have logged in to the hub cluster as a user with `cluster-admin` privileges.
* You have configured your {ztp} environment successfully.
* You have installed and configured the Assisted Installer service successfully.

.Procedure

. Ensure that the target `ManagedCluster` resource is deployed and available in the hub cluster by running the following command:
+
[source,bash]
----
$ oc get ManagedCluster <target_cluster_name>
----
+
* `<target_cluster_name>` is the name of the target cluster.

. Delete the ArgoCD AppProject and Application resources that are managing the `SiteConfig` CRs in the target cluster by running the following command:
+
[source,bash]
----
$ oc -n openshift-gitops get appprojects.argoproj.io
----
+
[source,bash]
----
$ oc -n openshift-gitops delete appprojects.argoproj.io <app_project_name>
----
+
* `<app_project_name>` is the name of the ArgoCD app project for the target cluster.

. Delete the ArgoCD Application resource for the target cluster:
+
[WARNING]
====
To avoid accidentally deleting an entire deployed environment for a project, apply a non-cascade deletion only. Ensure a non-cascade deletion by ensuring the `finalizers` specification is unset, and then delete the app. 
====

.. View the ArgoCD applications by running the following command:
+
[source,bash]
----
$ oc -n openshift-gitops get applications.argoproj.io
----
+
.Example output
[source,bash]
----
NAME        SYNC STATUS     HEALTH STATUS
clusters    Synced          Healthy
policies    Synced          Healthy
----

.. Verify that the `finalizers` specification is unset by running the following command:
+
[source,bash]
----
$ oc -n openshift-gitops get applications.argoproj.io clusters -ojson | jq .metadata.finalizers
----
+
.Example output
[source,bash]
----
null
----

.. Delete the ArgoCD application for the target cluster by running the following command:
+
[source,bash]
----
$ oc -n openshift-gitops delete applications.argoproj.io <application_name>
----
+
* `<application_name>` is the name of the ArgoCD application for the target cluster.
