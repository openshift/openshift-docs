= Changing the update channel for an Operator

[role="_abstract"]
You can change the update channel for an Operator by using the {product-title} web console.

[TIP]
====
If the approval strategy in the subscription is set to *Automatic*, the update process initiates as soon as a new Operator version is available in the selected channel. If the approval strategy is set to *Manual*, you must manually approve pending updates.
====

.Prerequisites

* An Operator previously installed using Operator Lifecycle Manager (OLM).

.Procedure

. In web console, navigate to *Ecosystem* -> *Installed Operators*.

. Click the name of the Operator you want to change the update channel for.

. Click the *Subscription* tab.

. Click the name of the update channel under *Update channel*.

. Click the newer update channel that you want to change to, then click *Save*.

. Monitor or approve the update depending on your subscription's approval strategy:
+
--
* **Automatic strategy:** The update begins automatically. Navigate back to the *Ecosystem* -> *Installed Operators* page to monitor the progress. When complete, the status changes to *Succeeded* and *Up to date*.
* **Manual strategy:** Navigate to the *Subscription* tab to manually review and approve the pending update.
--
