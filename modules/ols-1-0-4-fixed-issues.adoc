= Fixed issues

{ols-official} 1.0.4 fixes the following issues:

* Before this update, the {ols-long} Service treated the `api_version` parameter as an additional input for {azure-openai} providers. As a consequence, the Service generated a Pydantic validation error when specifying `api_version` for an Azure OpenAI provider, causing a failure in the {ols-long} Service. With this release, the Service now accepts the `api_version` parameter for {azure-openai} providers. https://issues.redhat.com/browse/OLS-1973[OLS-1973]
