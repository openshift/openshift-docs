= Behavior of resource configuration

The following information describes how the `GitOpsService` controller applies resource values defined in the `GitOpsService` custom resource (CR).

* *Specified values:*
The `GitOpsService` controller applies the defined resource values to the corresponding components. For example, if you define resources for the backend only, the configuration applies to the backend deployment but not to the plugin.

* *Default values:*  
If you do not specify resource values in the `GitOpsService` CR, the `GitOpsService` controller applies default fixed values to the deployment.
