= RHSA-2026:8407 - {product-title} {product-version}.64 image release, fixed issues, and security update

Issued: 23 April 2026

[role="_abstract"]
{product-title} release {product-version}.64 is now available. The list of bug fixes that are included in the update is documented in the link:https://access.redhat.com/errata/RHSA-2026:8407[RHSA-2026:8407] advisory. The RPM packages that are included in the update are provided by the link:https://access.redhat.com/errata/RHBA-2026:8405[RHBA-2026:8405] advisory.

Space precluded documenting all of the container images for this release in the advisory.

You can view the container images in this release by running the following command:

[source,terminal]
----
$ oc adm release info 4.14.64 --pullspecs
----

== Fixed issues

* Before this update, the `osDiskImage.source.id` property in the user-provisioned infrastructure ARM template was incorrectly placed. As a consequence, installation failures occurred. With this release, the `storageProfile.osDiskImage.source.id` property location in the user-provisioned infratructure ARM template was corrected. As a result, the user-provisioned infrastructure ARM template installation on {azure-first} no longer encounters invalid parameter errors. (link:https://redhat.atlassian.net/browse/OCPBUGS-79654[OCPBUGS-79654])
