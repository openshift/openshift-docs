= Migrate Grafana dashboards to a Perses by using the CLI

[role="_abstract"]
You can use the `percli` CLI to convert a Grafana dashboard JSON file to Perses format and apply it as a `PersesDashboard` custom resource.

.Prerequisites

* You have access to Grafana dashboard JSON export files.
* You have edit permissions in the target namespace.
* You have created the necessary datasources in Perses.
* You have the `percli` tool installed on your local machine.

.Procedure

. Convert the Grafana dashboard to Perses format:
+
[source,terminal]
----
$ percli convert grafana grafana-dashboard.json -o perses-dashboard.json
----

. Review the converted output and update datasource references as required.

. Apply the converted dashboard:
+
[source,terminal]
----
$ oc apply -f perses-dashboard.yaml
----

.Verification

* Verify that the dashboard was created:
+
[source,terminal]
----
$ oc get persesdashboards -n <target_namespace>
----
