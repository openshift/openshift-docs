= Configuring the DPU Operator 

* Create a `SriovOperatorConfig` custom resource (CR) to deploy the DPU host components:

.. Create a file named `dpu-operator-host-config.yaml` using the following YAML:
+
[source,yaml]
----
apiVersion: config.openshift.io/v1
kind: DpuOperatorConfig
metadata:
 name: dpu-operator-config
spec:
 mode: host
----

.. Create the resource by running the following command:
+
[source,terminal]
----
$ oc apply -f dpu-operator-host-config.yaml
----
