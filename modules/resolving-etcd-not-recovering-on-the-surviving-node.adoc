= Resolving etcd not recovering on the surviving node

[role="_abstract"]
If etcd is not running on the surviving node after fencing succeeds, reset the resource state.

.Procedure

. Clean up the etcd resource and verify Pacemaker status:
+
[source,terminal]
----
$ oc debug node/<surviving-node> -- chroot /host bash -c '
  pcs resource cleanup etcd
  pcs status
'
----
