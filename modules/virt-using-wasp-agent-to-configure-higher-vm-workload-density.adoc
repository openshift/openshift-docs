= Using wasp-agent to configure higher VM workload density

wasp-agent is a component to enable an OpenShift cluster to assign SWAP resources to burstable VM workloads only. See https://kubernetes.io/docs/tasks/configure-pod-container/quality-service-pod/
SWAP usage is supported on worker nodes only.

.Prerequisites

* The `oc` tool is available.
* Logged into cluster with cluster-admin role
* A defined memory over-commit ratio. By default: 150%
* A worker pool
