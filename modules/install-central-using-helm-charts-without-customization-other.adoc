= Install Central by using Helm charts without customization

[role="_abstract"]
You can install Red{nbsp}Hat Advanced Cluster Security for Kubernetes (RHACS) on your cluster without any customizations. You must add the Helm chart repository and install the `central-services` Helm chart to install the centralized components of Central and Scanner.
