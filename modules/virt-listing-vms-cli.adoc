= Listing virtual machines by using the CLI

You can list all virtual machines (VMs) in your cluster or limit the list to VMs in a certain namespace by using the {oc-first}.

.Prerequisites

* You have installed the {oc-first}.

.Procedure

* To list all VMs in your cluster, run the following command:
+
[source,terminal]
----
$ oc get vms -A
----

* To list VMs in a specific namespace, run the following command:
+
[source,terminal]
----
$ oc get vms -n <namespace>
----
