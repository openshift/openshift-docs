= Log out of the {rosa-cli}

[role="_abstract"]
Log out of `rosa`. Logging out also removes the `rosa` configuration file.

.Procedure
* Log out of your `rosa` session by running the following command:
+
[source,terminal]
----
$ rosa logout [arguments]
----
+
