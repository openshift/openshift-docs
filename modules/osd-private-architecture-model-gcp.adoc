= Private {product-title} on {GCP} without Private Service Connect architecture model

Private OSD clusters use some public subnets, but no control plane or worker nodes are deployed in public subnets. Red Hat SRE management access the cluster through a public load balancer endpoint that are restricted to Red Hat IPs. API server endpoint is private. A separate Red Hat API server endpoint is public (but restricted to Red Hat trusted IP addresses). The default ingress controller can be public or private.
