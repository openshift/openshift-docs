== Scale the NodePool

After your agents are approved, you can scale the node pools, `agentLabelSelector` configured in the node pool ensures that only matching agents added to the cluster. This also aids in descaling the node pool. To remove specific architecture nodes from the cluster, descale the corresponding node pool.
