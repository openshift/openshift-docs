= Run Terraform script via the Home region

All the Identity related resources, including Dynamic Groups, Policies and Tags, are created on the OCI Home 
Region (ie, PHX, IAD, FRA, etc). Other resources can be created on Compute Cloud@Customer (C3).  

Run two Terraform scripts on the bastion server created in previous steps. The first script creates the 
Dynamic Group Identity resources on your C3 OCI Home tenancy. The second script creates infrastructure resources on C3. 

For details, see link:https://app.slack.com/client/E030G10V24F/D05CCFA8UVB[Terraform Script Execution - Part 1 (Oracle documentation)].
