= Prerequisites

[role="_abstract"]
You must configure environment variables before using the `roxctl` CLI.

.Procedure

* To configure the `ROX_ENDPOINT` environment variable, run the following command:
+
[source,terminal,subs="+quotes"]
----
$ export ROX_ENDPOINT=_<host:port>_
----
+
where:
+
--
`<host:port>`:: Specifies the host and port information that you want to store in the  `ROX_ENDPOINT` environment variable.
--
