= Prerequisites

* You have configured the `ROX_ENDPOINT` environment variable using the following command:
+
[source,terminal,subs="+quotes"]
----
$ export ROX_ENDPOINT=_<host:port>_
----
+
where:
+
--
`<host:port>`:: Specifies the host and port information that you want to store in the  `ROX_ENDPOINT` environment variable.
--
