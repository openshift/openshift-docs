= Checking synchronization state on Carter Flats devices

[role="_abstract"]
Check the available high-level synchronization state for Carter Flats devices by reviewing exposed PTP status indicators and validating downstream synchronization.

.Prerequisites

Before you begin, ensure that the following conditions are met:

* You have configured a GNR-D boundary clock.
* You have access to cluster metrics.

.Procedure

. Check the available PTP status indicators exposed by the PTP Operator.
+
Review the reported synchronization state and clock class information for the configured node.

. Confirm that the reported synchronization state matches the expected upstream PTP Grand Master connection.
+
Verify that the node maintains the expected synchronization state.

. Validate downstream synchronization by connecting a downstream device to a configured TimeTransmitter interface.
+
Confirm that downstream synchronization is maintained.
