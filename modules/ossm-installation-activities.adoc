= {ProductName} Operators

Installing {ProductName} requires four Operators, OpenShift Elasticsearch, Jaeger, Kiali and {ProductName}. 


* *Elasticsearch* - Provides database storage for tracing and logging with Jaeger. It is based on the open source link:https://www.elastic.co/[Elasticsearch] project 
* *Jaeger* - Provides tracing to monitor and troubleshoot transactions in complex distributed systems. It is based on the open source link:https://www.jaegertracing.io/[Jaeger] project. 
* *Kiali* - Provides observability for your service mesh. By using Kiali you can view configurations, monitor traffic, and view and analyze traces in a single console. It is based on the open source link:https://www.kiali.io/[Kiali] project, provides observability for your service mesh. 
* *{ProductName}* - Helps you connect, secure, control, and observe the microservices that make up your applications. It is based on the open source link:https://istio.io/[Istio] project. The {ProductShortName} Operator defines and monitors the `ServiceMeshControlPlane` resources that manage the deployment, updating, and deletion of the {ProductShortName} components.
