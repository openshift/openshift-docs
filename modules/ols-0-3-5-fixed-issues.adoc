= Fixed issues

The following issues are fixed with {ols-official} 0.3.5:

* Previously, the {ols-long} service could not support locally hosted OpenAI-compatible API endpoints as application service providers. Now the {ols-short} service can support OpenAI-compatible API endpoints. To securely connect with these providers, the {ols-short} service supports the configuration and trust of additional Certificate Authority (CA) certificates. link:https://issues.redhat.com/browse/OLS-1715[OLS-1715]
