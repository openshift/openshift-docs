= Checklist for migrating {SMProduct} 2 gateways

Since the {SMProduct} 3 control plane does not manage gateways, you must manage them yourself.

To manage them yourself and move from {SMProduct} 2 to {SMProduct} 3, you must follow the procedure for "Service Mesh gateway injection" to:

* Disable the default gateways.
* Move away from the `ServiceMeshControlPlane` resource declaration.
* Use gateway injection.
