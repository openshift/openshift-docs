= Vulnerabilities

[role="_abstract"]
{product-title-short} fetches vulnerability definitions and updates from multiple vulnerability feeds. These feeds are both general in nature, such as NVD, or distribution-specific, such as Alpine, Debian, and Ubuntu. 

For more information on viewing and addressing vulnerabilities that are found, see xref:../operating/manage-vulnerabilities/vulnerability-management.adoc#vulnerability-management[Vulnerability management].
