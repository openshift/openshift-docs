= Creating a {product-title} cluster with STS

[role="_abstract"]
Choose from one of the following methods to deploy a {product-title} cluster that uses the AWS Security Token Service (STS). In each scenario, you can deploy your cluster by using {cluster-manager-first} or the {rosa-cli} (`rosa`):

* *Creating a {product-title} cluster with STS using the default options*: You can create a {product-title} cluster with STS quickly by using the default options and automatic STS resource creation.

* *Creating a {product-title} cluster with STS using customizations*: You can create a {product-title} cluster with STS using customizations. You can also choose between the `auto` and `manual` modes when creating the required STS resources.
