Question: Are the following three steps necessary if you are setting up providers from the web console?
====

. Enter the following command to log in as an administrator:
+
[source,terminal]
----
$ oc login -u kubeadmin -p cluster_password
----

. Enter the following command to obtain the cluster's IP address:
+
[source,terminal]
----
$ oc cluster-info
----

. Enter the following command to obtain a token:
+
[source,terminal]
----
$ oc whoami -t
----
