= Edit a Perses Dashboard by using the web console

[role="_abstract"]
You can edit an existing Perses dashboard in the {ocp-product-title} web console.

.Prerequisites

* You are logged in to the {ocp-product-title} web console.
* You have edit permissions in the namespace where the dashboard resides.

.Procedure

. In the web console, navigate to the *Observe* > *Dashboards* page.

. Select the dashboard that you must modify from the list.

. Click *Edit* to enable edit mode.

. Drag panels to rearrange their positions on the grid.

. Click the *Options* icon on a panel to change its individual configuration settings.

. Click *Add Group* to create and adjust new panel groups.

. Click *Save* to save your changes.

.Verification

* Confirm that the dashboard displays the updated layout and configurations.
