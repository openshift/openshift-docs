= Installing the {gcp-short} Filestore CSI Driver Operator

[role="_abstract"]
Since the Google Compute Platform ({gcp-short}) Filestore Container Storage Interface (CSI) Driver Operator is not installed in {product-title} by default, you must install the {gcp-short} Filestore CSI Driver Operator in your cluster.
