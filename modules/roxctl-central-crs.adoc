= roxctl central crs

Manage cluster registration secrets that allow communication between Central and secured clusters for the initial setup.

:FeatureName: Cluster registration secrets
include::snippets/technology-preview.adoc[]

.Usage
[source,terminal]
----
$ roxctl central crs [flags]
----

.Options
[cols="2,2",options="header"]
|===
|Option |Description

|`--generate`
|Generate a cluster registration secret.

|`--list`
|List all cluster registration secrets.

|`--revoke`
|Revoke a cluster registration secret.

|===
