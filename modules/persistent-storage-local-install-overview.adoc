= Installing the Local Storage Operator

[role="_abstract"]
To install the Local Storage Operator (LSO) to provision and manage local persistent storage volumes in your cluster:

. Create the `openshift-local-storage` project.

. Install and configure the LSO by using either the CLI or {product-title} web console.
