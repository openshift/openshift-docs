= {microshift-short} OpenShift APIs

[role="_abstract"]
Use the supported OpenShift APIs in {microshift-short} to configure networking routes and security context constraints for your workloads.

In addition to standard Kubernetes APIs, {microshift-short} includes a small subset of the APIs supported by {OCP}.
