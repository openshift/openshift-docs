= Create a dashboard by using the web console

[role="_abstract"]
You can create Perses dashboards by using the graphical editor in the {ocp-product-title} web console.

.Prerequisites

* You have enabled the monitoring plugin.
* You have at least one datasource that exists in your target namespace, or a global datasource is available.
* You have the edit permissions for Perses Dashboards in the target namespace.

.Procedure

. In the {ocp-product-title} web console, navigate to *Observe* -> *Dashboards (Perses)*.

. Click *Create*.

. In the dashboard metadata fields, provide the following information:
+
* *Name*: A unique name for the dashboard.
* *Display Name*: An optional, user-friendly display name.
* *Project*: Select the target namespace from the dropdown.

. Add one or more panels to the dashboard:
+
.. Click *Add Panel*.
.. Select from various panel types, including *Time series*, *Stat*, *Gauge*, *Bar chart*, *Table*, among others.
.. Configure the panel:
+
* *Title*: A descriptive title for the panel.
* *Datasource*: Select a datasource from the available list.
* *Query*: Enter a query in the appropriate query language (PromQL, LogQL, or TraceQL).
* *Visualization options*: Adjust display settings as required.

. Arrange panels by using drag-and-drop and resize handles.

. Optional: Configure dashboard variables:
+
.. Click *Dashboard settings* -> *Variables*.
.. Click *Add variable*.
.. Enter the variable name, type, and its type-specific query or value.

. Save the dashboard:
+
.. Click *Save*.

.Verification

* The dashboard appears in the Perses dashboards list.
* Open the dashboard and verify that all panels display data correctly.
* If you configured variables, confirm that they function as expected.
