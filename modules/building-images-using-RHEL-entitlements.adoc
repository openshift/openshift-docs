= Building images using the RHEL entitlements

If you have Red Hat Enterprise Linux (RHEL) entitlement, the Insights operator automatically manages your entitlement using the Simple Content Access (SCA) feature. SCA allows RHEL systems to consume subscription content without manually managing the entitlement keys. The Insights operator imports your SCA entitlements and stores them in a secret named `etc-pki-entitlement` in the `openshift-config-managed` namespace. 

Traditionally, the cluster administrator would manually copy the `etc-pki-entitlement` secret from openshift-config-managed to every namespace wherever required. In {ocp-product-title} 4.10 and later, {builds-title} can share the `etc-pki-entitlement` secret from the `openshift-config-managed` namespace to other namespaces using the Shared Resources Container Storage Interface (CSI) Driver Operator.
