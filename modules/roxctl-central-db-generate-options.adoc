[id="roxctl-central-db-generate-options_{context}"]
= Options

[role="_abstract"]
Options for the `roxctl central db generate` command.

.Options
[cols="2,2",options="header"]
|===
|Option |Description

|`--debug`
|If set to `true`, the command reads templates from the local file system. The default value is `false`.

|`--debug-path string`
|Specify the path to the Helm templates in your local file system. For more details, run the `roxctl central db generate` command.

|`--enable-pod-security-policies`
|If set to `true`, the command creates `PodSecurityPolicy` resources. The default value is `true`.
|===
