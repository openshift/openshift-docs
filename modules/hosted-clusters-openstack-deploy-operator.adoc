= Deploying the HyperShift Operator

.Procedure

. Install the HyperShift Operator into the management cluster:
+
[source,terminal]
----
hypershift install --tech-preview-no-upgrade
----
+
[NOTE]
====
HyperShift on OpenStack is possible behind a feature gate, which is why you must install the operator with `--tech-preview-no-upgrade`. After the platform is GA, you can install the operator without that flag.
====

. Verify that the operator is running in the `hypershift` namespace:
+
[source,terminal]
----
oc -n hypershift get pods
----
+
.Example output
[source,terminal]
----
NAME                        READY   STATUS    RESTARTS   AGE
operator-755d587f44-lrtrq   1/1     Running   0          114s
operator-755d587f44-qj6pz   1/1     Running   0          114s
----
