= Install the OADP CLI plugin

[role="_abstract"]
The {oadp-short} command-line interface (CLI) plugin is available from the *Command-line tools* page in the {product-title} web console when the {oadp-short} Operator is installed.

== Prerequisites

* You have access to an {product-title} cluster with the {oadp-short} Operator installed.

== Procedure

. Log in to the {product-title} web console as a user with access to the cluster.
. Click the *?* icon in the toolbar and select *Command-line tools*.
. Download the `oc-oadp` binary for your operating system and architecture.
. Extract the archive and place the `oc-oadp` binary in a directory on your `PATH`.
. Verify the installation:
+
[source,terminal]
----
$ oc oadp version
----
