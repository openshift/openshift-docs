= Required GCP permissions for installer-provisioned infrastructure

When you attach the `Owner` role to the service account that you create, you grant that service account all permissions, including those that are required to install {product-title}.

If your organizationâ€™s security policies require a more restrictive set of permissions, you can create link:https://cloud.google.com/iam/docs/creating-custom-roles[custom roles] with the necessary permissions. The following permissions are required for the installer-provisioned infrastructure for creating and deleting the {product-title} cluster.
