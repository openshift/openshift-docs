= Memory and CPU requirements

[role="_abstract"]
The following table lists the minimum memory and CPU values required to run the admission controller on a secured cluster.

|===
| Deployments | CPU | Memory

| < 25,000
| 0.5 cores
| 300 MiB

| < 50,000
| 0.5 cores
| 600 MiB
|===
