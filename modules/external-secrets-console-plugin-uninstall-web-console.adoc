= Uninstalling the {external-secrets-console-plugin}

[role="_abstract"]
You can uninstall the {external-secrets-console-plugin} from your cluster using the {product-title} web console.

.Prerequisites

* You have access to the {product-title} web console.
* The {external-secrets-console-plugin} is installed.

.Procedure

. Log in to the {product-title} web console.

. Navigate to *Ecosystem* -> *Installed Operators*.

. Click the Options menu {kebab} next to the *External Secrets Management Console* entry, and then click *Uninstall Operator*.

. In the confirmation dialog, select the *Delete all operand instances for this operator* checkbox and then click *Uninstall*.

.Verification

* Verify that *Secrets Management* no longer appears under *Plugins*.
