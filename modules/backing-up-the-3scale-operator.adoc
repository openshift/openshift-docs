= Backing up the 3scale Operator

Back up the operator resources, and secret and APIManager custom resources. For more information, see xref:oadp-creating-backup-cr.adoc[Creating a Backup CR].

.Prerequisites

* You created the Data Protection Application (DPA). 

.Procedure

. Back up the operator related resources such as `operatorgroup`, `namespaces`, and `subscriptions` by creating a YAMl file: 
+
[source,terminal]
----
$ oc create -f <backup>.yaml
----
+
[source,yaml]
----
apiVersion: velero.io/v1
kind: Backup
metadata:
  name: operator-install-backup
  namespace: openshift-adp
spec:
  csiSnapshotTimeout: 10m0s
  defaultVolumesToFsBackup: false
  includedNamespaces:
  - threescale   <1>
  includedResources:
  - operatorgroups
  - subscriptions
  - namespaces
  itemOperationTimeout: 1h0m0s
  snapshotMoveData: false
  ttl: 720h0m0s
----
<1> Namespace where the 3scale Operator is installed.
+
NOTE: You can also back up and restore `ReplicationControllers`, `DeploymentConfigs`, and `Pod` objects to ensure that all manually set environments are backed up and restored. This does not affect the flow of restoration.

. Back up the `secret` custom resources (CRs):
+
[source,terminal]
----
$ oc create -f <backup>.yaml
----
+
[source,yaml]
----
apiVersion: velero.io/v1
kind: Backup
metadata:
  name: operator-resources-secrets
  namespace: openshift-adp
spec:
  csiSnapshotTimeout: 10m0s
  defaultVolumesToFsBackup: false
  includedNamespaces:
  - threescale
  includedResources:
  - secrets
  itemOperationTimeout: 1h0m0s
  labelSelector:
    matchLabels:
      app: 3scale-api-management
  snapshotMoveData: false
  snapshotVolumes: false
  ttl: 720h0m0s
----

. Backup the `apimanagers` CRs:
+
[source,terminal]
----
$ oc create -f <backup>.yaml
----
+
[source,yaml]
----
apiVersion: velero.io/v1
kind: Backup
metadata:
  annotations:
    velero.io/resource-timeout: 10m0s
    velero.io/source-cluster-k8s-gitversion: v1.29.8+f10c92d
    velero.io/source-cluster-k8s-major-version: "1"
    velero.io/source-cluster-k8s-minor-version: "29"
  creationTimestamp: "2024-09-24T05:23:57Z"
  generation: 6
  labels:
    velero.io/storage-location: ts-dpa-1
  name: operator-resources-apim
  namespace: openshift-adp
  resourceVersion: "76771"
  uid: bc2f4b76-dedb-4532-8698-93df7477f508
spec:
  csiSnapshotTimeout: 10m0s
  defaultVolumesToFsBackup: false
  includedNamespaces:
  - threescale
  includedResources:
  - apimanagers
  itemOperationTimeout: 1h0m0s
  snapshotMoveData: false
  snapshotVolumes: false
  storageLocation: ts-dpa-1
  ttl: 720h0m0s
  volumeSnapshotLocations:
  - ts-dpa-1
status:
  completionTimestamp: "2024-09-24T05:23:58Z"
  expiration: "2024-10-24T05:23:57Z"
  formatVersion: 1.1.0
  hookStatus: {}
  phase: Completed
  progress:
    itemsBackedUp: 1
    totalItems: 1
  startTimestamp: "2024-09-24T05:23:57Z"
  version: 1
----
