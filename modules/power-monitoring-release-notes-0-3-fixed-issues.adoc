= {PM-shortname-c} 0.3 (Technology Preview) fixed issues

[role="_abstract"]
The {PM-shortname-c} 0.3 (Technology Preview) release addresses an issue related to Prometheus rules in the {PM-operator} dashboard.

Updated Prometheus rules:: Before this update, the {PM-operator} dashboard used an invalid Prometheus rule, which caused the panel for `OTHER Power Consumption(W) by Pods` to display incorrect data. With this update, the rule is corrected, ensuring the dashboard now shows accurate power consumption data.
