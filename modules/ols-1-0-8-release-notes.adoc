= {ols-long} version 1.0.8

[role="_abstract"]
{ols-official} 1.0.8 provides maintenance updates, security fixes, and stability improvements. This release ensures continued support and optimized performance for clusters running {ocp-product-title} 4.16 and later.

[id="ols-1-0-8-enhancements_{context}"]
== Enhancements

[role="_abstract"]
{ols-official} 1.0.8 introduces deeper integration with Red Hat Advanced Cluster Management, allowing users to attach YAML configurations for nodes, add-ons, and Application Sets directly to AI prompts. These enhancements provide the virtual assistant with precise cluster context for more accurate troubleshooting and analysis.

The following enhancements have been made with {ols-official} 1.0.8:

* With this update, it is now possible to attach the cluster details as YAML code to the {ols-long} prompt from the *Nodes* and *Add-ons* tabs of the Advanced Cluster Management UI.

* With this update, it is now possible to attach `ApplicationSet` details as YAML to the {ols-long} prompt from the *Application details* page of the Advanced Cluster Management UI.
