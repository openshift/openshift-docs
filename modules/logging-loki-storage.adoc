= Loki object storage

[role="_abstract"]
The {loki-op} supports AWS S3 and other S3-compatible object stores such as S3 Compatible storage (NetApp StorageGRID and others) and {rh-storage}. Azure, Google Cloud Storage (GCS), and Swift are also supported.

The recommended nomenclature for Loki storage is `logging-loki-_<your_storage_provider>_`.

The following table shows the `type` values within the `LokiStack` custom resource (CR) for each storage provider.

[options="header"]
.Secret type quick reference
|===
| Storage provider          | Secret `type` value
| AWS                       | s3
| Azure                     | azure
| Google Cloud              | gcs
| S3 Compatible storage     | s3
| {rh-storage}              | s3
| Swift                     | swift
|===
