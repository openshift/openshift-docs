= Options

[role="_abstract"]
Options for the `roxctl central userpki delete` command.

.Options
[cols="2,2",options="header"]
|===
|Option |Description

|`-f`, `--force`
|If set to `true`, proceed with the deletion without confirmation. The default value is `false`.

|`--retry-timeout duration`
|Specify the timeout after which the command retries API requests. A value of zero means that the command waits for the entire request duration without retrying. The default value is `20s`.

|`-t`, `--timeout duration`
|Specify the timeout for API requests representing the maximum duration of a request. The default value is `1m0s`.
|===
