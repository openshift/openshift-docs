= Viewing vulnerabilities in virtual machines

[role="_abstract"]

In the *Vulnerability Management* -> *Results* page, you can get information about the vulnerabilities in virtual machines (VMs).

.Procedure

. In the {product-title-short} portal, go to *Vulnerability Management* -> *Results*.
. Select the *Virtual Machines* tab. The table displays all VMs and the vulnerabilities found. The *Scanned packages* column indicates if `roxagent` is running, and if the packages have been indexed correctly.
