= Retrieving debugging information

[role="_abstract"]
Gather debugging information to investigate issues with {product-title-short}.

.Procedure

* Run the following command to gather the debugging information for investigating issues:
+
[source,terminal,subs="+quotes"]
----
$ roxctl central debug dump
----
* To generate a diagnostic bundle with the {product-title-short} administrator password or API token and central address, use the `roxctl` CLI.
