= Simple Content Access 

In {ocp-product-title} 4.10 and later, {builds-title} can use Red Hat subscriptions (RHEL entitlements) by referencing shared resources and the Simple Content Access(SCA) feature provided by the Insights operator. 

SCA allows Red Hat systems to automate entitlement management:

* The SCA imports your subscription credentials to a well-known `Secret` object.
* The Insights operator periodically imports the SCA certificate from OpenShift cluster manager.
* The Insights operator creates or updates the `etc-pki-entitlement` secret in the `openshift-config-managed` namespace.
