= View Perses dashboards

[role="_abstract"]
You can view Perses dashboards in the {product-title} web console from the *Observe* > *Dashboards (Perses)* page.

.Prerequisites

* You have view permissions in the namespace that contains the dashboards.
* You have at least one `PersesDashboard` resource exists in the namespace.

.Procedure

. In the {product-title} web console, navigate to *Observe* -> *Dashboards (Perses)*.

. Select a namespace from the project selector.
+
The dashboards available to you in that namespace are listed.

. Click a dashboard name to open it.

. Optional: Adjust the time range by using the time range picker in the toolbar.

. Optional: Interact with dashboard variables by selecting values from the variable drop-down lists at the top of the dashboard.

. Optional: To view multiple namespaces, select additional namespaces from the project selector.
+
[NOTE]
====
You can only view dashboards in namespaces where you have been granted at least the `persesdashboard-viewer-role` ClusterRole through a `RoleBinding`.
====
