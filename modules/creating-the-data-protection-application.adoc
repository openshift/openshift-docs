= Creating the Data Protection Application

Create the Data Protection Application (DPA) for 3scale. For more information, see xref:oadp-installing-dpa-1-3.adoc[Installing the Data Protection Application].

.Procedure

* Create an example DPA:
+
[source,terminal]
----
$ oc create -f <dpa>.yaml
----
+
[source,yaml]
----
apiVersion: oadp.openshift.io/v1alpha1
kind: DataProtectionApplication
metadata:
  name: <dpa_sample>
  namespace: openshift-adp 
spec:
  configuration:
    velero:
      defaultPlugins:
        - openshift 
        - aws
        - csi
      resourceTimeout: 10m 
    nodeAgent: 
      enable: true 
      uploaderType: kopia 
  backupLocations:
    - name: default
      velero:
        provider: aws
        default: true
        objectStorage:
          bucket: <bucket_name> 
          prefix: <prefix> 
        config:
          region: <region>
          profile: "default"
          s3ForcePathStyle: "true" 
          s3Url: <s3_url> 
        credential:
          key: cloud
          name: cloud-credentials
----
