= Private {product-title} on {GCP} with Private Service Connect architecture model

Private with GCP Private Service Connect: cluster API server endpoint and application routes private. Public subnets or NAT gateways are not required in your VPC for egress Red Hat SRE management access the cluster over the GCP Private Service Connect-enabled private connectivity.  API server endpoint and default ingress controller are private. Additional ingress controllers can be public or private.
