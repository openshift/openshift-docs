= Power monitoring Technology Preview 0.5 enhancements
//swap out names for attributes where appropriate

With this update, Red Hat OpenShift power monitoring, based on the Kepler Project:

** Provides more accurate data
** Removes some functionality
** Is not backwards compatible with previous versions of OpenShift power monitoring.

This update includes new metrics and dashboards. The older metrics and dashboards are no longer supported. If you are managing your own custom dashboard or queries, you need to update to the newer versions.
