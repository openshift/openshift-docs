= How you can use the Cluster Samples Operator

[role="_abstract"]
To manage sample image streams and templates in {product-title}, you can use the Cluster Samples Operator. The Cluster Samples Operator creates default samples during initial startup to initiate image streams and templates in the `openshift` namespace.
