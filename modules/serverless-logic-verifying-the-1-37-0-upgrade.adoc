= Verifying the 1.37.0 upgrade

[role="_abstract"]
After restoring workflows and services, verify that the upgrade was successful and all components are functioning as expected.

.Procedure

. Verify that the {ServerlessLogicOperatorName} 1.37.0 is installed correctly by running the following command:
+
[source,terminal]
----
$ oc get csv -n openshift-serverless-logic
----

. Verify that the output shows the operator in the `Succeeded` phase as follows:
+
[source,terminal]
----
NAME                          DISPLAY                               VERSION   REPLACES                      PHASE
logic-operator.v1.37.0        OpenShift Serverless Logic Operator   1.37.0                                  Succeeded
----
