= Managing collections by using the API

[role="_abstract"]
You can configure collections programmatically by using the `CollectionService` API object, which provides methods for creating, updating, and testing collection configurations.

You can configure collections by using the `CollectionService` API object. For example, you can use `CollectionService_DryRunCollection` to return a list of results equal to the live preview panel in the {product-title-short} portal. For more information, go to *Help* -> *API reference* in the {product-title-short} portal.
