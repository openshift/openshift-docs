= How autonomous triggering works

[role="_abstract"]
The agent subscribes to Prometheus alerts and triggers investigations automatically without requiring user text input.

When a Prometheus alert fires, Alertmanager sends a webhook notification to the agent. The agent evaluates the alert against the configured alert subscription policies and, if the alert matches a policy, begins an autonomous investigation.
