= About {ocp-ai} vLLM 
:context: ols-about-openshift-ai-vllm

[role="_abstract"]
{ocp-ai} supports the Virtual Large Language Model (vLLM) serving runtime for use with {ols-long} through an {openai}-compatible API.

While {ols-long} has not been fully tested with this runtime, you can use the {openai} configuration instructions to connect to your {ocp-ai} vLLM endpoint.
