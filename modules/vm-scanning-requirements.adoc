= Requirements for virtual machine scanning

[role="_abstract"]
To use {product-title} ({product-title-short}) to scan virtual machines (VMs), you must ensure that they meet specific requirements.

{ocp-virt-first} must meet the following requirements:

* The {ocp-virt} Operator must be installed in the cluster you want to scan.
* The HyperConverged resource must be patched to enable `vsock` support.

VMs must meet the following requirements:

* VM resources must include vsock support with the setting `spec.domain.devices.autoattachVSOCK: true`.
* VMs must run {op-system-base-full}. You must register any host where the scan agent is installed with Red Hat Subscription Manager and have an active {op-system-base} subscription.
* VMs must have network access to download repository-to-Common Platform Enumeration (CPE) mappings.
* DNF must be installed.

For optimal performance, host VMs on metal nodes.
