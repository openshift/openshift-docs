= Clusters section

[role="_abstract"]
The *Clusters* section displays information about the clusters associated with the compliance scan.

The *Clusters* section has the following groups:

Cluster:: Lists the one or more clusters associated with a compliance scan.
Operator status:: Indicates the current health or operational status of the Operator.
