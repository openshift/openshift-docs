= Encrypt the etcd disk

You can encrypt the etcd disk to secure the cluster and protect sensitive data that is stored in the database. The etcd data is not encrypted by default. However, you can encrypt the etcd disk by using the Linux Unified Key Setup-on-disk-format (LUKS) management tools for block device encryption.
