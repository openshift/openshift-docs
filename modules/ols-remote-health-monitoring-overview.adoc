= Remote health monitoring overview

[role="_abstract"]

Remote Health Monitoring uses the Telemeter Client and Insights Operator to gather and report cluster information for {red-hat} analysis and support.

You can learn how {red-hat} collects data in the {ocp-product-title} documentation. To stop sending chat transcripts or feedback, you must opt out of remote health monitoring. Follow the steps in the "About remote health monitoring" section of the {ocp-product-title} documentation.
