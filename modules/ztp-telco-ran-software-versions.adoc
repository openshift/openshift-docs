* cert-manager Operator is a platform agnostic operator.  
Its support lifecycle does not extend to the full lifecycle of {product-title}. 
With all platform agnostic operators, customers need to anticipate the need to update to a newer minor version of the operator at the end of the operator lifecycle of the current version or when planning to update the {product-title} cluster to stay supported. 
While a given operator minor version may not support the latest available {product-title} release, there will always be a version of the operator available for every supported version of {product-title} at any given point in time. 
For terms of support, see the Platform Agnostic Operators section of link:https://access.redhat.com/support/policy/updates/openshift_operators[OpenShift Operator Life Cycles].
