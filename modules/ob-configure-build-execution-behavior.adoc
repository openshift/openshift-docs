= Configure build execution behavior

[role="_abstract"]
Configure how builds execute by setting step resources, Pod configurations, strategy parameters, and system parameters.
These settings manage resource allocation, execution behavior, and build strategies.
Adjust these parameters to optimize build performance and resource usage for your environment.

This section covers the following topics:

* Step resource definition and Pod configuration
* Strategy parameters definition
* System parameters definition
