= Before you begin

[role="_abstract"]
Verify you have the following before starting this walkthrough:

*Hardware requirements*

* At least one bare-metal worker node with an SR-IOV-capable network interface controller (NIC)
* The NIC must support SR-IOV and have available virtual functions

*Access requirements*

* Cluster administrator access to your {product-title} cluster
* Command-line access with the `oc` CLI tool installed

*Knowledge requirements*

* Basic understanding of Kubernetes pods and networking
* Familiarity with YAML configuration files
* Basic command-line skills

[NOTE]
====
This walkthrough uses static IP addressing for simplicity. Dynamic IP assignment with DHCP or Whereabouts is covered in later guides.
====
