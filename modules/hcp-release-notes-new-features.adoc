Monitor connectivity from the data plane to the control plane::
+
In this release, you can monitor connectivity from the data plane to the control plane by using the `ControlPlaneConnectionAvailable` condition. For more information, see xref:../hosted_control_planes/hcp-observability.adoc#hcp-connect-control-plane_hcp-observability[Connectivity monitoring from the data plane to the control plane].

Back up etcd data for {hcp} by using the etcd snapshot method::
+
