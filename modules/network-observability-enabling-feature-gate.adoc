= Enable network observability feature gate

[role="_abstract"]
Enable the `NetworkObservabilityInstall` feature gate to configure network observability installation during cluster creation.

Network observability Day 0 configuration is a Technology Preview feature that requires enabling the `NetworkObservabilityInstall` feature gate through a manifest file during cluster installation.

[IMPORTANT]
====
Enabling the `CustomNoUpgrade` feature set cannot be undone and prevents minor version updates. Use this feature set only on test clusters where you can fully test Technology Preview features.
====

.Prerequisites

* You must have the `openshift-install` binary.

.Procedure

. Create the initial installation configuration by running the following command:
+
[source,terminal]
----
$ openshift-install create install-config
----

. Create the installation manifests by running the following command:
+
[source,terminal]
----
$ openshift-install create manifests
----

. Create a file named `99-feature-gate.yaml` in the `manifests` directory:
+
[source,terminal]
----
$ cat <<EOF > manifests/99-feature-gate.yaml
apiVersion: config.openshift.io/v1
kind: FeatureGate
metadata:
  name: cluster
spec:
  featureSet: CustomNoUpgrade
  customNoUpgrade:
    enabled:
    - NetworkObservabilityInstall
EOF
----
+
where:
+
`spec.featureSet`:: Set to `CustomNoUpgrade` to enable a custom set of Technology Preview features.
`spec.customNoUpgrade.enabled`:: Lists the specific feature gates to enable. Include `NetworkObservabilityInstall` to enable network observability Day 0 configuration.

. Continue with the cluster installation by running the following:
+
[source,terminal]
----
$ openshift-install create cluster
----

.Verification

* After cluster installation completes, verify that the feature gate is enabled by running the following command:
+
[source,terminal]
----
$ oc get featuregate cluster -o yaml
----
+
The output should show `NetworkObservabilityInstall` in the enabled features list:
+
[source,yaml]
----
spec:
  featureSet: CustomNoUpgrade
  customNoUpgrade:
    enabled:
    - NetworkObservabilityInstall
----
