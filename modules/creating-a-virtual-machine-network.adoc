= Creating a virtual machine network from a physical network

[role="_abstract"]
To provide physical network access to virtual machines, create a virtual machine network.

.Procedure

. From the *Virtualization* perspective, click *Physical networks*.

. Click the three dots to the right of the network that you want to edit.

. Click *Create a virtual machines network using this physical network*.

. In the *Network definition* step:
.. Enter a **Name** for the VM network.
.. Select the **Physical network** that you previously created.
.. Optional: Select **VLAN tagging** and enter a **VLAN ID**.

. In the *Project mapping* step, define which projects can access this network:

. Click *Create*.
