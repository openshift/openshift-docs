= roxctl helm output usage

[role="_abstract"]
Usage syntax for the `roxctl helm output` command.

.Usage
[source,terminal]
----
$ roxctl helm output <central_services or secured_cluster_services> [flags]
----

where:

--
`<central_services or secured_cluster_services>`:: Specifies the path to either the central services or the secured cluster services to generate a Helm chart output.
--
