= Bug fixes in version 4.10.4

[role="_abstract"]
This release provides the following bug fixes:

//ROX-35012
* Before this update, loading compliance profiles from the Compliance Operator at scale could cause Central to crash during startup. With this release, Red{nbsp}Hat has increased the timeout for compliance profile operations. As a result, Central can successfully load large numbers of compliance profiles without crashing.

This release also addresses the following security vulnerabilities:

//ROX-35017
* golang.org/x/crypto/ssh: Authorization bypass due to skipped source-address validation (link:https://access.redhat.com/security/cve/CVE-2026-46595[CVE-2026-46595])

//ROX-35021
* golang.org/x/net/idna: Privilege escalation via incorrect Punycode label processing (link:https://access.redhat.com/security/cve/CVE-2026-39821[CVE-2026-39821])

//ROX-35140
* golang.org/x/crypto/ssh/knownhosts: Revocation bypass via unchecked SignatureKey (link:https://access.redhat.com/security/cve/CVE-2026-42508[CVE-2026-42508])
