= About {product-title}

[role="_abstract"]
You can use {prodname} as a control plane for configuring and deploying ingress gateways based on the link:https://gateway-api.sigs.k8s.io/[Kubernetes Gateway API] standard. Extend the benefits of Gateway API by using {prodname} to attach policies to gateways. With these policies, you can do the following tasks:

* Configure gateways with TLS policies for:
** Certificate management
** Authentication
** Authorization
** Rate limiting

* Integrate DNS policies for:
** Multicluster load balancing
** Health checks
** Remediation

In addition, you can use the following included templates:
* Observability dashboards
* Observability metrics
* Tracing
* Alerts
