= Automatic pruning of task runs and pipeline runs

[role="_abstract"]
You can automatically prune stale `TaskRun` and `PipelineRun` resources to free up cluster resources. {pipelines-title} provides a configurable pruner that removes unused objects based on your policies.

[NOTE]
====
You can configure the pruner for your entire cluster by using the `TektonConfig` custom resource. You can also override configuration for a namespace by using namespace annotations. However, you cannot selectively auto-prune an individual task run or pipeline run.
====
