= Next steps after completing preparatory tasks

[role="_abstract"]
After you completed preparatory configurations, you can complete additional tasks.

These additional tasks are listed as follows:

* xref:../../scalability_and_performance/what-huge-pages-do-and-how-they-are-consumed-by-apps.adoc#what-huge-pages-do_huge-pages[Configure the Node Tuning Operator with huge pages support].
* After you deploy your cluster, you can xref:../../networking/networking_operators/sr-iov-operator/installing-sriov-operator.adoc#installing-sr-iov-operator_installing-sriov-operator[install the SR-IOV Operator], xref:../../networking/hardware_networks/configuring-sriov-device.adoc#nw-sriov-networknodepolicy-object_configuring-sriov-device[configure an SR-IOV network device], and xref:../../machine_management/creating_machinesets/creating-machineset-osp.adoc#machineset-yaml-osp-sr-iov_creating-machineset-osp[create SR-IOV compute machines].
* After you deploy your cluster, you can improve the performance of your cluster by completing any of the following tasks:
** Create a xref:../../networking/hardware_networks/using-dpdk-and-rdma.adoc#nw-openstack-ovs-dpdk-testpmd-pod_using-dpdk-and-rdma[a test pod template for clusters that use OVS-DPDK on {rh-openstack}].
** Create a xref:../../networking/hardware_networks/configuring-sriov-device.adoc#nw-openstack-sr-iov-testpmd-pod_configuring-sriov-device[test pod template for clusters that use SR-IOV on {rh-openstack}].
** Create a xref:../../scalability_and_performance/cnf-tuning-low-latency-nodes-with-perf-profile.adoc#installation-openstack-ovs-dpdk-performance-profile_cnf-tuning-low-latency-nodes-with-perf-profile[performance profile template for clusters that use OVS-DPDK on {rh-openstack}].
