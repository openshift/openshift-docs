= Data plane workload backup

[role="_abstract"]
As part of the process to back up and restore by using the {oadp-short} Operator, you can back up the data plane workload.

If the data plane workload is not important, you can skip this procedure. 

To back up the data plane workload by using the {oadp-short} Operator, see "Backing up applications". After you complete those steps, you can restore your hosted cluster by using {oadp-short}.
