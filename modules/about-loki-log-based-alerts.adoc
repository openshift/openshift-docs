= Log-based alerts for Loki

[role="_abstract"]
Configure log-based alerts for Loki by creating `AlertingRule` custom resources. Log-based alerting enables you to trigger alerts based on log patterns and volumes, complementing metric-based alerting to provide comprehensive observability. These alerts integrate with {product-title} monitoring and can route to external alerting systems.
