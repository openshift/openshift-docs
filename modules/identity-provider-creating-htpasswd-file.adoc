= Creating the htpasswd file

[role="_abstract"]
Create an htpasswd file before you configure the htpasswd identity provider so user names and hashed passwords are available for the cluster secret.

Before you create the secret and custom resource, generate a flat htpasswd file on your workstation. Use the procedure that matches your operating system:

* Creating an htpasswd file using Linux
* Creating an htpasswd file using Windows
