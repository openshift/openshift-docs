= roxctl helm derive-local-values usage

[role="_abstract"]
Usage syntax for the `roxctl helm derive-local-values` command.

.Usage
[source,terminal]
----
$ roxctl helm derive-local-values --output <path> \
<central_services> [flags]
----

where:

--
`<path>`:: Specifies the path where you want to save the generated local values file.
`<central_services>`:: Specifies the path to the central services configuration file.
--
