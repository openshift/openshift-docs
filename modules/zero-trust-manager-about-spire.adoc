= SPIRE server


A SPIRE server is composed of one or more SPIRE agents. The server manages and issues SPIFFE identities within a trust domain. It stores registration entries (selectors that determine under what conditions a SPIFFE ID should be issued) and signing keys. The server also performs node and workload attestation to verify the identity of the node or workload requesting a {svid-full} based on predefined conditions. For more information, see link:https://spiffe.io/docs/latest/spire-about/spire-concepts/#spire-architecture-and-components[SPIRE architecture and components].
