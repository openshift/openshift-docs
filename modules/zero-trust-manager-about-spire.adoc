= SPIRE Server

[role="_abstract"]
Manage and issue SPIFFE identities within a trust domain using the SPIRE Server. By storing registration entries and signing keys, you can control identity issuance and perform node attestation with the SPIRE Agent. For more information, see link:https://spiffe.io/docs/latest/spire-about/spire-concepts/#all-about-the-server[About the SPIRE Server].
