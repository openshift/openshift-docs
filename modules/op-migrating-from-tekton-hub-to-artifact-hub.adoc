= Migrating from {tekton-hub} to Artifact Hub

[role="_abstract"]
{tekton-hub} earlier provided a hosted catalog of prebuilt Tekton resources, including Tasks and Pipelines. This service is deprecated in favor of _Artifact Hub_, a centralized catalog for Tekton resources.

[IMPORTANT]
====
* The hub resolver now defaults to the `artifact` type.
* {tekton-hub} (`type: tekton`) requires additional configuration to continue functioning.
====
