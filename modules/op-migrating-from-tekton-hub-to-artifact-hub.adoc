= Migrating from Tekton Hub to Artifact Hub

[role="_abstract"]
{tekton-hub} provided a catalog of prebuilt Tekton tasks and pipelines. Red Hat deprecated this service. Use _{artifact-hub}_, a centralized catalog for Tekton resources, instead.

[IMPORTANT]
====
* The hub resolver now defaults to the `artifact` type.
* Tekton Hub (`type: tekton`) requires additional configuration to continue functioning.
====
