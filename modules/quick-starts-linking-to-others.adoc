= Linking to other quick starts

.Procedure

* In the `nextQuickStart` section of the YAML file, provide the `name`, not the `displayName`, of the quick start to which you want to link. For example:
+
[source,yaml]
----
nextQuickStart:
  - add-healthchecks
----
