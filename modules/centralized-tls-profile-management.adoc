= Centralized TLS profile management and consistency

[role="_abstract"]

// JIRA: ROX-32017

Release Note Text: With this release, {product-title-short} Transport Layer Security (TLS) profiles observe the TLS setting that was set on the OpenShift cluster. Be aware that settings for Central and an OpenShift-based secure cluster might differ. There is no change for non-OpenShift secured clusters.
