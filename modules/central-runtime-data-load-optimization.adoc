= Central runtime data load optimization

[role="_abstract"]

// JIRA: ROX-32157
Central's runtime data loading process has been optimized for better performance.

This optimization reduces Central startup time, improves memory efficiency, and enhances responsiveness when processing data from secured clusters. The improvements are particularly noticeable in environments with large numbers of deployments and network policies.
