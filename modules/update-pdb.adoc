= Ensuring that workloads run uninterrupted with pod disruption budgets
//Please check

Configuration of link:https://docs.redhat.com/en/documentation/openshift_container_platform/4.18/html/postinstallation_configuration/post-install-cluster-tasks#post-install-pod-disruption-budgets[PDB (Pod Disruption Budget)] is one of the most important application side settings that is often overlooked. The upgrade of worker nodes can be halted if the PDB is not set properly.
