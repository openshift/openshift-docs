= Customizing the network traffic table

[role="_abstract"]
Filter and manage the network observability flow table to isolate specific metadata or visualize cluster traffic trends over a selected time range.

.Prerequisites
* You have `cluster-admin` permissions.
* The Network Observability Operator is installed.

.Procedure
. Navigate to *Observe* -> *Network Traffic* in the {product-title} web console.
. Click the *Network Traffic* tab.
. Click the *Manage columns* icon and select the metadata fields you want to display.
. Click *Show Histogram* to display time-series histogram above the table.
.. To filter the table by a specific time range, move the selected range on the histogram. Alternatively, press the Page Up or Page Down keys while the histogram is focused.
.. To shift the displayed time range, click and drag the histogram. The selected time range in the top right corner of the console updates automatically. Alternatively, use the Left Arrow or Right Arrow keys while the histogram is focused.
.. To zoom in or out of the displayed time range, use the scroll wheel or press the Plus (+) or Minus (-) keys while the histogram is focused.

.Verification
* Confirm the table displays the selected columns.
* Verify that the graph updates dynamically as you apply filters.
