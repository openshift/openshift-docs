= CVEs reported only by the StackRox Scanner

[role="_abstract"]
The single CVE reported only by StackRox Scanner was a false positive based on manual review of VEX data.

The single CVE reported only by StackRox Scanner was a false positive. CVE-2022-4122 was flagged for the `podman` package in version `5:5.2.2-1.rhaos4.17.el9.x86_64`. However, a manual review of VEX data from RHSA-2024:9102 indicated that this vulnerability was fixed in version `5:5.2.2-1.el9`. Therefore, the package version scanned was the first to contain the fix and was no longer affected.
