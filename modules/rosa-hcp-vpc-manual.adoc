= Requirements for manually creating an AWS Virtual Private Cloud

[role="_abstract"]
If you do not use a managed infrastructure tool such as Terraform or AWS CloudFormation to create your Virtual Private Cloud (VPC), you can create it manually through the link:https://us-east-1.console.aws.amazon.com/vpc/[AWS console]. A manually created VPC must meet specific requirements for use with {product-title}.
