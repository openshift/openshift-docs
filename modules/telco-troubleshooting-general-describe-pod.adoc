= Describing a pod

Describing a pod gives you information about that pod to help with troubleshooting.
The `Events` section provides you with detailed information about the pod and the containers inside of that pod.

.Procedure

* Describe a pod by running the following command:
+
--
[source,terminal]
----
$ oc describe po -n <namespace> busybox-1
----
