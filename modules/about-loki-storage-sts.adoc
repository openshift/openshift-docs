= Deploying a Loki log store on a cluster that uses short-term credentials

[role="_abstract"]
For some storage providers, you can use the Cloud Credential Operator utility (`ccoctl`) during installation to implement short-term credentials. These credentials are created and managed outside the {ocp-product-title} cluster, providing enhanced security through automatic credential rotation and reduced credential lifetime.

[NOTE]
====
Short-term credential authentication must be configured during a new installation of {loki-op}, on a cluster that uses this credentials strategy. You cannot configure an existing cluster that uses a different credentials strategy to use this feature.
====
