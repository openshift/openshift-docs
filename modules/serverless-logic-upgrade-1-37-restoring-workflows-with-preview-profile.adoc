= Restoring workflows with the preview profile

[role="_abstract"]
After upgrading the {ServerlessLogicOperatorName} to version 1.37.0, you must redeploy workflows that use the `preview` profile to complete the upgrade process.

.Procedure

. Ensure that you have a copy of the corresponding SonataFlow custom resource (CR) and any other Kubernetes resources created for the workflow. For example, if you are using custom property configurations, ensure that you have a copy of the user-provided `ConfigMap` containing the `application.properties` file.

. Delete the workflow by running the following command: 
+
[source,terminal]
----
$ oc delete -f <workflow_name> -n <target_namespace>
----
