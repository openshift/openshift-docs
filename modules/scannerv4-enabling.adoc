= Enabling Scanner V4

[role="_abstract"]
Scanner V4 is enabled by default when {product-title-short} is newly installed. However, if you have not previously enabled Scanner V4 and are upgrading from version 4.7 or earlier, you must enable it explicitly during the upgrade to use it.

You can enable Scanner V4 after installation by using the Operator or Helm.
