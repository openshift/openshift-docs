= roxctl central init-bundles generate usage

[role="_abstract"]
Usage syntax for the `roxctl central init-bundles generate` command.

.Usage
[source,terminal]
----
$ roxctl central init-bundles generate <init_bundle_name> [flags]
----

where:

--
`<init_bundle_name>`:: Specifies the name for the initialization bundle you want to generate.
--
