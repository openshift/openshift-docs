= Preparing to install a cluster that uses OVS-DPDK

[role="_abstract"]
You must prepare the underlying {rh-openstack} data plane and supporting resources before you install a cluster that uses OVS-DPDK on it.

* Create an OVS-DPDK flavor and deploy a test instance on {rh-openstack} before you install a cluster. For information about creating flavors, see link:https://docs.redhat.com/en/documentation/red_hat_openstack_services_on_openshift/18.0/html/configuring_the_compute_service_for_instance_creation/assembly_creating-flavors-for-launching-instances_instance-flavors[Creating flavors for launching instances]. For information about preparing the RHOSO data plane for OVS-DPDK workloads, see link:https://docs.redhat.com/en/documentation/red_hat_openstack_services_on_openshift/18.0/html/deploying_a_network_functions_virtualization_environment/assembly_create-data-plane-sriov-dpdk_rhoso-nfv[Creating a data plane for SR-IOV and DPDK].
