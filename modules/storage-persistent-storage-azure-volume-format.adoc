= Volume format

[role="_abstract"]
{product-title} verifies that a volume contains the file system specified by the `fsType` parameter in the persistent volume definition before mounting it to a container. An unformatted device is erased and automatically formatted with the specified file system.

This allows using unformatted Azure volumes as persistent volumes, because
{product-title} formats them before the first use.
