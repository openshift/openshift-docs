= Control plane VM requirements

[role="_abstract"]
Each control plane VM must meet minimum resource requirements to run the {product-title} control plane components reliably.

Each control plane VM requires the following minimum resources:

* 8 vCPUs
* 16 GiB RAM
* 120 GiB storage
+
[NOTE]
====
Use {lvms-first} backed by NVMe drives for virtualized control plane storage.
====
