= Verifying cluster-wide proxy

[role="_abstract"]
For a customer would be to make sure that all traffic related to git or container goes through proxy only

the openshift container platform takes care of how the proxy would be managed and configured, so we do not need to mention it
this should be checked after build is installed or at least before running, the assumption is that a customer using proxy is already set is up on the OCP and only need to verify when using builds
