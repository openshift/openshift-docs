= Using AssumeRole with Amazon ECR

[role="_abstract"]
You can use AssumeRole to grant access to AWS resources without manually configuring each user's permissions. Instead, you can define a role with the required permissions so that the user is granted access to assume that role. `AssumeRole` enables you to grant, revoke, or otherwise generally manage more fine-grained permissions.
