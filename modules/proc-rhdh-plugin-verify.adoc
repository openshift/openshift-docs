= Confirming that developers can discover and access APIs

[role="_abstract"]
After you complete the integration, verify that API products appear in the Developer Hub catalog. Verify that developers can request access and generate API keys.

.Prerequisites

* You installed the {rhdh} plugin.

.Procedure

. Get the {rhdh} route URL by running the following commands:
+
[source,terminal]
----
$ export RHDH_URL=$(oc get route backstage-rhdh -n ${RHDH_NS} -o jsonpath='{.status.ingress[0].host}')
$ echo "https://${RHDH_URL}"
----

. Navigate to the {rhdh} URL in your web browser.

. Navigate to the `/kuadrant` endpoint to view the {prodname} plugin page:
+
[source,text]
----
https://${RHDH_URL}/kuadrant
----

. Verify that `APIProduct` objects appear in the catalog by navigating to the catalog view.

. Select an API entity and verify that the Kuadrant tab is visible and displays {prodname} policy and configuration information.

.Verification

* You can create and manage API keys through the plugin interface based on your assigned role.
