= Exploring the REST API by using the Swagger UI

[role="_abstract"]
The interactive Swagger UI provides a platform to explore and test {ols-long} REST API endpoints. You can use this interface to verify request parameters and response schemas directly from your browser to ensure seamless API integration.

You can explore and test the {ols-long} REST API endpoints by using the interactive Swagger UI. 

.Prerequisites
* The {ols-long} service is running. 

.Procedure
. Open a web browser and navigate to `http://localhost:8080/docs`.
. Expand any endpoint tag to view its parameters, request body schema, and possible response codes.
. Click *Try it out* to send live requests directly from your browser.
+
[NOTE]
====
If authentication is enabled, you must provide a valid `Bearer` token by clicking *Authorize* before sending requests. Unauthenticated requests return `401` or `403` status codes.
====
