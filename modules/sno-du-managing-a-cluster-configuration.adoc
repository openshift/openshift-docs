= Managing a cluster configuration

.Procedure

* Use the following example to configure the managed cluster:
[source,yaml]
----
apiVersion: cluster.open-cluster-management.io/v1
kind: ManagedCluster
metadata:
    labels:
        common: "true"
        group-du-sno: ""
        sites: site-sno-du-1
    name: sno-du-1
spec:
    hubAcceptsClient: true
----
