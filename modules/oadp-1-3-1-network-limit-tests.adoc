= {oadp-short} 1.3.1 network limitation tests for backing Kopia

The main focus of this {oadp-first} testing is to identify the minimal amount of bandwidth required for Kopia to successfully complete backups.

Additionally, this testing  explores scenarios with high latency, inadequate bandwidth, and jittery connections to understand how Kopia behaves under varying network conditions.
