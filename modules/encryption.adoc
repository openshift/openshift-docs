= Encryption

[role="_abstract"]
Red{nbsp}Hat uses AWS managed keys that are rotated annually for encryption.

Red{nbsp}Hat uses "AWS managed keys" which are rotated by AWS each year. For information on the use of keys, see "AWS KMS key management". For more information about RDS, see "Amazon RDS Security".
