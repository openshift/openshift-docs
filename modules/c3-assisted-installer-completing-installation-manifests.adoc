= Adding custom manifests

Create, modify and upload the four mandatory custom manifests provided by Oracle:

** In the `C3/custom_manifests_C3/`manifests` folder:

*** `oci-ccm.yml`
*** `oci-csi.yml`

** In the C3/custom_manifests_C3/`openshift` folder:

*** `machineconfig-ccm.yml`
*** `machineconfig-csi.yml` 

.Prerequisites

* Prepare the custom manifests. For details, see step 8 and onwards of link:https://www.oracle.com/a/otn/docs/compute_cloud_at_customer_assisted_installer.pdf?source=:em:nl:mt::::PCATP[Install the Cluster using the RH Assisted Installer UI].

.Procedure

. Create and save the `oci-ccm.yml` and `oci-csi.yml` files: 

.. On the *Custom manifests* page, in the *Folder* field, select `manifests`. 

.. In the *File name* field, enter `oci-ccm.yml`.

.. Paste the file contents from the file you prepared.

.. In the *Content* section, click the *Paste content* icon. 

.. If you are using Firefox, click *OK* to close the dialog box, and then press *Ctrl+V*. Otherwise, skip this step.  

.. Click *Add another manifest* and repeat the above steps for the `oci-csi.yml` file.

. Create and save the `machineconfig-ccm.yml` and `machineconfig-csi.yml` files:

.. Click *Add another manifest* and, in the *Folder* field, select `openshift`. 

.. In the *File name* field, enter `machineconfig-ccm.yml`.

.. Paste the file contents from the file you prepared.

.. In the *Content* section, click the *Paste content* icon. 

.. If you are using Firefox, click *OK* to close the dialog box, and then press *Ctrl+V*. Otherwise, skip this step.  

.. Click *Add another manifest* and repeat the above steps for the `machineconfig-csi.yml` file.

. Click *Next* to save the custom manifests.

. From the *Review and create* page, click *Install cluster* to create your {product-title} cluster. This takes approximately thirty minutes.
