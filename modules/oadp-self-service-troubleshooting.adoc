= Troubleshooting {oadp-short} Self-Service

.Error `pods is forbidden`

A non-admin user is trying to access a namespace that they do not have access to. The cluster administrator has not authorized the non-admin user to access the namespace. The resolution is to authorize the non-admin user access the namespace.

[source, terminal]
----
$ oc get pod -n openshift-adp
Error from server (Forbidden): pods is forbidden: User "nac-user" cannot list resource "pods" in API group "" in the namespace "openshift-adp"
----
