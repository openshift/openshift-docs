= About GitLab authentication

[role="_abstract"]
Use GitLab authentication to allow users to log in to {product-title} with GitLab account credentials.

If you use GitLab version 7.7.0 to 11.0, you connect using OAuth integration. If you use GitLab version 11.1 or later, you can use OpenID Connect (OIDC) to connect instead of OAuth. For more information, see "OAuth integration" and "OpenID Connect".
