= Configuring Google workload identity federation

[role="_abstract"]
{product-title-short} integrations can authenticate against the {gc} by using workload identities. Select the *Use workload identity* option upon creation to enable workload identity authentication in a Google Cloud integration.

[IMPORTANT]
====
The Google service account associated with the {product-title-short} pod through the workload identity must have the IAM permissions required by the integration. For example, to set up a workload identity for integrating with Google Artifact Registry, connect a service account with the *roles/artifactregistry.reader* role.
====
