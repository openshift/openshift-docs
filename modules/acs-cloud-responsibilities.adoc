= Shared responsibilities for {product-title-managed-short}

[role="_abstract"]
In {product-title-managed} ({product-title-managed-short}), Red Hat manages the services for the main control center, also known as _Central services_. Red{nbsp}Hat is responsible for platform monitoring, updating software for Central services, and other aspects of maintaining hosted components. Customers are responsible for maintaining access to their systems, updating software on secured clusters, and backing up data on secured clusters.

[cols="2,1,1",options="header"]
|===

|Resource or action
|Red Hat responsibility
|Customer responsibility


|Hosted components, also called Central components
a|* Platform monitoring
* Software updates
* High availability
* Backup and restore
* Security
* Infrastructure configuration
* Scaling
* Maintenance
* Vulnerability management
a| * Access and identity authorization

|Secured clusters (on-premise or cloud)
|
a| * Software updates
* Backup and restore
* Security
* Infrastructure configuration
* Scaling
* Maintenance
* Access and identity authorization
* Vulnerability management

|===
