= Shared responsibilities for {product-title-managed-short}

While Red Hat manages the {product-title-managed-short} services, the customer has certain responsibilities, as shown in the following table.

[cols="2a,3a",options="header"]
|===

|Resource
|Responsibility

|Platform monitoring |Red Hat

a|Hosted components:

* cluster life cycle
* infrastructure configuration
* scaling
* maintenance
|Red Hat

a|Control plane:

* high availability
* uptime
* backups and restore
security
|Red Hat

|Vulnerability management |Customer

|Secured cluster component deployment |Customer

|Secured cluster customer data |Customer

|===
