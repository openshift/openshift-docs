= ServiceNow integration overview

[role="_abstract"]
You can integrate {product-title} with ServiceNow by using the Red{nbsp}Hat Security Container Vulnerability Response Integration or by configuring a generic webhook integration.

You can use {product-title-short} with ServiceNow by using one of the following methods:

* Use the Red{nbsp}Hat Security Container Vulnerability Response Integration to pull data from your {product-title-short} instances to the ServiceNow Vulnerability Response application.
* Configure {product-title} to send events to ServiceNow by configuring a generic webhook integration in {product-title-short}.

To integrate {product-title-short} with ServiceNow by using a generic webhook integration, complete the following steps:

. In ServiceNow, configure a REST API endpoint to use in {product-title-short}.
. In {product-title-short}, complete these steps:
.. Configure the generic webhook.
.. Identify policies for which you want to send notifications, and update the notification settings for those policies.
