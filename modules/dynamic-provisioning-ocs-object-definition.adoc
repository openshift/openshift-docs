= Red Hat OpenShift Container Storage object definition

When using Red Hat OpenShift Container Storage, the storage classes for dynamic
volume provisioning are created when Red Hat OpenShift Container Storage 4.4 is
deployed from the Operator Hub as described in
link:https://access.redhat.com/documentation/en-us/red_hat_openshift_container_storage/4.4/html-single/deploying_openshift_container_storage/index#verify_that_the_storage_classes_are_created_and_listed[Verify that the storage classes are created and listed].
