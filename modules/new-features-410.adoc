= New features

[role="_abstract"]
To strengthen your security posture, implement the new configuration and monitoring standards in {product-title-short} 4.10. You can configure global node taints, define standardized base images, and monitor file activity to support strict workload isolation and compliance. When you register new secured clusters, select the cluster registration secret (CRS) method instead of an init bundle to securely isolate your bootstrapping credentials.
