= Cert-Manager Operator

New in this release::
* https://issues.redhat.com/browse/CNF-13731[CNF-13731: Extend Core reference configuration with Cert Manager]
* Cert-Manager is a new optional component in this release.

Description::
+
--
The Cert-Manager Operator for {product-title} manages the lifecycle of TLS certificates for cluster components and workloads.
Cert-Manager automates certificate issuance, renewal, and rotation, eliminating manual certificate management.
The reference configuration includes Cert-Manager to optionally manage certificates for the API server and ingress controller endpoints.
--

Limits and requirements::

* The reference configuration includes only the ACME DNS01 challenge type for platform certificate issuance.

Engineering considerations::

* Use {rh-rhacm} `CertificatePolicy` resources on the hub cluster to monitor certificate expiration and compliance across managed clusters.
