= Monitoring Gateway API metrics and performance

[role="_abstract"]
Monitor Gateway API metrics to analyze the performance and health of your traffic routing. Reviewing these metrics helps you identify latency issues, pinpoint bottlenecks, and optimize your overall network configuration.

.Prerequisites

* You have access to the {product-title} cluster with `cluster-admin` privileges.
* You have access to the {product-title} web console.

.Procedure

. Log in to the {product-title} web console.
. Navigate to *Observe* -> *Metrics*.
. In the *Expression* field, enter a relevant Gateway API or HAProxy metric query. 
. Click *Run queries* to view the performance data and analyze the traffic behavior.
. Review the graphical output to identify performance spikes, request latency, or dropped connections.
