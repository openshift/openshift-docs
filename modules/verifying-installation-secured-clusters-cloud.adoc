= Verifying installation of secured clusters

[role="_abstract"]
After installing {product-title-managed-short}, you can verify that the installation was successful by accessing your ACS Console and checking the Dashboard.

To verify installation, access your ACS Console from the {cloud-console}. The Dashboard displays the number of clusters that {product-title-managed-short} is monitoring, along with information about nodes, deployments, images, and violations.
