= Build advanced settings

[role="_abstract"]
You can configure secure parameters, metadata annotations, retention policies, and system result reporting for build executions.
Advanced settings control build lifecycle behavior, security context, and observability beyond basic source and output configuration.

Secure parameters:: Configure environment variables and parameter values that contain sensitive data, ensuring they are not exposed in build logs or status fields.
Annotations and labels:: Define metadata annotations and labels that propagate to `BuildRun` resources and resulting container images.
Retention policies:: Specify how long `BuildRun` history and associated artifacts remain available before automatic cleanup occurs.
System results:: Enable detailed result reporting that captures build execution metrics, build step durations, and resource consumption data.

Configure advanced settings to maintain secure build environments, implement compliance requirements, control cluster storage consumption, and gather detailed build performance metrics.
