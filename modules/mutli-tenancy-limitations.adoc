= Limitations

[role="_abstract"]
Multi-tenancy within {product-title-short} is not possible for resources with a global scope, which all users within a Central instance share.

Achieving multi-tenancy within {product-title-short} is not possible for resources with a *global scope*.

The following resources have a global scope:

* Access
* Administration
* Detection
* Integration
* VulnerabilityManagementApprovals
* VulnerabilityManagementRequests
* WatchedImage
* WorkflowAdministration

All users within a {product-title-short} Central instance share these resources and you cannot scope them.
