= {microshift-short} Kubernetes APIs

[role="_abstract"]
You can manage the fully accessible Kubernetes API in {microshift-short} by using the `kubectl` or {OCP} (`oc`) command-line tools.

Using these command-line tools with {microshift-short} can help you access all of the resources you need to work with your deployments.
