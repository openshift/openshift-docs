= Configuring identity providers using the web console

[role="_abstract"]
Configure identity providers through the {product-title} web console when you prefer a guided UI workflow instead of applying a custom resource with the CLI.

.Prerequisites

* You must be logged in to the web console as a cluster administrator.

.Procedure

. Navigate to *Administration* -> *Cluster Settings*.
. Under the *Configuration* tab, click *OAuth*.
. Under the *Identity Providers* section, select your identity provider from the *Add* drop-down menu.
+
[NOTE]
====
You can specify multiple IDPs through the web console without overwriting
existing IDPs.
====
