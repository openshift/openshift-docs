= Support for Compliance Operator tailored profiles scheduling and maintenance

[role="_abstract"]

// JIRA: ROX-26032

{product-title-short} now supports seamless integration of existing tailored profiles from Compliance Operator. You can now schedule and manage your custom security configurations directly from the {product-title-short} interface. This update fully accommodates profiles with suppressed rules or multi-profile combinations and profiles that are using the latest `CustomRules` functionality.
