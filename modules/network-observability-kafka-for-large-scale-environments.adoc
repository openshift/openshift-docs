= About Kafka for large-scale environments

[role="_abstract"]
Use the Kafka Operator to manage high-throughput and low-latency data feeds for network flow forwarding. This architecture provides a resilient and scalable solution for handling telemetry data in large-scale cluster environments.

You can install the Kafka Operator as Red Hat AMQ Streams from the Operator Hub. See "Red Hat AMQ Streams documentation".

[NOTE]
====
To uninstall Kafka, refer to the uninstallation process that corresponds with the method you used to install.
====
