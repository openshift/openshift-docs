[id="rosa-cli-no-permissions-required_{context}"]
= ROSA CLI commands with no required permissions

The following ROSA CLI commands do not require permissions or policies in order to execute. They do require an access-key and secret-key configured or a role attached.


[id="rosa-list-cluster_{context}"]
== rosa list cluster

.Input
[source,terminal]
----
$ rosa list cluster
----

[id="rosa-list-versions_{context}"]
== rosa list versions

[source,terminal]

----
