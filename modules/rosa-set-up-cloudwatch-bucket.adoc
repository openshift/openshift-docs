= Setting up the _CloudWatch_ bucket

Set up the `CloudWatch` log group for the log forwarder to send logs to by completing the following steps: 

.Procedure

. Create the `CloudWatch` log group by running the following command: 
+
[source,terminal]
----
$ aws logs create-log-group â€“log-group-name <your_log_group_name>
----
+
. Go to your chosen ROSA cluster and configure the log forwarder to use the `CloudWatch` bucket by applying the following JSON sample:  
+
[source,json]
----
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "CreatePutLogs",
            "Effect": "Allow",
            "Action": [
                "logs:CreateLogStream",
                "logs:PutLogEvents"
            ],
            "Resource": "<your_log_group_arn>:*"
        },
        {
            "Sid": "DescribeLogs",
            "Effect": "Allow",
            "Action": [
                "logs:DescribeLogGroups",
                "logs:DescribeLogStreams"
            ],
            "Resource": "*"
        }
    ]
}
----
+
. Attached the policy to the `CloudWatch` role by running the following command: 
+
[source,terminal]
----
$ 	aws iam put-role-policy \
    --role-name CustomerLogDistribution-RH \
    --policy-name Allow-CloudWatch-Writes \
    --policy-document file://cloudwatch-policy.json
----
