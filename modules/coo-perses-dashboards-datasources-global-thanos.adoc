= Create a global Thanos Querier datasource

[role="_abstract"]
You can create a global datasource to connect Perses to the platform Thanos Querier instance run by the Cluster Monitoring Operator. This makes platform metrics available to dashboards across all projects and namespaces in the cluster.

.Prerequisites

* You have installed the Cluster Observability Operator version 1.5 or later.
* You have enabled the monitoring plugin.
* You have cluster administrator permissions.
* You have created a Kubernetes Secret named `thanos-querier-datasource-secret` containing valid authentication credentials in the operator namespace.

.Procedure

. Create a `PersesGlobalDatasource` custom resource file using the `v1alpha2` API specification:
+
[source,yaml]
----
apiVersion: perses.dev/v1alpha2
kind: PersesGlobalDatasource
metadata:
  name: thanos-querier-datasource
spec:
  config:
    display:
      name: "Thanos Querier Datasource"
    default: true
    plugin:
      kind: "PrometheusDatasource"
      spec:
        proxy:
          kind: HTTPProxy
          spec:
            url: https://thanos-querier.openshift-monitoring.svc.cluster.local:9091
            secret: thanos-querier-datasource-secret
  client:
    tls:
      enable: true
      caCert:
        type: file
        certPath: /ca/service-ca.crt
----
* `spec.config.default`: Set to `true` to make this the default metrics datasource cluster-wide.
* `spec.config.plugin.spec.proxy.secret`: Specifies the Kubernetes Secret holding the proxy authentication credentials.
* `spec.client.tls.caCert.certPath`: Specifies the internal path to the service CA certificate to ensure secure TLS communication.

. Apply the global datasource configuration by running the following command:
+
[source,terminal]
----
$ oc apply -f thanos-querier-datasource.yaml
----

.Verification

* Verify that the global Thanos datasource was successfully created:
+
[source,terminal]
----
$ oc get persesglobaldatasources
----
+
You should see an output similar to the following example:
+
[source,terminal]
----
NAME                        AGE
thanos-querier-datasource   1m
----
