= Dedicated Host configuration options

[role="_abstract"]
You can configure a machine template to place machines on {aws-first} Dedicated Hosts. With dynamic host allocation, {aws-short} automatically selects an available Dedicated Host from a pool of hosts that match specified tags.

:FeatureName: {aws-short} Dedicated Host support
include::snippets/technology-preview.adoc[]

include::snippets/apply-machine-configuration-method.adoc[tag=method-machine-template]

.Sample Dedicated Host configuration with dynamic host allocation
[source,yaml]
----
apiVersion: infrastructure.cluster.x-k8s.io/v1beta2
kind: AWSMachineTemplate
# ...
spec:
  template:
    spec:
      tenancy: host
      hostAffinity: host
      dynamicHostAllocation:
        tags:
          <tag_name>: <tag_value>
# ...
----
+
where:
+
--
`<tag_name>`:: Specifies the tag key to match against available Dedicated Hosts.
`<tag_value>`:: Specifies the tag value to match against available Dedicated Hosts.
--
