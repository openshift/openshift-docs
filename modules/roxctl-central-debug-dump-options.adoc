[id="roxctl-central-debug-dump-options_{context}"]
= Options

[role="_abstract"]
Options for the `roxctl central debug dump` command.

.Options
[cols="2,2",options="header"]
|===
|Option |Description

|`--logs`
|If set to `true`, the command includes logs in the Central dump. The default value is `false`.

|`--output-dir string`
|Specify the output directory for the bundle content. The default value is an automatically generated directory name within the current directory.

|`-t`, `--timeout duration`
|Specify the timeout for API requests, which is the maximum duration of a request. The default value is `5m0s`.
|===
