= LVMO deployment

LVMO is automatically deployed on to the cluster in the `openshift-storage` namespace after {microshift-short} starts.

LVMO uses `StorageCapacity` tracking to ensure that pods with an LVMO PVC are not scheduled if the requested storage is greater than the free storage of the volume group. For more information about `StorageCapacity` tracking, read link:https://kubernetes.io/docs/concepts/storage/storage-capacity/[Storage Capacity].
