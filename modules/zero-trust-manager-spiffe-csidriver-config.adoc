== Deploying and configuring the SPIFFE Container Storage Interface Driver

You can configure the `SpiffeCSIDriverConfigSpec` Custom Resource (CR) to deploy and configure a SPIRE agent.

.Prerequisites

* You have access to the cluster as a user with the `cluster-admin` role.

.Procedure

. Create a YAML file that defines the `SpiffeCSIDriverConfigSpec` CR object:
+
[source,yaml]
----
apiVersion: operator.openshift.io/v1alpha1
kind: SpiffeCSIDriverConfig
metadata:
  name: cluster
spec: {}
----

. Run `oc apply -f <filename>.yaml`

.Verification

* Get the name of the pods on which the SPIFFE Container Storage Interface (CSI) Driver is installed by running the following command:
+
[source,terminal]
----
$ oc get po -l app.kubernetes.io/name=spiffe-csi-driver -n zero-trust-workload-identity-manager
----
+
