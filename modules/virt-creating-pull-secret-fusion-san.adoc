= Creating a kubernetes pull secret

You must create a pull secret in the `ibm-fusion-access` namespace you created when installing the {IBMFusionSAN} Operator. This pull secret holds the IBM entitlement key you need for pulling the required container images from the IBM container registry.

.Prerequisites

* You installed the `oc` CLI.
* You have access to the cluster as a user with the `cluster-admin` role.
