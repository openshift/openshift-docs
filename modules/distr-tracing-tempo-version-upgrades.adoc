= Version upgrades

[role="_abstract"]
For version upgrades, OLM queries for available Operators and available upgrades for installed Operators. When OLM upgrades the {TempoOperator}, the Operator scans for running TempoStack instances and upgrades those instances to match the Operator's updated version.

OLM runs in the {ocp-product-title} by default.
