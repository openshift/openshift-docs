= Version upgrades

[role="_abstract"]
For version upgrades, OLM queries for available Operators and upgrades for installed Operators. When OLM upgrades the {TempoOperator} to the new version, the Operator scans for running TempoStack instances that it manages and upgrades them to the version corresponding to the Operator's new version.

OLM runs in the {ocp-product-title} by default.
