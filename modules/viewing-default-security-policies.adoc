= Viewing default security policies

[role="_abstract"]
You can use the {product-title-short} portal to view default policies, clone them, and edit the cloned default policies. Default policies are not supported with the policies as code feature.

.Procedure

* In the {product-title-short} portal, go to *Platform Configuration* -> *Policy Management*. The *System* label in the *Origin* column indicates default policies. 
+
[NOTE]
====
You cannot delete default policies or edit policy criteria for default policies.
====
