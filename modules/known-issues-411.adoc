= Known issues in version 4.11

[role="_abstract"]
The following known issues exist in {product-title-short} 4.11. Review these items before deploying or upgrading to this version.

* Due to a recently identified data gap in the Red Hat VEX security data feed, {product-title-short} version 4.11 no longer supports vulnerability reporting for Red Hat hardened images (Project Hummingbird). See "Red Hat hardened image scanning" for more information.
