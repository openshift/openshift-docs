= roxctl central db generate openshift pvc

[role="_abstract"]
Generate an OpenShift YAML manifest for a database deployment with a persistent volume claim (PVC) in Central.

.Usage
[source,terminal]
----
$ roxctl central db generate openshift pvc [flags]
----
