= Capture and reuse build information

[role="_abstract"]
Access build snapshots for debugging and reproducibility. The `BuildRun` custom resource (CR) captures a snapshot of the build configuration at execution time, enabling you to reproduce builds with the exact same settings and troubleshoot issues by examining the configuration used during a specific build run.
