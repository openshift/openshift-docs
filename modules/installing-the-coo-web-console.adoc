= Installing the {coo-short} by using the web console

[role="_abstract"]
Install the {coo-full} to enable log visualization in the {ocp-product-title} web console.

.Prerequisites

* You have access to the cluster with `cluster-admin` privileges.
* You have access to the {ocp-product-title} web console.

.Procedure

. In the {ocp-product-title} web console, navigate to *Operators → OperatorHub*.

. Type `Cluster Observability Operator` in the *Filter by keyword* box.

. Select the *Cluster Observability Operator* tile and click *Install*.

. On the *Install Operator* page:
.. Select *development* from the *Update channel* list.
.. Select *openshift-operators* from the *Installed Namespace* list.
+
[NOTE]
====
The {coo-short} must be installed in the `openshift-operators` namespace.
====
.. Select an *Update approval* strategy:
*** *Automatic* updates the Operator automatically when a new version is available.
*** *Manual* requires manual approval before updates are applied.

. Click *Install*.

.Verification

. Navigate to *Operators → Installed Operators*.

. Verify that the *Cluster Observability Operator* is listed with a status of *Succeeded* in the `openshift-operators` namespace.
