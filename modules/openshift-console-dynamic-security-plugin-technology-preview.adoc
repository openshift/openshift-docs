= OpenShift console dynamic security plugin (Technology Preview)

//ROX-28072
[role="_abstract"]
In {product-title-short} 4.10, you gain access to a new dynamic plugin for the OpenShift console on your secured clusters. This plugin adds a Security tab that displays real-time vulnerability information about your cluster. You can now analyze vulnerability data directly within your OpenShift console interface without switching applications. This capability is a Technology Preview feature.

For more information, see "Accessing vulnerability information in the {ocp} web console".
