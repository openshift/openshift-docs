= RBAC alignment across OpenShift Platform Plus offerings

[role="_abstract"]

// JIRA: ROX-32064
Role-Based Access Control (RBAC) has been aligned with other OpenShift Platform Plus offerings.

This alignment provides a more consistent permissions model across Red{nbsp}Hat security and platform products, simplifying access management for organizations using multiple OpenShift Platform Plus components.
