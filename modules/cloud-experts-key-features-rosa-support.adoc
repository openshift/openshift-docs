You can open a ticket directly from the {cluster-manager-url} or visit the Red{nbsp}Hat Customer Portal to search or browse through the Red{nbsp}Hat knowledge base of articles and solutions relating to Red{nbsp}Hat products or submit a support case to Red{nbsp}Hat Support.

== Limited support
If a {product-title} cluster is not upgraded before the "end of life" date, the cluster continues to operate in a limited support status. The SLA for that cluster will no longer be applicable, but you can still get support for that cluster.

Additional support resources:

* link:https://access.redhat.com/[Red{nbsp}Hat Support]
* link:https://aws.amazon.com/premiumsupport/[AWS Support]
+
AWS support customers must have a valid AWS support contract.
