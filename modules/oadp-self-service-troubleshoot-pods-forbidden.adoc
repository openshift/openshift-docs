= Error `pods is forbidden`

If you get a `pods is forbidden` error, it means, as a non-admin user you are trying to access a namespace that you do not have access to. 
The cluster administrator has not authorized the non-admin user to access the namespace.

.Error

[source, terminal]
----
$ oc get pod -n openshift-adp
Error from server (Forbidden): pods is forbidden: User "nac-user" cannot list resource "pods" in API group "" in the namespace "openshift-adp"
----

.Solution

The cluster administrator must authorize the non-admin user to access the namespace. For more details, see "OADP Self-Service namespace permissions".
