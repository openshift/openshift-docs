= Minimum cluster resource requirements

{ols-long} requires the following minimum resources to ensure that the service operates efficiently without affecting other cluster workloads.

[cols="1,1,1"]
|===
| Component | Minimum CPU (Cores) | Minimum Memory

| Application server
| 0.5
| 1024 MB

| Postgres database
| 0.3
| 300 MB
|===
