= Technology Preview features

[role="_abstract"]
You can try the Technology Preview features that are available with {product-title} {version}.

[IMPORTANT]
====
Technology Preview features are not supported with Red Hat production service level agreements (SLAs) and might not be functionally complete. {rh} does not recommend using them in production.

Technology Preview features provide early access to upcoming product features, enabling customers to test functionality and provide feedback during the development process. For more information, see https://access.redhat.com/support/offerings/techpreview[{rh} Technology Preview Features - Scope of Support].
====

Support for `GRPCRoute` policy attachment::

Beginning with {prodname} 1.4.0, `GRPCRoute` policy attachment support as a Technology Preview feature. `GRPCRoute` policy attachment support has many built-in functions for traffic management and is the preferred solution for non-browser-based traffic. For example, when you need to push continuous updates to a client or need long-lived connections, `GRPCRoute` policy attachment support is recommended.
