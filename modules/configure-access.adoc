= Configuring access

[role="_abstract"]
{product-title-short} provides the `roxctl` command-line interface (CLI) to make it easy to integrate {product-title-short} policies into your build pipeline. The `roxctl` CLI prints detailed information about problems and how to fix them so that developers can keep high standards in the early phases of the container lifecycle.

To securely authenticate to the {product-title} API server, you must create an API token.
