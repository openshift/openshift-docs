= Creating a config map by using the CLI

[role="_abstract"]
To provide configuration data to your pods, you can use the {oc-first} to create a config map from directories, specific files, or literal values.

.Procedure

* Create a config map:
+
[source,terminal]
----
$ oc create configmap <configmap_name> [options]
----
