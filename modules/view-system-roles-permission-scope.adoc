= Viewing the permission set and access scope for a system role

[role="_abstract"]
You can view the permission set and access scope for the default system roles.

.Procedure
. In the {product-title-short} portal, go to *Platform Configuration* -> *Access control*.
. Select *Roles*.
. Click one of the roles to view its details. The details page shows the permission set and access scope for the selected role.
+
[NOTE]
====
You cannot change permission set and access scope for the default system roles.
====
