= Object storage types

[role="_abstract"]
Configure supported object storage types for the {coo-full}, including Amazon S3, MinIO, Microsoft Azure Blob Storage, Google Cloud Storage, and their respective authentication methods. Each object storage type has its own set of required fields which are configured directly in the `ObservabilityInstaller` CR.
