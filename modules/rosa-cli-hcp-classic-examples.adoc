= Required minimum permissions for ROSA CLI commands

The following required minimum permissions for the listed ROSA CLI commands are applicable for HCP and Classic clusters.

[id="rosa-create-OIDC-providers-hcp-classic_{context}"]
== create OpenID Connect (OIDC) provider in auto mode
Required minimum permissions to create an OIDC provider in auto mode.

.Input
[source,terminal]
----
$ rosa create oidc-config --mode auto
----
.Minimum permissions
[source,json]
----
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "VisualEditor0",
            "Effect": "Allow",
            "Action": [
                "iam:ListOpenIDConnectProviders",
                "iam:ListOpenIDConnectProviderTags"
            ],
            "Resource": "*"
        }
    ]
}
----
