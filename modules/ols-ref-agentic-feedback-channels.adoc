= Providing feedback

[role="_abstract"]
You can report performance anomalies, analytical errors, or general feedback about the autonomous agentic troubleshooting feature through the following channels.

* *Red{nbsp}Hat Support*: To report issues that affect your cluster health or require urgent attention, open a support case through the link:https://access.redhat.com/support/cases/[Red{nbsp}Hat Customer Portal].

* *Feedback in the web console*: Use the feedback mechanism in the {ocp-product-title} web console to report suggestions, usability issues, or general feedback about the agentic troubleshooting experience.

* *Bugzilla*: To report bugs or request enhancements, file a report in link:https://bugzilla.redhat.com/[Red{nbsp}Hat Bugzilla] under the {ols-long} component.

If you observe any of the following, report the issue with as much detail as possible:

* Incorrect or hallucinated diagnoses
* Remediation proposals that do not address the root cause
* Unexpected agent behavior or scope violations
* Performance issues during agent operations
