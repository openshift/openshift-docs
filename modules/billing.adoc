= Billing

[role="_abstract"]
{product-title-managed-short} subscriptions are available on the Amazon Web Services (AWS) marketplace, with hourly charges per secured core (vCPU) of nodes in secured clusters.

Customers can buy a {product-title-managed-short} subscription on the Amazon Web Services (AWS) marketplace.
The service cost is charged hourly per secured core, or vCPU of a node belonging to a secured cluster.

.Subscription cost example
====
If you have established a connection to two secured clusters, each with 5 same nodes with 8 vCPUs (such as Amazon EC2 m7g.2xlarge), the total number of secured cores is 80 (2 x 5 x 8 = 80).
