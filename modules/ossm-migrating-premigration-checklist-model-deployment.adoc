= Check your deployment model

Your deployment model determines your migration path.

.Procedure

. Run the following command to find your deployment model:
+
[source,terminal]
----
oc get smcp <smcp-name> -n <smcp-namespace> -o jsonpath='{.spec.mode}'
----
