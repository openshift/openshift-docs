= Autonomous agentic troubleshooting compared with user-initiated chat

[role="_abstract"]
Autonomous agentic troubleshooting differs from user-initiated {ols-long} chat in several ways.

.Comparison of autonomous agentic troubleshooting and user-initiated chat
[cols="1,1,1",options="header"]
|===
| Capability
| User-initiated {ols-short} chat
| Autonomous agentic troubleshooting

| Trigger
| A user types a question or request in the {ols-short} chat interface.
| The agent automatically responds to Prometheus alerts. No user input is required.

| Scope
| Responds to a single user query.
| Investigates, diagnoses, and proposes remediation for cluster-wide alerts.

| Execution
| Provides informational answers only.
| Can execute approved remediation actions in a sandboxed environment.

| Approval
| Not applicable.
| All remediation actions require explicit human approval through the Decision Queue.

| Continuity
| Session-based interaction.
| Operates continuously based on configured alert subscription policies.
|===
