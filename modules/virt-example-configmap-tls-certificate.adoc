= Example: ConfigMap created from a TLS certificate

The following example is of a ConfigMap created from `ca.pem` TLS certificate. 

[source,yaml]
----
apiVersion: v1
kind: ConfigMap
metadata:
  name: tls-certs
data:
  ca.pem: |
    -----BEGIN CERTIFICATE-----
    ... <base64 encoded cert> ...
    -----END CERTIFICATE-----
----
