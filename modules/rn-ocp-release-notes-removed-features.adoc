= Removed features

This section includes removed features for {product-title} {product-version}.

Deprecation and Removal of Dynamic Accelerator Slicer (DAS)::
+
The Dynamic Accelerator Slicer (DAS) Operator was introduced to allow dynamic GPU partitioning in {product-title} until the Dynamic Resource Allocation (DRA) partitionable device feature is available. With the DRA feature available as a technology preview feature in {product-title} {product-version}, the DAS Operator has been deprecated and removed.
+
For more information on DRA, see xref:../nodes/pods/nodes-pods-allocate-dra.adoc#nodes-pods-allocate-dra[Allocating GPUs to pods by using DRA].
