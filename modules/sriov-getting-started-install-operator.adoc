= Install the SR-IOV Network Operator

[role="_abstract"]
Install the SR-IOV Network Operator to manage SR-IOV network devices in your cluster.

.Procedure

. In the {product-title} web console, navigate to *Operators* -> *OperatorHub*.

. In the search field, enter `SR-IOV` to find the SR-IOV Network Operator.

. Click *SR-IOV Network Operator*, then click *Install*.

. On the Install Operator page, accept the default settings and click *Install*.

. Wait for the installation to complete. The operator status changes to *Succeeded* when installation is complete.

.Verification

. Verify the SR-IOV Network Operator is installed:
+
[source,terminal]
----
$ oc get csv -n openshift-sriov-network-operator
----
+
.Example output
[source,terminal]
----
NAME                                         DISPLAY                   VERSION   REPLACES   PHASE
sriov-network-operator.4.18.0-202403011234   SR-IOV Network Operator   4.18.0               Succeeded
----

. Verify the SR-IOV Network Operator pods are running:
+
[source,terminal]
----
$ oc get pods -n openshift-sriov-network-operator
----
+
.Example output
[source,terminal]
----
NAME                                      READY   STATUS    RESTARTS   AGE
network-resources-injector-xxxxx          1/1     Running   0          5m
operator-webhook-xxxxx                    1/1     Running   0          5m
sriov-network-config-daemon-xxxxx         1/1     Running   0          5m
sriov-network-operator-xxxxx              1/1     Running   0          5m
----

All pods should show `Running` status.
