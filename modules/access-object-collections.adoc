= Accessing deployment collections

[role="_abstract"]
Create, edit, clone, and delete deployment collections in {product-title} ({product-title-short}) by using the *Platform Configuration* -> *Collections* page.

.Procedure

* Click *Platform Configuration* -> *Collections*. The page displays a list of currently-configured collections. You can perform the following actions:

** Search for collections by entering text in the *Search by name* field, and then press *->*. 
** Click a collection name in the collection list to view the collection in read-only mode.
** Click the overflow menu {kebab} for an existing collection to edit, clone, or delete it.
+
[NOTE]
====
You cannot delete a collection that is actively used in {product-title-short}.
====
** Click *Create collection* to create a new deployment collection.
