[id="delete-kn-trigger_{context}"]
= Deleting a trigger using kn

.Procedure

* Delete a trigger:
+
[source,terminal]
----
$ kn trigger delete <trigger_name>
----

.Verification

. List existing triggers:
+
[source,terminal]
----
$ kn trigger list
----

. Verify that the trigger no longer exists:
+
.Example output
[source,terminal]
----
No triggers found.
----
