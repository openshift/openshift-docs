= {ols-long} version 1.0.5

[role="_abstract"]

{ols-official} 1.0.5 provides maintenance updates, security fixes, and stability improvements. This release ensures continued support and optimized performance for clusters running {ocp-product-title} 4.16 and later.

[id="ols-1-0-5-enhancements_{context}"]
== Enhancements

[role="_abstract"]

{ols-official} 1.0.5 is now generally available for {ocp-product-title} 4.16 and later. This release introduces the ability to attach CronJob logs to AI prompts, providing {ols-long} with the necessary context to troubleshoot automated job failures and irregular behaviors.

{ols-official} 1.0.5 provides the following enhancements:

* This release makes {ols-official} 1.0.5 generally available, and is supported on {ocp-product-title} 4.16 and later.

* With this release, you can attach `cron job` logs to the question you submit to {ols-official}, enabling the Service to provide more context-aware troubleshooting, diagnostics, and analysis of automated job outcomes, errors, or irregular behaviors.
