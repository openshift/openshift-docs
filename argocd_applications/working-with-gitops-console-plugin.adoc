= Working with the GitOps Console plugin
:context: working-with-gitops-console-plugin

toc::[]

[role="_abstract"]
:FeatureName: GitOps Console plugin
include::snippets/technology-preview.adoc[]

The GitOps Console plugin extends the {OCP} web console by adding GitOps resources. The plugin is available as part of the {gitops-title} Operator and provides a console UI for managing Argo CD and Argo Rollouts custom resources.

After you install the {gitops-title} Operator and enable the GitOps Console plugin, the {product-title} web console displays a *GitOps* navigation tab in the *Administrator* perspective. The GitOps navigation tab replaces the previous *Environments* tab and related pages in the *Developer* perspective.

The GitOps navigation tab provides access to the following Argo CD and Argo Rollouts resources:

* Applications
* ApplicationSets
* AppProjects
* Rollouts

[id="prerequisites_{context}"]
== Prerequisites

* You have access to {OCP} 4.19 or later.
* You have installed the {gitops-title} Operator.
