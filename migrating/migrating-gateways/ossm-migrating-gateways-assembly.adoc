= Migrating gateways from Service Mesh 2 to Service Mesh 3
include::_attributes/common-attributes.adoc[]
:context: ossm-migrating-gateways-assembly

toc::[]

This section is if you have been using gateways with {SMProductName} {SMv2Version} and are migrating them to {smproduct} 3.0. Migrating gateways between {SMProduct} 2 and {SMProduct} 3 is very similar to migrating regular workloads.

There are two scenarios for migrating gateways from {SMProductName} 2 to {SMProduct} 3:

* Gateway canary migration
* Gateway in place migration

Follow the procedure for the scenario that best fits your environment.
