= Observing network traffic flows
include::_attributes/common-attributes.adoc[]
:context: observing-flows

[role="_abstract"]
The Network Observability Operator provides a comprehensive suite of visualization tools to monitor, troubleshoot, and audit cluster network traffic. By capturing data at the eBPF layer and correlating it with {product-title} infrastructure, the Operator enables you to analyze traffic patterns across pods, services, and virtualized workloads.
