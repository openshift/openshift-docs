The {SMPlugin} extends the {ocp-short-name} web console with a {SMProductShortName} menu and enhanced tabs for workloads and services.

[IMPORTANT]
====
If your browser does not trust the certificate, you can manually accept it in the Kiali standalone UI before accessing the {SMPluginShort}.

== Installing {SMPlugin-s}

You can install the {SMPluginShort} by using the {ocp-short-name} web console or the {ocp-short-name} CLI.

[NOTE]
====
{SMPluginShort} is only supported on {ocp-short-name} 4.15 and above. For {ocp-short-name} 4.14 users, only the standalone Kiali console is accessible.
====

== Uninstalling {SMPlugin-s}

You can uninstall the {SMPluginShort} by using the {ocp-short-name} web console or the {ocp-short-name} CLI.

[NOTE]
====
You must uninstall the OSSMC plugin before removing the Kiali Operator. Deleting the Operator first may leave OSSMC and Kiali CRs stuck, requiring manual removal of the finalizer. Use the following command with `<CR type>` as `kiali` or `ossmconsole` to remove the finalizer, if needed:
[source,terminal]
----
$ oc patch <CR type> <CR name> -n <CR namespace> -p '{"metadata":{"finalizers": []}}' --type=merge
----
====
