= Monitor build health with metrics

include::_attributes/common-attributes.adoc[]
:context: monitoring

toc::[]

[role="_abstract"]
{builds-shortname} exposes several metrics for cluster administrators to monitor the performance and functioning of your build resources. The build controller metrics are exposed on port `8383`.

[IMPORTANT]
====
These metrics are experimental. Metrics support in {builds-shortname} is in active development, and the existing labels and metrics can change or be removed in future versions. Red{nbsp}Hat does not provide support for these metrics.
====
