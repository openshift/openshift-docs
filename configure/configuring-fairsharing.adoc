= Configuring fair sharing
:context: configuring-fairsharing

toc::[]

Fair sharing uses preemption to achieve an equal or weighted share of the borrowable resources between the tenants of a cohort.

You can configure fair sharing by setting the `spec.preemptionPolicy` value in the `Kueue` custom resource (CR) to `FairSharing`.
