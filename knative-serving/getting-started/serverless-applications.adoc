You can create a serverless application by using one of the following methods:

* Create a Knative service from the {ocp-product-title} web console.
+
For {ocp-product-title}, see link:https://docs.openshift.com/container-platform/latest/applications/creating_applications/odc-creating-applications-using-developer-perspective.html#odc-creating-applications-using-developer-perspective[Creating applications] for more information.

* Create a Knative service by using the Knative (`kn`) CLI.
* Create and apply a Knative `Service` object as a YAML file, by using the `oc` CLI.
