= Using persistent storage
include::_attributes/attributes-microshift.adoc[]
:context: using-persistent-storage-microshift

toc::[]

[role="_abstract"]
Managing storage is a distinct problem from managing compute resources. {microshift-short} uses the Kubernetes persistent volume (PV) framework so that node administrators can provision persistent storage for a node. Developers can use persistent volume claims (PVCs) to request PV resources without having specific knowledge of the underlying storage infrastructure.
