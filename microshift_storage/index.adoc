= Storage overview
include::_attributes/attributes-microshift.adoc[]
:context: storage-overview-microshift

[role="_abstract"]
{microshift-short} supports multiple types of storage, both for on-premise and cloud providers. You can manage container storage for persistent and non-persistent data in a {microshift-short} node. {microshift-short} storage is broadly classified into two categories, namely ephemeral storage and persistent storage.
