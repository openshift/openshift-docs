= Exploring the CVO

To see the current version that your cluster is on, type:

----
$ oc get clusterversion

NAME    VERSION   AVAILABLE PROGRESSING SINCE STATUS
version 4.3.0-0.0 True      False       10h   Cluster version is 4.3.0-0.0
----
