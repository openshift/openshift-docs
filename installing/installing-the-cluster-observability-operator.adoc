= Installing the {coo-short}
include::_attributes/common-attributes.adoc[]
:context: installing-the-cluster-observability-operator

toc::[]

[role="_abstract"]
The {coo-full} provides log visualization in the {ocp-product-title} web console. Install the {coo-short} if you want to view and query logs under *Observe -> Logs* in the web console.

[NOTE]
====
The {coo-short} is optional. You can collect, store, and forward logs without it by using only the {loki-op} and {clo}. Install the {coo-short} only if you need the web console visualization feature.
====
