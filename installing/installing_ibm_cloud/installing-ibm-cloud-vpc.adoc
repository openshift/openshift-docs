= Installing a cluster on {ibm-cloud-title} into an existing VPC
:context: installing-ibm-cloud-vpc

toc::[]

[role="_abstract"]
In {product-title} version {product-version}, you can install a cluster into an existing Virtual Private Cloud (VPC) on {ibm-cloud-name}. The installation program provisions the rest of the required infrastructure, which you can then further customize. To customize the installation, you change parameters in the `install-config.yaml` file before you install the cluster.
