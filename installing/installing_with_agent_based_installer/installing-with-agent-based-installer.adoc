[role="_abstract"]
Install an {product-title} cluster with the Agent-based Installer, a lightweight installation method that automates cluster provisioning and allows configuration customizations in a disconnected environment.

[id="prerequisites_{context}"]
== Prerequisites

* You reviewed details about {product-title} installation and update processes.
* You read the documentation on selecting a cluster installation method and preparing it for users.
* If you use a firewall or proxy, you configured it to allow the sites that your cluster requires access to.
