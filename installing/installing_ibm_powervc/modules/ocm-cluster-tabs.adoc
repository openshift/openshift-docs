= Cluster tabs

[role="_abstract"]
Selecting an active, installed cluster shows tabs associated with that cluster. The following tabs display after the cluster's installation completes:

* Overview
* Access control
* Add-ons
* Cluster history
* Networking
* Machine pools
* Support
* Settings
