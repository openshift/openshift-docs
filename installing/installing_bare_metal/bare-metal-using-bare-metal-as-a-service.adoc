= Using bare metal as a service
include::_attributes/common-attributes.adoc[]
:context: bare-metal-using-bmaas

toc::[]

{product-title}'s Bare Metal as a Service (BMaaS) feature enables you to provision and manage bare-metal nodes using the Metal^3^ API. These nodes, external to the {product-title} cluster, can host workloads requiring direct hardware access, such as legacy applications or high-performance computing tasks. BMaaS provides the following capabilities:

* Provisioning of bare-metal nodes, including configuration and preparation for workloads.
* Lifecycle management, such as power management, firmware updates, and decommissioning.

As standalone systems, these nodes operate independently of the {product-title} cluster and support diverse workloads by integrating bare-metal resources with containerized and virtualized applications. BMaaS supports {op-system-base-full}, {op-system-first}, and Microsoft Windows on the bare-metal nodes.
