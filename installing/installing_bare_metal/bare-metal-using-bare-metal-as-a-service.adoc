= Using bare metal as a service
include::_attributes/common-attributes.adoc[]
:context: bare-metal-using-bmaas

toc::[]

{product-title}'s Bare Metal as a Service (BMaaS) feature enables you to provision and manage bare-metal hosts by using the Metal^3^ API. These hosts, external to the {product-title} cluster, can run workloads that might not be suitable for containerisation or virtualisation, such as applications requiring direct hardware access, high-performance computing tasks or legacy applications. BMaaS provides the following capabilities:

* Provisioning of bare-metal hosts, including initial configuration.
* Lifecycle management, such as power management, firmware updates, and decommissioning.

As standalone systems, these hosts operate independently of the {product-title} cluster and support diverse workloads by integrating bare-metal resources with containerized and virtualized applications. BMaaS supports {op-system-base-full}, and CentOS Stream on the bare-metal hosts.
