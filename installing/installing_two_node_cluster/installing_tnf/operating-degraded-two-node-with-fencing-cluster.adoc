= Operating a degraded two-node with fencing cluster
:context: degraded-tnf

[role="_abstract"]
A Two-Node with Fencing (TNF) cluster enters a `degraded` state when one of its two control-plane nodes becomes unavailable. The remaining node continues to host the active control plane; however, the cluster loses its high-availability (HA) redundancy until the failed node recovers.
