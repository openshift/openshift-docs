:context: installation-config-parameters-vsphere
:platform: vSphere

toc::[]

[role="_abstract"]
Before you deploy an {product-title} cluster on vSphere, you can customize parameters in the `install-config.yaml` file to configure your cluster and the platform that hosts it.

When you create the `install-config.yaml` file, you can configure the values for your required parameters through the command line. Edit the `install-config.yaml` file to customize your cluster further before installation begins.
