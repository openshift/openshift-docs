= Installing a three-node cluster on vSphere
include::_attributes/common-attributes.adoc[]
:context: installing-vsphere-three-node

toc::[]

[role="_abstract"]
In {product-title} version {product-version}, you can install a three-node cluster on VMware vSphere. A three-node cluster consists of three control plane machines, which also act as compute machines, providing a smaller, more resource-efficient cluster.

You can install a three-node cluster by using either installer-provisioned or user-provisioned infrastructure.
