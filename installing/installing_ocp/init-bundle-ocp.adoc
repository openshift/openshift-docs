= Generating and applying an init bundle for RHACS on {osp}
include::modules/common-attributes.adoc[]
:context: init-bundle-ocp
:topic-helm:
:topic-operator:

toc::[]

[role="_abstract"]

Before you install the `SecuredCluster` resource on a cluster, you must create an init bundle. The cluster that has `SecuredCluster` installed and configured then uses this bundle to authenticate with Central. You can create an init bundle by using either the {product-title-short} portal or the `roxctl` CLI. You then apply the init bundle by using it to create resources.
