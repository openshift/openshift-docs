:context: verify-installation-rhacs-ocp
:toc: macro

toc::[]

[role="_abstract"]
You can confirm the operational status of Red{nbsp}Hat Advanced Cluster Security for Kubernetes (RHACS) by deploying sample applications that contain known vulnerabilities. You can run these workloads to test the build and deploy-time assessment features and view the resulting policy violations in the RHACS portal.
