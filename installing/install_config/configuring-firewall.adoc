= Configuring your firewall

include::_attributes/common-attributes.adoc[]
:context: configuring-firewall

toc::[]

[role="_abstract"]
If you use a firewall, you must configure your allowlist for the firewall to ensure {product-title} has access to the URLs it requires to pull container images and access Red Hat services. Additional URLs are required for features such as Telemetry, {red-hat-lightspeed}, cloud provider integrations, or certain build strategies.
