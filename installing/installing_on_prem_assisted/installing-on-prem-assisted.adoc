= Installing an on-premise cluster using the {ai-full}

toc::[]

The {ai-full} for {product-title} simplifies the deployment of OpenShift clusters. By providing a user-friendly wizard on the link:https://console.redhat.com/[{hybrid-console}], the {ai-full} reduces installation complexity, making it easier for users to set up and configure clusters with minimal manual intervention.
