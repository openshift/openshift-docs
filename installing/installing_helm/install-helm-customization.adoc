= Installing with customizations by using Helm charts
include::modules/common-attributes.adoc[]
:context: acs-install-helm-customization
:topic-helm:
:toc: macro
:toclevels: 3

toc::[]

[role="_abstract"]
You can install Red{nbsp}Hat Advanced Cluster Security for Kubernetes (RHACS) by configuring and deploying Helm charts to customize the installation for your specific environment. To fully secure your infrastructure, you must install the `central-services` chart to deploy Central and Scanner, generate an initialization bundle, and then install the `secured-cluster-services` chart on each cluster. This workflow ensures you correctly register your clusters and isolate sensitive credentials from standard settings.
