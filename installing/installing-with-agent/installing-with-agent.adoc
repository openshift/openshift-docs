== What is Agent installation?
Agent installation is based on the technology of Assisted Installer, that is link:https://console.redhat.com[console.redhat.com] and the Infrastructure Operator.
The installation also utilizes the OpenShift installer that is the input asset management. You can install a disconnected {product-title} cluster through the `openshift-install agent` subcommand for:

* Single-node {product-title} cluster (SNO): A node that is both a master and worker.
* Three-node {product-title} cluster :A compact cluster that has three master nodes that are also worker nodes.
* Highly available {product-title} cluster (HA): Three master nodes with any number of worker nodes.
