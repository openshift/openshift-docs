= Installing a cluster quickly on {rh-virtualization}
include::modules/common-attributes.adoc[]
:context: installing-rhv-default

toc::[]

In {product-title} version {product-version}, you can quickly install a default, non-customized, cluster on {rh-virtualization-first} using “installer-provisioned infrastructure.”

To install a default cluster, you prepare the environment, run the installer and answer its prompts. Then, the installer creates the {product-title} cluster.

To learn more, see
xref:../../architecture/architecture-installation.adoc#architecture-installation[this overview of the installation and update processes].
