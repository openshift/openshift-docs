In {product-title} version {product-version}, you can install a cluster
in a {rh-virtualization-first} environment.

The installer that deploys a cluster on {rh-virtualization} automates the process by using installer-provisioned infrastructure. This installer is available for Linux and macOS only.

To learn more about installing a {product-title} cluster on {rh-virtualization}, read link:https://access.redhat.com/articles/4903411[Quickstart Guide: Installing {product-title} on Red Hat Virtualization]
