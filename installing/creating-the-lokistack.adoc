= Creating the LokiStack

:context: creating-the-lokistack

toc::[]

[role="_abstract"]
Create a `LokiStack` custom resource (CR) to deploy the log store for application, audit, and infrastructure-related logs.

== Prerequisites

* You have installed the {loki-op}.
* You have chosen an initial deployment size based on your cluster size. You will adjust the size after installation based on observed log volume.
* You have configured object storage for LokiStack. See xref:../installing/configuring-storage-for-lokistack.adoc#configuring-storage-for-lokistack[Configuring object storage for LokiStack].
