= Verifying installation of RHACS on other platforms
:context: verify-installation-rhacs
:toc: macro
:toclevels: 3

toc::[]

[role="_abstract"]
You can validate the build and deploy-time assessment features of Red{nbsp}Hat Advanced Cluster Security for Kubernetes (RHACS) by running sample applications with known vulnerabilities. You can then access the RHACS portal to view the resulting security assessments and confirm that policy violations are detected.
