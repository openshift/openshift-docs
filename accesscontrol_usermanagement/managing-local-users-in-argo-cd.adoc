= Managing local users in Argo CD
:context: managing-local-users-in-argo-cd

toc::[]

The Argo CD Operator provides built-in support for managing local users with automatic API token generation and renewal. As an administrator, you can declaratively define local users in the Argo CD custom resource (CR), and the {gitops-title} Operator manages their API tokens throughout their lifecycle.
