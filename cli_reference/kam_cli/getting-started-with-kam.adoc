With the `kam` CLI tool, you can generate a functional GitOps setup  including your application.

.Prerequisites

* OpenShift GitOps Operator is installed in your cluster.
* Sealed Secrets Operator is installed in your cluster. Sealed Secrets Operator is necessary for `kam` to generate encrypted or and sealed secrets.
* link:https://github.com/rhd-gitops-example/taxi[A sample application]
* Credentials for your image repository user account.
* `kam` is installed in your cluster.
* A GitHub or GitLab access token.
* An SSH key connected to your GitHub or GitLab account.
