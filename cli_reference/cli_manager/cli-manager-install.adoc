//Brief explanation of what the cli manager is.
The {cli-manager} can be used in both connected and disconnected environments, and particularly makes installing and managing CLI plugins in disconnected environments easier. It is compatible with Krew and cluster administrators add CLI plugin custom resources which can then be accessed in both connected and disconnected environments.
