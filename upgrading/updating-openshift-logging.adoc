= Updating {LoggingProductName}

:context: updating-logging
toc::[]

You can update {logging-title-uc} by modifying the Operator subscription update channel for the installed Operators.

Considerations before updating to a minor version:

* If you are on a EUS release of {logging-title-uc}, it is recommended to update to an EUS release. For information about the support cycle for releases, see link:https://access.redhat.com/support/policy/updates/openshift_operators?extIdCarryOver=true&sc_cid=7013a000003StsiAAC#platform-agnostic[OpenShift Operator Life Cycles].

* You only need to update the Operators for {logging-title-uc} that you have installed.

** If you have only installed the {clo}, all you need to do is update the {clo}. For more information see: xref:../modules/updating-the-clo.adoc#updating-the-clo_updating-logging[Updating the {clo}]

** If you have installed both the {clo} and the {loki-op}, follow these procedures:
+
[IMPORTANT]
====
You must update the {clo} and the {loki-op} to the same major and minor version. 
====
