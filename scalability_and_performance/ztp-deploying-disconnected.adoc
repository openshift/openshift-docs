= Deploying distributed units at scale in a disconnected environment
include::modules/common-attributes.adoc[]
:context: ztp-deploying-disconnected

toc::[]

Use zero touch provisioning (ZTP) to provision distributed units at new edge sites in a
disconnected environment.
The workflow starts when the site is connected to the network and ends with the CNF workload
deployed and running on the site nodes.
