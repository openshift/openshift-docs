= Topology Aware Lifecycle Operator
include::modules/common-attributes.adoc[]
:context: cnf-topology-aware-lifecycle-operator

toc::[]

You can use the Topology Aware Lifecycle Operator to manage the software lifecycle of multiple single node Openshift (SNO) clusters. The Operator uses {rh-rhacm-first} policies to perform changes on the target clusters.

== Additional resources

The Topology Aware Lifecycle Operator is supported in a restricted network environment. For more information, see xref:../operators/admin/olm-restricted-networks.adoc#olm-restricted-networks[Using Operator Lifecycle Manager on restricted networks].

The Topology Aware Lifecycle Operator can use `PolicyGenTemplate` CRs for creating policies. For more information, see xref:../scalability_and_performance/ztp-deploying-disconnected.adoc#ztp-the-policygentemplate_ztp-deploying-disconnected[The PolicyGenTemplate].
