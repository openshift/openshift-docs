[role="_abstract"]
Updating {SMProductName} in ambient mode follows the same pattern as sidecar mode. You can check the specific {istio} update in the "Updating OpenShift Service Mesh" documentation. In addition to the control plane, update other resources, such as waypoint proxies, to ensure a smooth transition between control plane revisions. After updating, verify that Layer 7 (L7) features continue to function correctly.
