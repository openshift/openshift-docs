= Configuring metrics for the monitoring stack
:context: otel-configuring-metrics-for-monitoring-stack

toc::[]

[role="_abstract"]
You can use the `OpenTelemetryCollector` custom resource (CR) to configure the Prometheus Receiver. By using this CR, you can scrape metrics from the in-cluster monitoring stack. You can also configure a Prometheus `ServiceMonitor` CR for scraping the Collector pipeline metrics and enabled Prometheus Exporter.
