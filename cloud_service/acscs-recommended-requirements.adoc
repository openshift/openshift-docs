= Recommended resource requirements for Red Hat Advanced Cluster Security Cloud Service
include::modules/common-attributes.adoc[]
:context: acscs-recommended-requirements

toc::[]

[role="_abstract"]
A focused test developed the recommended resource guidelines by creating the following objects across a given number of namespaces:

* 10 deployments, with 3 pod replicas in a sleep state, mounting 4 secrets, 4 config maps
* 10 services, each one pointing to the TCP/8080 and TCP/8443 ports of one of the previous deployments
* 1 route pointing to the first of the previous services
* 10 secrets containing 2048 random string characters
* 10 config maps containing 2048 random string characters

During the analysis of results, the analysis identified the number of deployments as a primary factor for increasing used resources. The estimation of required resources uses the number of deployments.
