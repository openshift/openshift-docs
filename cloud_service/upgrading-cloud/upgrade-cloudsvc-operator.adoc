include::modules/common-attributes.adoc[]
:context: upgrade-cloudsvc-operator

toc::[]

[role="_abstract"]

Red{nbsp}Hat provides regular service updates for the components that it manages, including Central services. These service updates include upgrades to new versions of {product-title-managed}.

You must regularly upgrade the version of {product-title-short} on your secured clusters to ensure compatibility with {product-title-managed-short}.
