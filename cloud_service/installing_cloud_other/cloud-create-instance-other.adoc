= Creating an RHACS Cloud Service instance for Kubernetes clusters
include::modules/common-attributes.adoc[]
:context: cloud-create-instance-other

toc::[]

[role="_abstract"]
Access {rh-rhacscs-first} by selecting an instance in the {cloud-console}. An *ACS instance* has the {product-title-managed-short} management interface and services that Red{nbsp}Hat configures and manages for you. The management interface connects to your secured clusters, which have the services that scan and collect information about vulnerabilities. One instance can connect to and monitor many clusters.
