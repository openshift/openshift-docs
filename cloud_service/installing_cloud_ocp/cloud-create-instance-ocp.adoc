:context: cloud-create-instance-ocp

toc::[]

[role="_abstract"]
Access {rh-rhacscs-first} by selecting an instance in the {cloud-console}. An _ACS instance_ contains the {product-title-managed-short} management interface and services that Red{nbsp}Hat configures and manages for you. The management interface connects to your secured clusters, which contain the services that scan and collect information about vulnerabilities. One instance can connect to and monitor many clusters.
