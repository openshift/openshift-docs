[role="_abstract"]

Installing {ocp-short-name} {SMProductShortName} consists of three main tasks: installing the {ocp-short-name} Operator, deploying {istio}, and customizing the {istio} configuration. Then, you can also install the sample `bookinfo` application to push data through the mesh and explore mesh functionality.
