= Enabling TLS profiles for Kubelet
include::modules/common-attributes.adoc[]

toc::[]

You can configure a TLS profile for Kubelet when it is acting as a server to ensure that system pods and application pods use cryptographic libraries that do not allow known insecure protocols, ciphers, or algorithms. A TLS profile is a list of ciphers and the TLS version that can be used to ake a TLS connection.
