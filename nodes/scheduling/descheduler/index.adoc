= Descheduler overview
include::_attributes/common-attributes.adoc[]
:context: nodes-descheduler-about

toc::[]

[role="_abstract"]
The descheduler evicts running pods so that the default scheduler can place them on more suitable nodes. The default scheduler chooses where new pods run; the descheduler improves placement over time.
