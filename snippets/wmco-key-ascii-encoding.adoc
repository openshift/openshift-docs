If you created the key pair on a {op-system-base-full} system, if you copy the public key to a Windows system, make sure the public key is saved using ASCII encoding. For example, the following command copies a public key, encoding it for the ASCII character set:

[source,terminal]
----
$ echo "ssh-rsa <ssh_pub_key>" | Out-File <ssh-key-path> -Encoding ascii
----

where:

`<ssh_pub_key>`:: Specifies the SSH public key used to access the cluster.
`<ssh-key-path>`:: Specifies the path to the SSH public key.
