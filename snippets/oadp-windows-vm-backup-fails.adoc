[WARNING]
====
If you run a backup of a Windows VM immediately after the VM reboots, the backup might fail with `WaitingForPluginOperationsPartiallyFailed` error. This is because, in Windows,  there is a hard-coded 10 seconds `filesystem` freeze when a VM boots. Immediately after a VM boots, the Windows Volume Shadow Copy Service (VSS) and Guest Agent (GA) service are not ready. The `filesystem` freeze, and the VSS and GA service not being ready, cause the backup to fail. In such a case, re-try the backup a few minutes after the VM boots.
====
