[NOTE]
====
When you restore a backup of VMs by using the `kubevirt` plugin, it is observed that the Ceph storage capacity allocated for the restore is higher than expected. This behavior is observed only during the `kubevirt` restore and if the volume type of the VM is `block`.
====
