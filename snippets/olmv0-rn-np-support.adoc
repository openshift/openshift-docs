:_mod-docs-content-type: SNIPPET

In {product-title} {product-version}.{olm-np-z-stream}, {olmv0-first} supports the inclusion of network policy manifests in the resource bundles of Operators. Tailored network policies protect against data leaks and harden against many attack vectors on {product-title} clusters.

[IMPORTANT]
====
In order to update or install an Operator that includes network policy manifests, you must first update to {product-title} {product-version}.{olm-np-z} or later.
====

For more information, see link:https://access.redhat.com/articles/7133113[Operators shipping with network policies may require OCP cluster upgrade before they can be upgraded (Red Hat Knowledgebase)].
