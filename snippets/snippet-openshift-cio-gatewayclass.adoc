[IMPORTANT]
====
When using the Gateway API custom resource definitions (CRDs) provided in {ocp} 4.19 or newer, you must create a `GatewayClass` named `openshift-default` and specify a `controllerName` of `openshift.io/gateway-controller/v1`. For more details, see the link:https://docs.redhat.com/en/documentation/openshift_container_platform/4.22/html/ingress_and_load_balancing/configuring-ingress-cluster-traffic#nw-ingress-gateway-api-enable_ingress-gateway-api[Getting started with Gateway API for the Ingress Operator (OpenShift Container Platform documentation)].
====
