CPU Manager manages groups of CPUs and constrains workloads to specific CPUs.

CPU Manager is useful for workloads that have any of the following attributes:

* The workload requires as much CPU time as possible.
* The workload is sensitive to processor cache misses.
* The workload is a low-latency network application.
* The workload must coordinate with other processes, and benefits from sharing a single processor cache.
