= Configure SMO integration for O-RAN deployments
include::_attributes/common-attributes.adoc[]
:context: ocloud-configuring-smo-integration

toc::[]

In O-RAN deployments, a Service Management and Orchestration (SMO) system manages telco infrastructure through standardized REST APIs defined by the O-RAN Alliance O2 IMS specification. Connecting the O-Cloud Manager to an SMO enables automated, standards-compliant infrastructure management. If you are not using an SMO, skip this section and proceed directly to cluster provisioning.
