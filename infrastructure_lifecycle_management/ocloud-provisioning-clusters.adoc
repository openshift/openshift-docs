= Provision clusters with the O-Cloud Manager
include::_attributes/common-attributes.adoc[]
:context: ocloud-provisioning-clusters

toc::[]

[role="_abstract"]
After you define cluster templates and onboard bare-metal servers, you can provision clusters by submitting a `ProvisioningRequest` resource with site-specific parameters. The O-Cloud Manager allocates hardware, installs the cluster, and applies Day-2 configuration automatically.
