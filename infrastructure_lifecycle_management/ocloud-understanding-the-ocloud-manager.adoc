= Infrastructure and cluster lifecycle management with the O-Cloud Manager
include::_attributes/common-attributes.adoc[]
:context: ocloud-understanding-the-ocloud-manager

toc::[]

[role="_abstract"]
The O-Cloud Manager is an Operator that runs on a hub cluster to provide template-based lifecycle management for {product-title} clusters. It automates cluster provisioning, hardware management, Day-2 configuration, scaling, and upgrades through declarative custom resources built on {rh-rhacm-first} and the SiteConfig Operator.
