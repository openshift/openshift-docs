= Define cluster templates and onboard bare-metal servers
include::_attributes/common-attributes.adoc[]
:context: ocloud-defining-templates-and-onboarding-servers

toc::[]

[role="_abstract"]
Before you can provision clusters with the O-Cloud Manager, you must define reusable cluster blueprints and register bare-metal servers in resource pools. These resources establish the hardware profiles, default configurations, and allocation criteria that the O-Cloud Manager uses when processing a `ProvisioningRequest`.
