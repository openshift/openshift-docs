= {js-operator} overview

:context: js-about

toc::[]

The {js-operator} is designed to manage groups of Kubernetes Jobs as a single unit. It provides a unified API for deploying High-Performance Computing (HPC) and AI/ML training workloads on Kubernetes. {js-operator} simplifies the management of distributed training workloads by grouping {product-title} jobs.

Its key features include support for multi-template jobs, automatic headless service configuration for stable networking, and configurable failure and success policies for efficient resource utilization. Additionally, JobSet offers exclusive placement per topology domain, fast failure recovery, startup sequencing, and integration with Kueue for enhanced cluster management.
