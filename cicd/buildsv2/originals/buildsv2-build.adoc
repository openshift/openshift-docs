== Overview

Use a `Build` resource to define the various aspects of your build, including:

* Source files
* Remote sources
* Build strategy
* Parameter values
* Builder or Dockerfile
* Output location
* Environment variables
* Retention parameters

A `Build` is available within a namespace.
