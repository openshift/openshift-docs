= Running workloads and buildah as user namespaces on {pipelines-title}
include::_attributes/common-attributes.adoc[]
:context: running-workloads-and-buildah-as-user-namespaces-on-openshift-pipelines

toc::[]

Running {pipelines-shortname} as root user on a container can expose the host and container processes to other resources. You can reduce this type of exposure by using one of the following methods:

* You can run workloads in a user namespace.
* You can run the workload as a root user on the container and make it secure on the host.
