When you installed {ProductName} you created a Kiali Custom Resource Definition (CRD).  When you create a new Kiali instance, it creates a Custom Resource (CR) from the CRD.  This section describes configuration that is performed in the CRD or in the CR.

.Prerequisites

* Access to an OpenShift Container Platform cluster with cluster-admin user privileges.
* Kiali operator has been installed on the cluster.
