== Troubleshooting the control plane

Deploying the `ServiceMeshControlPlane` also creates the other components that make up {ProductName} as described in the xref:../../service_mesh/v2x/ossm-architecture.adoc#ossm-architecture_ossm-architecture[architecture] topic.
