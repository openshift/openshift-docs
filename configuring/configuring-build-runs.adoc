= Configuring build runs

include::_attributes/common-attributes.adoc[]
:context: configuring-build-runs

toc::[]

[role="_abstract"]
As a platform engineer, configure build runs by defining the build reference, parameters, service account, output settings, retention rules, and volumes in a `BuildRun` custom resource (CR). Use a `BuildRun` CR to manage how individual builds execute within a namespace on the cluster.
