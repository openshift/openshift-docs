= Configuring build strategies
include::_attributes/common-attributes.adoc[]
:context: configuring-build-strategies

toc::[]


A `BuildStrategy` or `ClusterBuildStrategy` custom resource (CR) helps you to define strategy parameters, system parameters, step resources definitions, annotations, and volumes to configure a build strategy. A `BuildStrategy` resource is available for use within a namespace, and a `ClusterBuildStrategy` resource is available for use throughout the cluster.

To configure a build strategy, create a `BuildStrategy` or `ClusterBuildStrategy` resource YAML file and apply it to the {ocp-product-title} cluster.
