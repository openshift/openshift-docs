include::_attributes/common-attributes.adoc[]
:context: configuring-openshift-builds

[role="_abstract"]
Platform engineers can define source code location, build strategy, parameters, output settings, retention rules, and volumes in a `Build` custom resource (CR). A `Build` CR enables consistent build pod configuration and provides a namespace-scoped method for managing build execution on the cluster.
