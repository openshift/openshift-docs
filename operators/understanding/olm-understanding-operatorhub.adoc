[NOTE]
====
The `marketplace` Operator populates OperatorHub with a pre-defined xref:../../operators/understanding/olm-rh-catalogs.adoc#olm-rh-catalogs_olm-rh-catalogs[set of Operator catalogs]. If you disable the `marketplace` Operator during installation, OperatorHub is not populated with these catalogs. You can xref:../../operators/admin/olm-managing-custom-catalogs.adoc#olm-creating-catalog-from-index_olm-managing-custom-catalogs[add a custom catalog source to a cluster] if you want to install Operators after disabling the `marketplace` Operator. {product-title} does not automatically update manually created catalog sources.
====
