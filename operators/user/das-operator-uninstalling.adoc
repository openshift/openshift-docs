include::_attributes/common-attributes.adoc[]
:context: das-operator-uninstalling

toc::[]

[role="_abstract"]
You can uninstall the Dynamic Accelerator Slicer (DAS) Operator from your {product-title} cluster by removing the operator subscription and cleaning up the associated resources.

[discrete]
== Prerequisites

* You have access to an {product-title} cluster using an account with `cluster-admin` permissions.
* The DAS Operator is installed in your cluster.
