Cluster administrators can install Operators to an {product-title} cluster by subscribing Operators to namespaces with OperatorHub.

ifdef::openshift-origin[]
[id="olm-adding-operators-to-a-cluster-prereqs"]
== Prerequisites

* Ensure that you have downloaded the link:https://console.redhat.com/openshift/install/pull-secret[pull secret from Red Hat OpenShift Cluster Manager] as shown in _Obtaining the installation program_ in the installation documentation for your platform.
+
If you have the pull secret, add the `redhat-operators` catalog to the `OperatorHub` custom resource (CR) as shown in _Configuring {product-title} to use Red Hat Operators_.
endif::[]
