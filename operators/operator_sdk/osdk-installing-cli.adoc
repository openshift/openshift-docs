= Installing the Operator SDK CLI
include::modules/common-attributes.adoc[]
:context: osdk-installing-cli

toc::[]

The Operator SDK provides a command-line interface (CLI) tool that Operator developers can use to build, test, and deploy an Operator. You can install the Operator SDK CLI on your workstation so that you are prepared to start authoring your own Operators.

{product-title} 4.6 supports Operator SDK v0.19.4, which can be installed from upstream sources.

[NOTE]
====
Starting in {product-title} 4.7, the Operator SDK is fully supported and available from official Red Hat product sources. See link:https://docs.openshift.com/container-platform/4.7/release_notes/ocp-4-7-release-notes.html#ocp-4-7-osdk-supported[{product-title} 4.7 release notes] for more information.
====
