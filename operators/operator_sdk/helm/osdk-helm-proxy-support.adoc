= Proxy support for Helm-based Operators
include::modules/common-attributes.adoc[]
:context: osdk-helm-proxy-support

toc::[]

Administrators configure Operator support for network proxies by setting the environment variable to `HTTP_PROXY`, `HTTPS_PROXY`, or `NO_PROXY` in the Operator deployment. The Operator inspects the environment and passes the variables along to the required Operands.
