= Cluster autoscaling for {product-title}
include::_attributes/attributes-openshift-dedicated.adoc[]
:context: osd-cluster-autoscaling

toc::[]

[role="_abstract"]
Apply autoscaling to {product-title} clusters to automatically adjust the number of worker nodes based on workload demands. This optimizes resource utilization and reduces costs by scaling up when demand increases and scaling down when resources are underutilized.

[IMPORTANT]
====
You can configure the cluster autoscaler only in clusters where the machine API is operational.

Only one cluster autoscaler can be created per cluster.
====
