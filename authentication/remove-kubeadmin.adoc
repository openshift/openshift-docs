= Removing the kubeadmin user
include::_attributes/common-attributes.adoc[]
:context: removing-kubeadmin

toc::[]

[role="_abstract"]
After installation, {product-title} creates a cluster administrator user called `kubeadmin` with the `cluster-admin` role. To improve cluster security, you can remove this user after configuring an identity provider and creating a new `cluster-admin` user.
