= Managing {builds-shortname} in a network-restricted environment
:context: using-builds

toc::[]

[role="_abstract"]
Configure {ocp-product-title} with an HTTP or HTTPS proxy to deny direct internet access. This configuration enforces organizational security policies by ensuring that the build processes securely pull external dependencies, base images, and source code from repositories through a monitored and controlled egress gateway.
