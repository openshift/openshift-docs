= Managing {builds-shortname} in a network-restricted environment
:context: using-builds

toc::[]

[role="_abstract"]
Configure {ocp-product-title} with an HTTP or HTTPS proxy to enforce security policies and prevent direct internet access for build processes. This setup enforces security policies by routing build pulls of dependencies, images, and code through a monitored outgoing network proxy gateway.
