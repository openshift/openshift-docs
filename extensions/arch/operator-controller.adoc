= Operator Controller
include::_attributes/common-attributes.adoc[]
:context: operator-controller

toc::[]

[role="_abstract"]
Operator Controller is the central component of {olmv1-first}. It extends Kubernetes with an API through which users can install Operators and extensions.
ifndef::openshift-rosa-hcp[]
As a core on-cluster component, the Operator Controller directly consumes the other {olmv1} component, catalogd, to resolve and deploy extensions.
endif::[]

ifdef::openshift-rosa-hcp[]
On {product-title} clusters, the Operator Controller executes within a secure, dedicated Red{nbsp}Hat-managed infrastructure layer off-cluster. It securely consumes metadata from the catalog service over an internal Red{nbsp}Hat-managed API tunnel. This means you cannot see operator-controller workloads running on your worker nodes.
endif::[]
