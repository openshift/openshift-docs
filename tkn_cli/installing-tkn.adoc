= Installing tkn
:context: installing-tkn

toc::[]

[role="_abstract"]
Use the CLI tool to manage {pipelines-title} from a terminal. You can install the CLI tool on different platforms.

[NOTE]
====
Both the archives and the RPMs contain the following executables:

* `tkn`
* `tkn-pac`
* `opc`
====

[NOTE]
====
The `openshift-pipelines-client` RPM is deprecated and might be removed in a future release of {pipelines-title}. Use standalone binaries for the `tkn`, `tkn-pac`, and `opc` CLI tools instead of the RPM package. Download the latest CLI binaries from https://developers.redhat.com/content-gateway/rest/browse/pub/cgw/pipelines/[{pipelines-title} binaries].
====
