[role="_abstract"]
Use the `tkn` CLI to manage pipelines, tasks, pipeline runs, and other {pipelines-shortname} resources from the command line.

Basic syntax::
+
[source,terminal]
----
$ tkn [command or options] [arguments...]
----

Global options::
+
[source,terminal]
----
--help, -h
----
