[role="_abstract"]
The {external-secrets-console-plugin-overview} is an Operator that manages the secrets and custom resource definitions (CRDs) for the secrets management Operators. Using {external-secrets-console-plugin-overview}, you can monitor and delete the custom resources of all the installed secret management Operators.

:FeatureName: {external-secrets-console-plugin-overview}
include::snippets/technology-preview.adoc[]

The following secrets management Operators can be managed with {external-secrets-console-plugin}:

* {external-secrets-operator}

* {cert-manager-operator}

* {secrets-store-operator}

You can use the plug-in to complete the following tasks:

* Monitor custom resource definitions (CRDs) for the secrets management Operators.

* View telemetry data, including the total number of managed secrets, certificates, and associated workloads.

* Monitor component health and performance through integrated alerts and Prometheus metrics.
