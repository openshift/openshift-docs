[role="_abstract"]
The {external-secrets-console-plugin} provides a unified OpenShift Console interface for viewing and managing secrets-related custom resources (CRs) from multiple Operators such as cert-manager, External Secrets Operator, and Secrets Store CSI Driver.

These release notes track the development of {external-secrets-console-plugin}.

For more information, see xref:../../security/external_secrets_management_console_plugin/index.adoc#external-secrets-console-plugin-overview[{external-secrets-operator-short} overview].
