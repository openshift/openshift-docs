= {external-secrets-operator} release notes

toc::[]

[role="_abstract"]
The {external-secrets-console-plugin} is a cluster-wide service that provides lifecycle management for secrets fetched from external secret management systems.

These release notes track the development of {external-secrets-console-plugin}.

For more information, see xref:../../security/external_secrets_management_console_plugin/index.adoc#external-secrets-console-plugin-overview[{external-secrets-operator-short} overview].
