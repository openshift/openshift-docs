= The Compliance Operator
include::_attributes/common-attributes.adoc[]
:context: understanding-compliance

toc::[]

[role="_abstract"]
The Compliance Operator helps {product-title} administrators assess and enforce security compliance across cluster resources and nodes. It uses OpenSCAP to scan against compliance profiles and identify configuration gaps that need remediation.

[IMPORTANT]
====
The Compliance Operator is available for {op-system-first} deployments only.
====
