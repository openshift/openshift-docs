= Integrating Red Hat OpenShift Service Mesh with Zero Trust Workload Identity Manager in a single-cluster
include::_attributes/common-attributes.adoc[]
:context: zero-trust-manager-mesh-integration

toc::[]

[role="_abstract"]
Deploy and configure SPIRE as the certificate authority (CA) for {SMProductName} workloads, replacing Istio's built-in CA with SPIFFE-compliant identities and automatically rotated short-lived certificates.

[id="prerequisites-mesh-integration_{context}"]
== Prerequisites

* You have access to an {product-title} cluster using an account with `cluster-admin` permissions.
* You have installed the `oc` CLI.
* You have installed {zero-trust-full} from the OperatorHub.
* You have installed the {SMProductName} 3.x Operator (`servicemeshoperator3`) from the OperatorHub.
