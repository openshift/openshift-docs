= SPIRE UpstreamAuthority plugins for Zero Trust Workload Identity Manager 
include::_attributes/common-attributes.adoc[]
:context: zero-trust-manager-plugins

toc::[]

[role="_abstract"]
To integrate SPIFFE Runtime Environment (SPIRE) with your existing certificate management infrastructure and keep Secure Production Identity Framework for Everyone (SPIFFE) identity standards, you can use UpstreamAuthority plugins. These plugins obtain intermediate signing certificates from external certificate authorities such as cert-manager or HashiCorp Vault.
