= Security Profiles Operator overview
include::_attributes/common-attributes.adoc[]
:context: spo-overview

[role="_abstract"]
{product-title} Security Profiles Operator (SPO) provides a way to define secure computing (seccomp) profiles and SELinux profiles as custom resources, synchronizing profiles to every node in a given namespace.

The SPO can distribute custom resources to each node while a reconciliation loop ensures that the profiles stay up-to-date.

The SPO manages SELinux policies and seccomp profiles for namespaced workloads.

You can create seccomp and SELinux profiles, bind policies to pods, record workloads, and synchronize all worker nodes in a namespace.

Use advanced Security Profiles Operator tasks to enable the log enricher, configure webhooks and metrics, or restrict profiles to a single namespace.

Use the Advanced Audit Logging Framework to access logs in Red Hat Enterprise Linux CoreOS (RHCOS) containers for container-level security audit features.

For troubleshooting, see the Security Profiles Operator troubleshooting documentation or engage Red Hat support.

To uninstall the Operator, remove the profiles before removing the Operator.
