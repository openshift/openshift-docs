= Security Profiles Operator overview
include::_attributes/common-attributes.adoc[]
:context: spo-overview

toc::[]

[role="_abstract"]
{product-title} Security Profiles Operator (SPO) provides a way to define secure computing (seccomp) profiles and SELinux profiles as custom resources, synchronizing profiles to every node in a given namespace. The SPO distributes custom resources to each node while a reconciliation loop ensures that the profiles stay up to date. The SPO manages SELinux policies and seccomp profiles for namespaced workloads. You can create seccomp and SELinux profiles, bind policies to pods, record workloads, and synchronize all worker nodes in a namespace.
