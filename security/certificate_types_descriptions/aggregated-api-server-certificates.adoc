== Purpose

Aggregated API Server certificates are used to secure connections between the KubeAPIServer and Aggregated API Servers.

== Management

These certificates are managed by the system and not the user.

== Expiration
This CA is valid for 30 days.

The issued client certificates are valid for 29 days.

CA and client certificates are rotated automatically through the use of controllers.

== Customization

You cannot customize the aggregated api server certificates.
