= Container image signatures

:context: security-container-signature

toc::[]

[role="_abstract"]
Red Hat delivers signatures for the images in the Red Hat Container Registries. Those signatures can be automatically verified when being pulled to {product-title} 4 clusters by using the Machine Config Operator (MCO). To verify the integrity of those images between Red Hat registries and your infrastructure, enable signature verification.
