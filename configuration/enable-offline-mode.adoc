= Enabling offline mode
include::modules/common-attributes.adoc[]
:context: enable-offline-mode

toc::[]

[role="_abstract"]
You can use {product-title} for clusters that are not connected to the internet by enabling the offline mode. In offline mode, {product-title} components do not connect to addresses or hosts on the internet.

[NOTE]
====
{product-title} does not find if the user-supplied hostnames, IP addresses, or other resources are on the internet. For example, if you try to integrate with a Docker registry hosted on the internet, {product-title} will not block this request.
====
