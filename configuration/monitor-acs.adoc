= Monitoring with Prometheus
include::modules/common-attributes.adoc[]
:context: monitor-acs

toc::[]

[role="_abstract"]
link:https://prometheus.io/[Prometheus] is an open-source monitoring and alerting platform.
You can use it to monitor health and availability of Central and Sensor components of {rh-rhacscs-first}. When you enable monitoring, {product-title-short} creates a new monitoring service on port number `9090` and a network policy allowing inbound connections to that port.
