= Install on Microsoft Azure
:context: install-on-azure

[role="_abstract"]
When deploying an OpenShift Container Platform cluster on Microsoft Azure, learn how to prepare your account, choose between automated and self-managed provisioning, and deploy the cluster so that you can run production workloads on Azure infrastructure while meeting your organization's networking and compliance requirements.
