== Prerequisites

. A Provisioned ROSA cluster
+
This lab assumes you have access to a successfully provisioned a ROSA cluster. If you have not yet created a ROSA cluster, see link:https://access.redhat.com/documentation/en-us/openshift_container_platform/4.16/html-single/getting_started/#rosa-getting-started-prerequisites_rosa-quickstart-guide-ui[Red Hat OpenShift Service on AWS quickstart guide] for more information.

. The OpenShift Command Line Interface (CLI)
+
For more information, see link:https://access.redhat.com/documentation/en-us/openshift_container_platform/4.16/html-single/cli_tools/#cli-getting-started[Getting started with the OpenShift CLI].

. A GitHub Account
+
Use your existing GitHub account or register at link:https://github.com/signup[https://github.com/signup].
