= Configuring physical networks
:context: configuring-physical-networks

[role="_abstract"]
You can create or configure a physical network in the {product-title} web console without using the node network configuration (NNCP) page. When you use the physical network page in the web console, you are creating networks that are more opinionated and focused on specific use cases. For complex settings, use the NNCP page.
