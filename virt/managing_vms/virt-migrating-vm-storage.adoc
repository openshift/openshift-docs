= Migrating VM storage
include::_attributes/common-attributes.adoc[]
:context: virt-migrating-vm-storage

toc::[]

[role="_abstract"]
To optimize performance or reduce costs, you can migrate virtual machines (VMs) between storage classes within a single cluster. You can use the {product-title} web console to migrate a single VM's disks or perform these migrations in bulk. Storage migration can be performed on both online and offline VMs.
