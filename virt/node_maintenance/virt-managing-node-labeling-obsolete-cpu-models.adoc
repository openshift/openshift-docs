= Managing node labeling for obsolete CPU models
include::modules/virt-document-attributes.adoc[]
include::modules/common-attributes.adoc[]
:context: virt-managing-node-labeling-obsolete-cpu-models
toc::[]

You can schedule a virtual machine (VM) on a node where the CPU model and policy attribute of the VM are compatible with the CPU models and policy attributes that the node supports. By specifying a list of obsolete CPU models in a xref:../../builds/builds-configmaps.adoc#builds-configmap-overview_builds-configmaps[ConfigMap], you can exclude them from the list of labels created for CPU models.
