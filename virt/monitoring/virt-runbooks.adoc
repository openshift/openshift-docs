= Alerts and runbooks
:context: virt-runbooks

toc::[]

[role="_abstract"]
Alerts notify you of issues in your environment. Each alert has a corresponding runbook that provides troubleshooting steps, diagnostic procedures, and resolution guidance for the condition that triggered the alert. Runbooks are available in the `openshift/runbooks` Git repository.
