= Configure local storage by using the hostpath provisioner
include::_attributes/common-attributes.adoc[]
:context: virt-configuring-local-storage-with-hpp

toc::[]

[role="_abstract"]
When you install the {VirtProductName} Operator, the Hostpath Provisioner (HPP) Operator is automatically installed. The HPP Operator creates the hostpath provisioner.

The hostpath provisioner is a local storage provisioner designed for {VirtProductName}. You can configure local storage for virtual machines by creating an `HostPathProvisioner` custom resource (CR) with a basic storage pool.

[IMPORTANT]
====
HPP storage pools must not be in the same partition as the operating system. Otherwise, the storage pools might fill the operating system partition. If the operating system partition is full, this might negatively impact performance, or the node can become unstable or unusable.
====
