= Recover individual files from VM backups
:context: virt-file-level-recovery

toc::[]

[role="_abstract"]
Restore specific files from virtual machine backups without requiring full VM restoration to reduce recovery time and avoid consuming cluster resources.

{oadp-short} virtual machine file restore (VMFR) provides file-level recovery from VM backups created with the `kubevirt` plugin.
You can browse backup contents through a web browser or access files programmatically using SSH-based tools such as `rsync`, `scp`, and `sftp`.
File-level recovery reduces recovery time objectives by recovering individual files in minutes instead of hours required for full VM restoration.
