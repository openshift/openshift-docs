= Set up file-level recovery for virtual machines
:context: virt-setup-file-recovery

toc::[]

[role="_abstract"]
Configure {oadp-short} to enable file-level recovery from VM backups and prepare infrastructure to support granular file recovery operations.

Enabling virtual machine file restore (VMFR) requires editing the `DataProtectionApplication` custom resource to activate the `vmFileRestore` feature and verify controller deployment.
Prerequisites include {oadp-short} Operator installation, {VirtProductName} deployment, and `kubevirt` Velero plugin configuration.
