= Using containerDisk volumes with virtual machines
include::modules/virt-document-attributes.adoc[]
:context: virt-using-containerdisk-volumes
toc::[]

A containerDisk volume is a virtual machine image stored inside a container image in a
container image registry that is available to {VirtProductName}. To use a containerDisk volume
with a virtual machine, you must create a container image with an existing virtual machine image
and push the container image to the registry. You can then attach the container disk to the virtual machine as an ephemeral containerDisk volume.
