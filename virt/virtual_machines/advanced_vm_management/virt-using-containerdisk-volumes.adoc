= Using containerDisk volumes with virtual machines
include::modules/virt-document-attributes.adoc[]
:context: virt-using-containerdisk-volumes
toc::[]

A containerDisk volume is a container disk that is directly attached to a virtual machine to be used for read-only, ephemeral storage.

To use a containerDisk volume with a virtual machine, you must have a container image with an existing virtual machine image prepared in an available container image registry. You can then attach the container disk to the virtual machine as an ephemeral containerDisk volume.
