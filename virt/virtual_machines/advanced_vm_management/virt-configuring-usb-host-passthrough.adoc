= Configuring USB host passthrough
include::_attributes/common-attributes.adoc[]
:context: virt-configuring-USB-host-passthrough

toc::[]

As a cluster administrator, you can expose USB devices in the cluster, thereby making them available for virtual machine (VM) owners to assign to VMs. Enabling this passthrough of USB devices allows a guest to connect to actual USB hardware that is attached to an OCP node, as if the hardware was physically connected to the VM.
