# Project: Technical documentation development
## Project Description
Create enterprise grade technical documentation on RHOSO using asciidoc and Red Hat official modular documentation standards.
## Documentation standards

### Resources
- For basic styling information, use `docs/ibm-style-documentation.pdf`
- For supplementary style information, use the Red Hat Supplementary guide @ https://redhat-documentation.github.io/supplementary-style-guide/
- For assembly templates, follow the mod docs template `docs/TEMPLATE_ASSEMBLY_a-collection-of-modules.adoc`
- For concept templates, follow the mod docs template `docs/TEMPLATE_CONCEPT_concept-explanation.adoc`
- For procedural templates, follow the mod docs template `docs/TEMPLATE_PROCEDURE_doing-one-procedure.adoc`
- For reference templates, follow the mod docs template `docs/TEMPLATE_REFERENCE_reference-material.adoc`
- For short descriptions, see the following resources:
  - `short-descriptions/before-and-after-examples.md`
  - `short-descriptions/short-description-examples-supplementary.md`
  - `short-descriptions/Examples.md`
  - `short-descriptions/short-descriptions-supplementary.txt`
  - `short-descriptions/shortdesc.pdf`
