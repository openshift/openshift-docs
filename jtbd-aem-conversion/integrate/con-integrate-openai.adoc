= Integrate OpenShift Lightspeed with OpenAI

[role="_abstract"]
As an administrator, you can integrate OpenAI as a large language model (LLM) provider for {ols-long}. Create a credential secret with your OpenAI API key, configure the `OLSConfig` custom resource, and verify the deployment.
