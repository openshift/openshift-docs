= Extend the Service with live cluster data and external tools

[role="_abstract"]
{ols-official} can access live cluster state to provide context-aware responses. Configure cluster interaction settings to control how the AI retrieves data from your cluster, enable custom MCP servers for external tool integration, and review the authentication flow for cluster introspection.
