= Integrate OpenShift Lightspeed with AWS Bedrock

[role="_abstract"]
As an administrator, you can integrate AWS Bedrock as a large language model (LLM) provider for {ols-long}. Review supported authentication methods, create a credential secret with your AWS Bedrock credentials, configure the `OLSConfig` custom resource, and verify the deployment.
