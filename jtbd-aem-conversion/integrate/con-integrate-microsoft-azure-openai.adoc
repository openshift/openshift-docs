= Integrate OpenShift Lightspeed with Microsoft Azure OpenAI

[role="_abstract"]
As an administrator, you can integrate Microsoft Azure OpenAI as a large language model (LLM) provider for {ols-long}. Create a credential secret with your Azure OpenAI credentials, configure the `OLSConfig` custom resource, and verify the deployment.
